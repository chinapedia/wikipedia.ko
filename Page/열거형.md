[컴퓨터 프로그래밍에서](../Page/컴퓨터_프로그래밍.md "wikilink") **열거형**(, enumeration), **이넘**(enum), **팩터**(factor ← [R 프로그래밍 언어와](../Page/R_\(프로그래밍_언어\).md "wikilink") 통계학의 [범주형 변수에서](https://ko.wikipedia.org/wiki/범주형_변수 "wikilink") 부르는 명칭)는 요소, 멤버라 불리는 명명된 [값](https://ko.wikipedia.org/wiki/값 "wikilink")의 집합을 이루는 [자료형](../Page/자료형.md "wikilink")이다. 열거자 이름들은 일반적으로 해당 언어의 [상수](https://ko.wikipedia.org/wiki/상수 "wikilink") 역할을 하는 [식별자](../Page/식별자.md "wikilink")이다.

일부 열거자 자료형은 언어에 [기본 소속되어](../Page/원시_자료형.md "wikilink") 있을 수 있다. [불리언 자료형을](../Page/불리언_자료형.md "wikilink") 예로 들면 거짓(False)과 참(True)의 값이 미리 정의된 열거형으로 볼 수 있다. 수많은 언어는 사용자들이 새로운 열거형을 정의할 수 있게 하고 있다.

열거형의 값과 변수는 일반적으로 고정 길이의 [비트 문자열로](https://ko.wikipedia.org/wiki/비트_문자열 "wikilink") 구현되며 이는 일부 [정수형](../Page/정수형.md "wikilink")과 호환되는 포맷과 크기로 되어 있다. 특히 [시스템 프로그래밍 언어와](https://ko.wikipedia.org/wiki/시스템_프로그래밍_언어 "wikilink") 같은 일부 언어들은 사용자가 각각의 열거자를 위해 사용할 비트 결합을 지정할 수 있게 한다.

## 파스칼 및 구문적으로 유사한 언어

### 파스칼

[파스칼에서](../Page/파스칼_\(프로그래밍_언어\).md "wikilink") 열거형은 괄호 목록에 값을 나열함으로써 내재적으로 선언할 수 있다:

``` Pascal
  var
    suit: (clubs, diamonds, hearts, spades);

```

이 선언은 형 동의어 선언에 등장할 수 있으며 이를 통해 여러 변수에 사용할 수 있다:

``` Pascal
  type
    cardsuit = (clubs, diamonds, hearts, spades);
    card = record
             suit: cardsuit;
             value: 1 .. 13;
           end;
  var
    hand: array [ 1 .. 13 ] of card;
    trump: cardsuit;
```

### 에이다

[에이다](https://ko.wikipedia.org/wiki/에이다 "wikilink")에서 "="의 사용은 "is"로 대체된다:

``` Ada
type Cardsuit is (clubs, diamonds, hearts, spades);
```

## C 및 구문적으로 유사한 언어

### C

원래의 [K\&R](../Page/C_\(프로그래밍_언어\).md "wikilink") 형태의 프로그래밍 언어인 [C는](../Page/C_\(프로그래밍_언어\).md "wikilink") 열거형이 없었다. C에 대해 [ANSI](https://ko.wikipedia.org/wiki/미국_국립_표준_협회 "wikilink") 표준에 추가되었으며 이것이 [ANSI C](../Page/ANSI_C.md "wikilink")(C89)로 되었다. C에서 열거 항목들은 `enum` 키워드, [struct](https://ko.wikipedia.org/wiki/struct "wikilink"), [유니언](https://ko.wikipedia.org/wiki/유니언_타입 "wikilink") 정의를 사용하는 명시적 정의로 만든다:

``` C
enum cardsuit {
   Clubs,
   Diamonds,
   Hearts,
   Spades
};

struct card {
   enum cardsuit suit;
   short int value;
} hand[13];

enum cardsuit trump;
```

### 펄

C의 문법 전통을 따르는 동적 자료형 언어(예: [펄](../Page/펄.md "wikilink"), [자바스크립트](../Page/자바스크립트.md "wikilink"))들은 일반적으로 열거를 제공하지 않는다. 그러나 펄에서는 [스트링](https://ko.wikipedia.org/wiki/스트링_리터털 "wikilink") [리스트와](../Page/리스트_\(컴퓨팅\).md "wikilink") [해시를](../Page/해시_테이블.md "wikilink") 사용하여 동일한 결과를 나타낼 수 있다.

``` C
my @enum = qw(Clubs Diamonds Hearts Spades);
my( %set1, %set2 );
@set1{@enum} = ();          # all cleared
@set2{@enum} = (1) x @enum; # all set to 1
$set1{Clubs} ...            # false
$set2{Diamonds} ...         # true
```

## 데이터베이스

일부 [데이터베이스](../Page/데이터베이스.md "wikilink")는 열거형을 직접 지원한다. [MySQL](../Page/MySQL.md "wikilink")은 테이블을 만들 때 문자열로 지정된 허용 가능한 값에 대해 `ENUM` 열거형을 제공한다.

## XML 스키마

[XML 스키마는](../Page/XML_스키마.md "wikilink") 다음과 같이 열거형을 지원한다.

``` XML
<xs:element name="cardsuit">
  <xs:simpleType>
    <xs:restriction base="xs:string">
      <xs:enumeration value="Clubs"/>
      <xs:enumeration value="Diamonds"/>
      <xs:enumeration value="Hearts"/>
      <xs:enumeration value="Spades"/>
    </xs:restriction>
  </xs:simpleType>
</xs:element>
```
