**부적초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [논산시](../Page/논산시.md "wikilink") [부적면](../Page/부적면.md "wikilink") 반송리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1933년                    | 6월 16일                 | 부적공립보통학교 개교(설립인가 [3월 21일](../Page/3월_21일.md "wikilink")) |
| ------------------------ | ---------------------- | -------------------------------------------------------- |
| 1949년                    | 12월 31일                | 부적국민학교로 개칭                                               |
| 1956년                    | 10월 18일                | 감곡국민학교 분리                                                |
| 1966년                    | 3월 4일                  | 부남국민학교 분리                                                |
| 1981년                    | 3월 5일                  | 병설유치원 개원                                                 |
| 1994년                    | 3월 1일                  | 부남분교장 편입                                                 |
| 1996년                    | 3월 1일                  | 부적초등학교 개칭                                                |
| 2007년                    | 2월 28일                 | 부남분교장 통폐합                                                |
| 12월 31일                  | 과학실, 영어실 현대화 완료        |                                                          |
| 복도 리모델링, 아름다운 화장실 개축(2층) |                        |                                                          |
| 2014년                    | 2월 14일                 | 제78회 졸업 (졸업생수 8,322명)                                    |
| 3월 1일                    | 7학급(특수학급1), 유치원 1학급 편제 |                                                          |
| 2015년                    | 2월 11일                 | 제79회 졸업(졸업생 수 8,336명)                                    |
| 3월 1일                    | 7학급(특수학급1), 유치원 1학급 편제 |                                                          |
| 9월 5일                    | 운동장 조성사업 완공 및 개장       |                                                          |
| 2016년                    | 2월 17일                 | 제80회 졸업(졸업생수 8,351명)                                     |
| 3월 1일                    | 7학급(특수학급1), 유치원 1학급 편제 |                                                          |
| 2017년                    | 2월 16일                 | 제81회 졸업(졸업생수 8,363명)                                     |
| 3월 1일                    | 7학급(특수학급1), 유치원 1학급 편제 |                                                          |
