<td><p>그림 = PORTRAIT OF CHARLES HODGE, Rembrandt Peale.jpg }}</p>
<p><strong>찰스 핫지</strong>(Charles Hodge, <a href="../Page/1797년.md" title="wikilink">1797년</a> <a href="../Page/12월_27일.md" title="wikilink">12월 27일</a>- <a href="../Page/1878년.md" title="wikilink">1878년</a> <a href="../Page/6월_19일.md" title="wikilink">6월 19일</a>)는 <a href="../Page/미국.md" title="wikilink">미국</a> <a href="../Page/장로교.md" title="wikilink">장로교</a> 신학자이며 1851년 부터 1878년 까지 <a href="../Page/프린스턴_신학교.md" title="wikilink">프린스턴 신학교의</a> 학장이었다. 그는 프린스턴 신학(Princeton Theology)과 19세기 동안 미국에서 정통 <a href="../Page/칼뱅주의.md" title="wikilink">칼뱅주의</a>신학 전통을 이끌었던 대표적인 인물이었다. 그는 하나님의 말씀으로서 성경의 권위를 강력히 주장했다. 그의 많은 사상들이 근본주의자들과 복음주의자들에 의해 20 세기에 받아들여졌다. 그의 아들도 같은 학교의 학장이며 조직신학 교수였던 <a href="https://ko.wikipedia.org/wiki/A._A._핫지" title="wikilink">A. A. 핫지이다</a>.</p>
<h2 id="생애">생애</h2>
<p>찰스 핫지의 아버지 휴(Hugh)는 18세기 초에 북 아이랜드로 부터 이민 온 <a href="../Page/스코틀랜드.md" title="wikilink">스코틀랜드</a> 출신의 아들이었다. 휴는  1777년 <a href="../Page/프린스턴_대학교.md" title="wikilink">프린스턴 대학을</a> 졸업하고 독립전쟁에 군의관으로 참전하고 후에 필라델피아에서 의사를 개업하였다.</p>
<h3 id="프린스턴_대학에_재학시절">프린스턴 대학에 재학시절</h3>
<p>핫지의 있어서 거듭남의 체험은 이 때에 일어났다. <a href="../Page/프린스턴_대학교.md" title="wikilink">프린스턴 대학교의</a> 6대 총장이었던 <a href="../Page/존_위더스푼.md" title="wikilink">존 위더스푼이</a> 대학생들의 도덕철학을 강조하고 <a href="https://ko.wikipedia.org/wiki/새뮤엘_스탠호프_스미스" title="wikilink">새뮤엘 스탠호프 스미스</a> 총장이 학생들에게 과학탐구를 소홀히 해서는 안 된다는 것을 강조했다고 할 것 같으면, <a href="https://ko.wikipedia.org/wiki/애쉬벨_그린" title="wikilink">애쉬벨 그린</a> 총장은 <a href="../Page/조너선_에드워즈.md" title="wikilink">조나단 에드워즈</a> 3대 총장처럼 학생들의 경건생활을 강조하였다. 경건생활을 강조하는 한 방법으로 주일 오후에 실시하던 성경공부를 폐지하고 소요리문답을 처음부터 끝까지 암송시키는 것을 의무화했다.[1]</p>
<h4 id="회심_체험">회심 체험</h4>
<p><a href="../Page/1815년.md" title="wikilink">1815년</a> <a href="../Page/1월_15일.md" title="wikilink">1월 15일</a> 프린스턴 장로교의 회중 앞에서, 그는 공개적으로 그의 신앙을 고백하였고, 이를 통하여 그 해에 있었던 부흥의 첫 회심자가 되었다.</p>
<p><br />
</p>
<h2 id="신학">신학</h2>
<p>그의 신학은 <a href="../Page/코튼_매더.md" title="wikilink">코튼 매더와</a> <a href="https://ko.wikipedia.org/wiki/조나단_에드워즈" title="wikilink">조나단 에드워즈처럼</a> 유기된 자의 지옥선포와 같은 <a href="../Page/칼뱅주의.md" title="wikilink">칼뱅주의</a> 교리와 <a href="../Page/존_위더스푼.md" title="wikilink">존 위더스푼의</a> <a href="../Page/스코틀랜드_상식_학파.md" title="wikilink">스코틀랜드 상식 학파의</a> 도덕철학을 추가하여 타락한 인간에게도 도덕적 감각을 가졌다고 주장하였다.[2]</p>
<h2 id="업적">업적</h2>
<p>핫지는 당시의 대표적인 신학자였던 <a href="../Page/프리드리히_슐라이어마허.md" title="wikilink">프리드리히 슐라이어마허의</a> 영향으로 ''신비주의''와 이성주의, 의식주의등에 함몰된 장로교를 개혁하기 위하여 '프린스톤 리뷰'라는 정기간행물을 발간하여 약 140편 이상의 논문을 출간하였다.[3]<br />
</p>
