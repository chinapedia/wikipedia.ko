**인천서운초등학교**는 [인천광역시](https://ko.wikipedia.org/wiki/인천광역시 "wikilink") [계양구](../Page/계양구.md "wikilink") [서운동에](https://ko.wikipedia.org/wiki/서운동_\(인천\) "wikilink") 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 2001년   | 3월 1일                | 인천서운초등학교 23학급 933명, 유치원 2학급 개교                                |
| ------- | -------------------- | ------------------------------------------------------------- |
| 7월 19일  | 도서실, 컴퓨터실, 서운문화회관 개관 |                                                               |
| 2002년   | 2월 15일               | 제1회 졸업식(107명)                                                 |
| 9월 11일  | 에디슨 교실 설치            |                                                               |
| 10월 30일 | 기상대 설치               |                                                               |
| 2007년   | 3월 2일                | 제3대 이현우 교장 취임                                                 |
| 2008년   | 7월 9일                | 과학실 현대화 사업                                                    |
| 10월 16일 | 학교생태숲 준공             |                                                               |
| 2009년   | 5월 15일               | 현대식 방송실 구축                                                    |
| 2010년   | 2월 28일               | 돌봄교실 구축                                                       |
| 4월 15일  | 음악실 조성               |                                                               |
| 9월 1일   | 제4대 선호록 교장 취임        |                                                               |
| 2011년   | 1월 14일               | 급식실 현대화 완공                                                    |
| 2013년   | 3월 1일                | 제5대 [김영준](https://ko.wikipedia.org/wiki/김영준 "wikilink") 교장 취임 |
| 2014년   | 5월 14일               | 과학체험교실 운영                                                     |
| 2016년   | 2월 12일               | 제15회 졸업식 (87명 졸업)                                             |
| 2017년   | 2월 10일               | 제16회 졸업장 수여식 거행(81명 총 2,138명 배출)                              |
| 9월 1일   | 제6대 오남주 교장 취임        |                                                               |
