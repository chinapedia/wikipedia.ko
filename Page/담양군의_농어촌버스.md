**담양군의 농어촌버스**는 [전라남도](https://ko.wikipedia.org/wiki/전라남도 "wikilink") [담양군](../Page/담양군.md "wikilink")의 대표적인 시내 대중교통 수단이다. 운수업체로 [동광담양고속](../Page/동광담양고속.md "wikilink")이 있다.

## 운임

2019년 1월 1일 기준이다.\[1\]

| 종류                 | 나이            | 현금 (원) | 카드 (원) |
| ------------------ | ------------- | ------ | ------ |
| 일반                 | 어른 (만 19세 이상) | 1,300  | 1,280  |
| 청소년 (만 13세 \~ 18세) | 1,000         | 980    |        |
| 어린이 (만 7세 \~ 12세)  | 650           | 630    |        |
|                    |               |        |        |

  - 본래 10km 까지는 기본 요금으로, 10km 초과 구간에 대해서 1km 당 116.14원씩 추가요금이 붙었으나, 2019년 1월 1일부로 담양군 관내 지역 요금을 단일화하였다. 담양군 이외 광주, 옥과, 순창, 장성 방면까지 이용할 경우 이전과 동일하게 추가요금이 가산된다.
  - [2013년](../Page/2013년.md "wikilink") [7월 19일부터](../Page/7월_19일.md "wikilink") 광역환승제도 시행으로 [광주광역시의 시내버스와](../Page/광주광역시의_시내버스.md "wikilink") [광주 도시철도로](../Page/광주_도시철도.md "wikilink") 환승할 때 환승할인을 받을 수 있다. 다만 광주광역시 경계를 진출입 할 경우에만 가능하다.
