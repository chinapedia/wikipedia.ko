문서 편집을 하다 보면, 한 문서에서 모두 설명하기보다 문서를 따로따로 분리하는 것이 더 좋을 수가 있습니다. 대표적인 예를 들어 보면 다음과 같습니다.

  - 한 문서가 여러 개의 주제를 가진 경우
  - 각각의 문서로 독립 가능한 큰 주제를 한 문서에서 설명하는 경우
  - 한 문서의 내용이 매우 방대하여 새로운 문서로 분리해야 할 필요성이 큰 경우

## 문서 크기를 기준으로 나누기

  - 대략적인 기준:

|            |                      |
| ---------- | -------------------- |
| **본문의 크기** | **해야할 일**            |
| \> 100 kB  | 거의 확정으로 나눌 필요가 있음    |
| \> 60 kB   | 나눌 필요가 상당히 있음        |
| \> 50 kB   | 향후 성장이 예상된다면 나눌 수 있음 |
| \< 40 kB   | 나눌 필요가 없음            |
| \< 1 kB    | 오히려 다른 문서와 병합이 필요함   |

## 문서 분할 제안

한 문서를 여러 문서에 나누려면 그 문서의 윗부분에  또는  틀을 다음과 같이 달아주면 됩니다.

  -
    {{분할 필요|다른 문서}}
    {{나누기|다른 문서}}

그 후에는 토론 문서에서 여러 이름 중 나눌 이름을 정하고, 분할 대상 문서를 각각의 문서로 나누어 재구성합니다.
