[이집트](../Page/이집트.md "wikilink")의 [공휴일](https://ko.wikipedia.org/wiki/공휴일 "wikilink")은 다음과 같다.\[1\]

| 날짜           | 영어 명칭                                     | 한글 명칭                                                             | 비고                                                    |
| ------------ | ----------------------------------------- | ----------------------------------------------------------------- | ----------------------------------------------------- |
| 01.07        | Coptic Christmas Day                      | [콥틱정교](../Page/알렉산드리아_콥트_정교회.md "wikilink") 성탄절                   |                                                       |
| 01.25        | Police and Revolution Day                 | 경찰의 날, 혁명 기념일                                                     |                                                       |
| 04,20        | Spring Festival                           | [봄의 날](https://ko.wikipedia.org/wiki/봄의_날 "wikilink")             |                                                       |
| 04.25        | Sinai Liberation Day                      | [시나이 해방 기념일](https://ko.wikipedia.org/wiki/시나이_해방_기념일 "wikilink") |                                                       |
| 05.01        | Labor Day                                 | [노동절](../Page/노동절.md "wikilink")                                  |                                                       |
| 05.24\~05.26 | Eid El Fitr                               | [이드 알피트르](https://ko.wikipedia.org/wiki/이드_알피트르 "wikilink")       | [소바이람](https://ko.wikipedia.org/wiki/소바이람 "wikilink") |
| 06.30        | Revolution Day                            | [2013년 이집트 쿠데타 기념일](../Page/2013년_이집트_쿠데타.md "wikilink")          |                                                       |
| 07.23        | Revolution Day                            | [1952년 이집트 혁명 기념일](../Page/1952년_이집트_혁명.md "wikilink")            |                                                       |
| 07.30\~08.03 | Feast of the Sacrifice Days (Eid Al Adha) | [이드 알아드하](https://ko.wikipedia.org/wiki/이드_알아드하 "wikilink")       | [대바이람](https://ko.wikipedia.org/wiki/대바이람 "wikilink") |
| 08.20        | Islamic New Year                          | [이슬람해 신년](https://ko.wikipedia.org/wiki/이슬람력 "wikilink")          |                                                       |
| 10.06        | Armed Forces Day                          | 국군의 날                                                             |                                                       |
| 10.29        | Prophet Mohamed’s Birthday                | [무함마드 탄생일](../Page/무함마드.md "wikilink")                            |                                                       |

  - [소바이람](https://ko.wikipedia.org/wiki/소바이람 "wikilink"), [대바이람](https://ko.wikipedia.org/wiki/대바이람 "wikilink"), [이슬람해 신년](https://ko.wikipedia.org/wiki/이슬람력 "wikilink"), [무함마드 탄생일은](../Page/무함마드.md "wikilink") [이슬람력](https://ko.wikipedia.org/wiki/이슬람력 "wikilink")에 따라 날짜가 변경될 수 있음.
  - 4.19 ([부활절](../Page/부활절.md "wikilink")), 5.23 ([소바이람 전야제](https://ko.wikipedia.org/wiki/소바이람_전야제 "wikilink"))는 정부에서 1-3일전에 휴일 여부를 결정, 공고함.
