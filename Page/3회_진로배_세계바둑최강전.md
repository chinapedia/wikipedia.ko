**3회 진로배 세계바둑최강전**은 [대한민국](../Page/대한민국.md "wikilink"), [중국](../Page/중국.md "wikilink"), [일본](../Page/일본.md "wikilink")의 한국 우승으로 3연패를 달성했다.

## 대회 일정

| 구분        | 일정                            | 장소                                                          |
| --------- | ----------------------------- | ----------------------------------------------------------- |
| 개막식       |                               | 중국 상하이                                                      |
| 1차전       | 1994년 12월 6일 \~ 1995년 12월 12일 |                                                             |
| 2차전       | 1995년 1월 11일 \~ 1995년 1월 17일  | 일본 도쿄 [일본기원](https://ko.wikipedia.org/wiki/일본기원 "wikilink") |
| 3차전       | 1995년 2월 21일 \~ 1995년 2월 22일  | 서울 힐튼호텔 국화룸                                                 |
| 시상식 및 폐막식 |                               |                                                             |
|           |                               |                                                             |

## 출전 기사 명단

  - [일본](../Page/일본.md "wikilink"): 미야자와 고로, 고마쓰 히데키, 다케미야 마사키, 가토마사오, 린하이펑
  - [대한민국](../Page/대한민국.md "wikilink"): [유창혁](../Page/유창혁.md "wikilink") 六단, [서봉수](../Page/서봉수.md "wikilink") 九단, [양재호](https://ko.wikipedia.org/wiki/양재호 "wikilink") 九단, [이창호](https://ko.wikipedia.org/wiki/이창호_\(바둑_기사\) "wikilink") 七단, [조훈현](../Page/조훈현.md "wikilink") 九단
  - [중국](../Page/중국.md "wikilink"): 류징, 정훙, 차오다위안, 마샤오춘, 녜웨이핑
