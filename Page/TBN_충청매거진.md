**주윤하, 안미소의 TBN 충청매거진**은 [TBN 대전교통방송](../Page/대전교통방송.md "wikilink")(대전·세종·충청 FM 102.9㎒, 서산 FM 103.9㎒, 충주 FM 93.5㎒)에서 평일 오후 4시 5분부터 5시 52분까지 방송되는 대전과 세종 충청권 지역의 라디오 정보 프로그램이다.

## 코너소개

### 매일 코너

  - 알짬뉴스 : 오늘의 알짬(핵심을 뜻하는 우리말)뉴스\!
  - 알짬퀴즈 : 오늘의 주제와 관련된 재미있는 퀴즈를 풀어보는 시간
  - 국도 정보 : 대전지방국토관리청 리포터의 실시간 국도 소통 정보
  - 아모르 상식 : '아마도 당신이 모르고 있을 상식'의 줄임말로 가볍게 알아두면 좋을 만한 상식
  - 한 발 빠른 대전시내 교통정보 : 대전 통신원을 통해 과거에 비해 빨라진 시내 러시아워 교통정보 제공

### 비정기 코너

  - 코로나 세계지도 : 세계 속에 코로나19 현황에 지속적으로 관심을 갖고 전염병 예방을 위한 경각심도 일깨우는 시간
  - 장보러 가유 : 우리 지역 전통시장 상인회로부터 듣는 시장 이야기

## 요일 코너

### 월요일

  - 대전이야기 : 허태정 대전시장이 직접 전하는 생생한 대전 시정 소식

### 화요일

  - 동구/중구/대덕/교육이야기 : 대전 각 구청장과 교육감이 전하는 구정 소식과 교육 소식
  - 꼭지뉴스 : 대전교통방송 유진희 기자기 전하는 우리 지역 뉴스
  - 초록힐링 : 국립자연휴양림이 소개하는 휴양림에서 힐링하기

### 수요일

  - 스포츠 전망대 : 최동호 평론가의 알찬 스포츠 소식
  - 충남이야기 : 충청남도 공보관실 이주민 주무관이 전하는 충남 소식
  - 여행 갈까유? : 산악인 이상은의 전국 방방곡곡 여행지 소개
  - 영화 뭐 보쥬? : 대전독립영화협회 민병훈 국장의 볼 만한 영화 추천

### 목요일

  - 더불어 사는 세상 : 대전사회적기업협의회가 소개하는 우리 동네 착한 기업
  - 온라인/오프라인의 발견 : 코로나19로 인해 달라진 생활상을 온라인과 오프라인에 걸쳐 발견해보는 시간
  - 유성/서구이야기 : 대전 각 구청장의 구정 소식
  - 산업안전수호대 : 안전보건공단 대전세종광역본부 협업으로 '산업재해 예방' 정책과 실천 방법 소개
  - 알려드려유 : 건강보험 정보(국민건강보험공단), 대체의학(전경미 교수), 산림 정책(산림청), 시민참여 방송(대전시청자미디어센터 정창욱 멘토) 등

### 금요일

  - 안전있슈 : 대전시 안전정책과 민동희 과장이 전하는 대전시 안전 관련 이슈
  - 실버6599 : 도로교통공단 이경은 교수와 함께 알아보는 노인교통사고 실태와 예방법
  - 창업으로 가는 길 : '대전창조경제혁신센터'를 통해 실제 창업가로부터 듣는 창업 스토리
