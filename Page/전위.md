**전위**(, ) 또는 **전기 퍼텐셜**은 시간에 따라 변하지 않는 [전기장](../Page/전기장.md "wikilink")에서 단위 전하가 가지게 되는 [전기적 위치 에너지다](../Page/전기적_위치_에너지.md "wikilink"). [국제단위계](https://ko.wikipedia.org/wiki/국제단위계 "wikilink")에서 단위는 [볼트다](https://ko.wikipedia.org/wiki/볼트_\(단위\) "wikilink").

## 정의

맥스웰 방정식에 따르면 \(\nabla \times E = - \frac{\partial B}{\partial t}\)인데, 정전기학에서는 전하가 움직이지 않으므로, 자기장이 없고, 따라서 이 값이 0이 된다. 즉 \(E\)의 [회전이](../Page/회전_\(벡터\).md "wikilink") 0이 되므로 이를 어떤 스칼라함수의 [기울기로](../Page/기울기_\(벡터\).md "wikilink") 표현할 수 있다. 보통 그 스칼라 함수를 \(-V\)라고 쓰고, \(V\)를 전위라고 부른다. 즉 \(E = - \nabla V\)가 된다.

[위치 에너지가](../Page/위치_에너지.md "wikilink") 절대적인 값이 아니라, 두 지점 사이의 위치 에너지 차이만이 절대적인 값인 것처럼, 전위 또한 상대적인 값이다. 즉 특정 지점의 전위는 임의로 정의할 수 있고, 두 지점의 [전위차](https://ko.wikipedia.org/wiki/전위차 "wikilink")만이 물리적 의미를 갖는다. (전위차는 [전압](https://ko.wikipedia.org/wiki/전압 "wikilink")이라고도 불린다.) 수학적인 이유는 \(V\)에 어떤 상수를 더해준 값인 \(V+a\)에 대해서도 \(E = - \nabla V = - \nabla \left(V + a \right)\)가 되어 같은 전기장이 나오기 때문이다.

다만, 이론 물리학에서는 통상적으로 계에서 "무한히 먼 곳"에서의 전위를 0으로 놓는다. [전기 회로에서는](../Page/전기_회로.md "wikilink") 보통 [접지](https://ko.wikipedia.org/wiki/접지 "wikilink")(接地, )의 전위를 0으로 놓거나, 아니면 회로도에 어느 점의 전위를 0으로 정의하는지 표시한다.

전위는 에너지와 전하의 비(단위전하가 가지고 있는 정전기 퍼텐셜 에너지)이다. 즉, 전위의 [국제 단위인](https://ko.wikipedia.org/wiki/국제단위계 "wikilink") [볼트는](https://ko.wikipedia.org/wiki/볼트_\(단위\) "wikilink") 다음과 같다.

  -
    볼트(V) = [줄](https://ko.wikipedia.org/wiki/줄_\(단위\) "wikilink")(J)/[쿨롱](https://ko.wikipedia.org/wiki/쿨롱 "wikilink")(C).

\[\Delta V = V_f - V_i = { U_f \over q } - { U_i \over q } = { \Delta U \over q }\]

\[\Delta V = V_A - V_B = - { \Delta U \over q_0 } =  - { \int_{B}^{A} { \vec{E} \cdot d \vec{s} } }\] (전위차의 정의)

\[V = { U \over q_0}\] (전위의 정의)

\[V_f - V_i = - { \int_{i}^{f} { \vec{E} \cdot d \vec{s} } }\] 두 점의 전위차(\(V_f- V_i\))는 두 점 사이의 전기장을 선적분 한 값이다. 이때, 전기장은 [보존장](https://ko.wikipedia.org/wiki/보존장 "wikilink")이기 때문에 적분구간의 모양은 상관 없고, 적분구간의 시작점과 끝점만 중요하다.

### 하나의 점전하에 의해 생기는 전위

\(V = { 1 \over 4 \pi \epsilon_0 } { q \over r }\)

  -
    아무것도 없는 공간에 존재하는 [점전하](https://ko.wikipedia.org/wiki/점전하 "wikilink") q가 있을 때, 이 전하에서 r만큼 떨어진 곳의 전위는 위의 공식과 같이 정의할 수 있다.
    이 공식은 무한히 먼 곳의 전위를 0V라고 놓은 다음에 세운 공식이다.
    \(\epsilon_0\) = 진공의 [유전율](https://ko.wikipedia.org/wiki/유전율 "wikilink")

### 여러 점전하에 의해 생기는 전위

\(V = \sum_{i=1}^{n} V_i = { 1 \over 4 \pi \epsilon_0 } \sum_{i=1}^{n} {q_i \over r_i}\)

  -
    공간에 점전하 n개가 존재할때, 특정 지점의 전위는 위의 공식과 같이 구할 수 있다.
    \(q\) = 각 전하의 전하량
    \(r\) = 전위를 구할 지점에서 각 전하까지의 거리

### [전기 쌍극자의](https://ko.wikipedia.org/wiki/전기_쌍극자 "wikilink") 전위

\(V = { 1 \over 4 \pi \epsilon_0 } { {p \cos \theta} \over r^2}\)


\=== 연속적인 전하 분포 === \(V = \int_ {} dV = {1 \over 4 \pi \epsilon_0} \int_ {} {dq \over r}\)



### [전기장](../Page/전기장.md "wikilink")과의 관계

\(E_s = {- \delta V \over \delta s}\)

\(E_x = - {\delta V \over \delta x}; E_y = - {\delta V \over \delta y}; E_z = - {\delta V \over \delta z}.\)

### 대전입자 계의 전기 퍼텐셜 에너지

\(U = W = q_2V = {1 \over 4 \pi \epsilon_0} {q_1 q_2 \over r}\)
