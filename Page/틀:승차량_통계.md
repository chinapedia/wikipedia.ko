## 사용법

{{승차량 통계|4536|5528|6075|...|이름 = |주석 = |합칸n = |칸모양n = }}

승차량 통계를 작성합니다. 이름 변수에는 노선 또는 회사가 들어가며, 통계 변수는 최대 10개까지 지원합니다.

합칸n은 colspan 변수를 주어서 칸을 합치는 역할을, 칸모양n은 각 칸에 스타일을 지정할 수 있는 역할을 맡고 있습니다. 이들 변수는 생략 가능합니다.

이 틀은 틀 조각의 형태를 취하고 있으므로, 실제 틀의 사용은 다음과 같습니다.

    {{승차량 통계/열기|칸수 = }}
    {{승차량 통계/제목줄|2002|2003|...|앞 = |뒤 = }}
    {{승차량 통계|4536|5528|...|이름 = |주석 = }}
