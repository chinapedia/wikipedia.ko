[섬네일](https://ko.wikipedia.org/wiki/파일:Gsl_nov_grand_final.jpg "wikilink") **소니 에릭슨 글로벌 스타크래프트 II 리그 November**(Sony Ericsson Global StarCraft II League November)는 [글로벌 스타크래프트 II 리그의](https://ko.wikipedia.org/wiki/글로벌_스타크래프트_II_리그 "wikilink") 13번째 정규 시즌으로 [2011년](https://ko.wikipedia.org/wiki/2011년 "wikilink") [11월 1일](https://ko.wikipedia.org/wiki/11월_1일 "wikilink") 개막했다.

## 리그 일정

  - Code A 예선 : 10월 26일, 27일
  - Code S 조지명식 : 10월 27일
  - Code S : 11월 1일 \~ 12월 3일
  - Code A : 11월 7일 \~ 12월 7일
  - 승격강등전 :

## 변경된 리그 방식

이번 시즌에 새로 적용되는 리그 방식은 Code S와 Code A의 관계가 더더욱 긴밀해진다. 우선 Code A는 기존 32명에서 48명으로 확충되며 이 인원은 기존 Code A 시드권자, 예선 통과자, 해외 대회 Code A 시드, 그리고 Code S 32강 조 4위로 구성되며, 그 다음 라운드인 Code A 36강 인원은 48강에서 승리한 선수와 Code S 32강 조 3위로 구성된다. 즉 기존 Code S 32강에서 한 번만 이겨도 잔류가 가능하고 그렇지 않더라도 승격강등전으로 진출하는 것과는 달리 이제 경우에 따라 Code A로 바로 추락하는 것은 물론 예선으로 바로 추락하는 경우도 생길 수 있게 되었다. Code S 16강에 진출해도 여기에서 조 3, 4위를 기록하게 될 경우 Code A 24강으로 추락. 남은 Code S 8강 진출자는 해외 대회 Code S 시드권자 2명과 함께 다음 시즌 Code S 잔류권을 확보하게 된다.

Code A에서는 기본적으로 24강에서 승리한 선수는 바로 Code S에 진출하게 되며 이외에 36강 까지 진출한 선수는 해외 대회 승격강등전 시드권자 2명과 함께 승격강등전에 진출한다. 총 30명, 6명 5개조, 풀리그로 이루어진 이 승격강등전에서 조 1, 2위를 차지한 선수는 Code S, 그 이하는 Code A에 그대로 남게 된다.

이로 인해 Code S, Code A 개막 일정도 변경되어 Code S가 Code A보다 먼저 시작하게 되며 또한 먼저 종료된다. Code A 24강과 승격강등전은 Code S 결승 이후에 치뤄진다.\[1\]
