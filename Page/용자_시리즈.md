**용자 시리즈**()는 [선라이즈](https://ko.wikipedia.org/wiki/선라이즈_\(스튜디오\) "wikilink"), 타카라, [나고야 TV](https://ko.wikipedia.org/wiki/나고야_TV "wikilink"), 도큐 에이전시가 [1990년](../Page/1990년.md "wikilink")부터 [1997년](../Page/1997년.md "wikilink")에 걸쳐 제작한 일련의 로봇 애니메이션 작품의 통칭이다.(보통 선라이즈와 반다이가 같이 제작하지만 용자 시리즈는 예외적으로 반다이가 아닌 타카라와 제작하였다.)

## 용자 시리즈의 자이언트로봇

### 용자 엑스카이저

1.  킹 엑스카이저(キングエクスカイザー) (엑스카이저 + 킹 로더)
2.  울트라 레이커(ウルトラレイカー) (그린 레이커 + 블루 레이커)
3.  갓 맥스(ゴッドマックス) (스카이 맥스 + 대시 맥스 + 드릴 맥스)
4.  드래곤 카이저(ドラゴンカイザー) (엑스카이저 + 드래곤 제트)
5.  그레이트 엑스카이저(グレートエクスカイザー) (킹 엑스카이저 + 드래곤 카이저) (얼티밋 합체)

### 태양의 용자 파이버드

1.  파이버드(ファイバード) (파이어 제트 + 카토리 유우타로)
2.  무장합체 파이버드 (武装 ファイバード) (파이버드 + 플레임 블레스터)
3.  썬더바론(サンダーバロン) (에이스바론 + 드릴바론 + 로드바론 + 아쿠아바론 + 스카이바론)
4.  가디언(ガーディオン) (가드스타 + 가드파이어 + 가드레스큐)
5.  슈퍼 가디언(スーパーガーディオン) (가디언 + 가드윙)
6.  그랑버드(グランバード) (파이어 셔틀 + 카토리 유우타로)
7.  제트합체 그랑버드 (ジェット グランバード) (그랑버드 + 블레스터 제트)
8.  그레이트 파이버드(グレートファイバード) (파이버드 + 그랑버드) (얼티밋 합체)

### 전설의 용자 다간

1.  다간X(ダ・ガーンX) (다간 + 어스파이터 + 어스라이너)
2.  스카이 세이버(スカイ セイバー) (점보 세이버 + 제트 세이버 + 셔틀 세이버)
3.  랜드 바이슨(ランドバイソン) (빅 랜더 + 터보 랜더 + 마하 랜더 + 드릴 랜더)
4.  페가서스 세이버(ペガサスセイバー) (스카이 세이버 + 호크 세이버)
5.  가온(ガ・オーン) (G버드 + 가온)
6.  그레이트 다간GX(グレートダ・ガーンGX) (다간X + 가온) (얼티밋 합체)

### 용자특급 마이트가인

1.  마이트가인(マイトガイン) (가인 + 마이트 윙 + 로코모라이저)
2.  트라이범버(トライボンバー) (라이오 범버 + 다이노 범버 + 버드 범버)
3.  가드다이버(ガードダイバー) (화이어 다이버 + 폴리스 다이버 + 드릴 다이버 + 제트 다이버)
4.  배틀범버(バトルボンバー) (트라이범버 + 혼 범버)
5.  마이트카이저(マイトカイザー)
6.  그레이트 마이트가인(グレートマイトガイン) (마이트가인 + 마이트카이저)
7.  마이트건너(マイトガンナー)
8.  그레이트 마이트가인 퍼펙트 모드(グレートマイトガイン パーフェクトモード) (그레이트 마이트가인 + 마이트건너) (얼티밋 합체)

### 용자경찰 제이데커

1.  제이데커(ジェイデッカー) (데커드 + 제이로더)
2.  빌드타이거(ビルドタイガー) (맥크레인 + 파워죠 + 덤프슨)
3.  쉐도우마루(シャドウ丸)
4.  슈퍼 빌드타이거(スーパービルドタイガー) (빌드타이거 + 드릴보이)
5.  건맥스(ガンマックス)
6.  제이데커 맥스캐논 모드(ジェイデッカー マックスキャノンモード) (제이데커 + 건맥스)
7.  듀크 화이어(デュークファイヤー) (듀크 + 화이어로더)
8.  화이어 제이데커(ファイヤージェイデッカー) (제이데커 + 듀크 화이어)
9.  화이어 제이데커 맥스캐논 모드(ファイヤージェイデッカー マックスキャノンモード) (화이어 제이데커 + 건맥스) (얼티밋 합체)

### 황금용자 골드런

1.  골드런(ゴルドラン) (드란 + 고르곤)
2.  어드벤저(アドベンジャー)
3.  실버리온(シルバリオン) (제트 실버 + 스타 실버 + 드릴 실버)
4.  스카이 골드런(スカイゴルドラン) (골드런 + 소라카게)
5.  갓 실버리온(ゴッドシルバリオン) (실버리온 + 파이어 실버)
6.  캡틴 샤크(キャプテンシャーク)
7.  레온카이저(レオンカイザー) (레온 + 카이저)
8.  그레이트 골드런(グレートゴルドラン) (스카이 골드런 + 레온카이저) (얼티밋 합체)

### 용자지령 다그온

1.  화이어 다그온(ファイヤーダグオン) (다그 화이어 + 화이어 점보 + 화이어 래더 + 화이어 레스큐)
2.  라이너 다그온(ライナーダグオン) (다그 터보 + 다그 아머 + 다그 윙)
3.  쉐도우 다그온(シャドーダグオン) (다그 쉐도우 + 가드 울프 + 가드 호크 + 가드 타이거)
4.  다그 베이스(ダグベース)
5.  슈퍼 라이너 다그온(スーパーライナーダグオン) (라이너 다그온 + 다그 드릴)
6.  건키드(ガンキッド)
7.  라이안(ライアン)
8.  썬더 다그온(サンダーダグオン) (다그 썬더 + 썬더 셔틀)
9.  파워 다그온(パワーダグオン) (다그파이어 + 파워 셔벨)
10. 슈퍼 화이어 다그온(スーパーファイヤーダグオン) (화이어 다그온 + 파워 다그온) (얼티밋 합체)
11. 파이널 다그온 (다그 파이어 + 다그 터보 + 다그 아머 + 다그 윙 + 다그 드릴 + 다그 쉐도우 + 다그 썬더)

### 용자왕 가오가이가

1.  가오가이가(ガオガイガー) (가이가 + 드릴 가오 + 라이너 가오 + 스텔스 가오)
