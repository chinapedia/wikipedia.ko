**2000년 하계 올림픽 남자 근대5종** 경기는 2000년 9월 30일에 [시드니](https://ko.wikipedia.org/wiki/시드니 "wikilink")에서 열렸다.

[러시아](https://ko.wikipedia.org/wiki/러시아 "wikilink")의 [드미트리 스바트코프스키가](https://ko.wikipedia.org/wiki/드미트리_스바트코프스키 "wikilink") 5,376점으로 금메달을 땄다.

## 경기 일정

<small>모든 경기 시간은 [오스트레일리아 기준](https://ko.wikipedia.org/wiki/오스트레일리아 "wikilink") ([UTC+10](https://ko.wikipedia.org/wiki/UTC+10:00 "wikilink"))</small>

| 날짜               | 경기 시간  | 라운드 |
| ---------------- | ------ | --- |
| 2000년 9월 30일 토요일 | 6시 45분 | 사격  |
| 8시               | 펜싱     |     |
| 11시 25분          | 수영     |     |
| 13시 45분          | 자전거 경주 |     |
| 16시 20분          | 달리기    |     |
