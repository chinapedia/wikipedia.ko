**부산국제매직페스티벌**()은 매년 [대한민국](../Page/대한민국.md "wikilink") [부산광역시](https://ko.wikipedia.org/wiki/부산광역시 "wikilink")에서 열리는 대규모의 [마술](../Page/마술.md "wikilink")축제로 2005년에 처음 개최되었다. 국내외 유명 매지션과 아마추어 매지션들이 함께 참여한다.

## 연혁

  - 제01회 부산국제매직페스티벌 2005 - 2005년 08월 01일 \~ 07일, 벡스코 01층 전시장
  - 제02회 부산국제매직페스티벌 2006 - 2006년 08월 10일 \~ 15일, 벡스코 01층 전시장
  - 제03회 부산국제매직페스티벌 2007 - 2007년 08월 02일 \~ 05일, 벡스코 01층 전시장
  - 제04회 부산국제매직페스티벌 2008 - 2008년 08월 01일 \~ 04일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 벡스코 컨벤션홀
  - 제05회 부산국제매직페스티벌 2009 - 2009년 08월 06일 \~ 09일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산 KBS홀
  - 제06회 부산국제매직페스티벌 2010 - 2010년 08월 05일 \~ 08일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 벡스코 컨벤션홀
  - 제07회 부산국제매직페스티벌 2011 - 2011년 08월 04일 \~ 07일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
  - 제08회 부산국제매직페스티벌 2012 - 2012년 08월 03일 \~ 06일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
  - 제09회 부산국제매직페스티벌 2013 - 2013년 08월 08일 \~ 11일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
  - 제10회 부산국제매직페스티벌 2014 - 2014년 08월 07일 \~ 10일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
  - 제11회 부산국제매직페스티벌 2015 - 2015년 08월 06일 \~ 09일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
  - 제12회 부산국제매직페스티벌 2016 - 2016년 08월 05일 \~ 08일, 개막식 : 해운대 해수욕장 야외특설무대 / 본행사 : 부산문화회관
