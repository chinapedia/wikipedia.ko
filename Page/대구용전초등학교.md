**대구용전초등학교**(大邱龍田初等學校)은 [대구광역시](../Page/대구광역시.md "wikilink") [달서구](https://ko.wikipedia.org/wiki/달서구 "wikilink")에 위치한 공립 초등학교이다. 교목은 향나무이고 교화는 장미이다.

| ------------------------ | -------------------------------- |
| 1999.09.01               | 대구용전초등학교로 개교(12학급), 초대 박달원 교장 취임 |
| 2000.02.18               | 제1회 졸업식 (41명)                    |
| 2000.03.01               | 18학급 편성(유치원 2학급 - 공룡반, 기린반)      |
| 2002.12.30               | 교사 중축 공사 준공(7개 교실)               |
| 2003.03.01               | 40학급 편성(유치원 2학급)                 |
| 2004.09.06 \~ 2006.08.31 | 교육인적자원부 지정 e-Learning연구학교 운영     |

| 2010.09.01               | 제6대 김종수 교장 취임                    |
| 2010.09.08               | 영어체험실 준공                         |
