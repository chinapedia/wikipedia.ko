**eXERD**는 (주)[토마토시스템](https://ko.wikipedia.org/wiki/토마토시스템 "wikilink")에서 개발된 [이클립스](https://ko.wikipedia.org/wiki/이클립스 "wikilink") 기반의 [데이터 베이스](https://ko.wikipedia.org/wiki/데이터_베이스 "wikilink") [모델링](https://ko.wikipedia.org/wiki/모델링 "wikilink") 도구이다. **eXERD**는 단독 실행과 [이클립스](https://ko.wikipedia.org/wiki/이클립스 "wikilink") [플러그인](../Page/플러그인.md "wikilink") 형식으로 제공된다. 현재 최신 버전은 2.4.2이며, [이클립스](https://ko.wikipedia.org/wiki/이클립스 "wikilink") 3.4 버전(Ganymede)에서 3.7 버전(Indigo), 4.3 버전(Kepler)에서 4.5 버전(Mars)까지 대응한다.

## 기능

  - 다중 다이어그램
  - 논리, 물리 이름 동시 편집
  - 인라인 편집
  - 관계 강조
  - 자동 정렬
  - 작업 메시지 공지
  - 검토 및 자동 수정
  - 표준 용어사전
  - 도메인
  - 포워드 엔지니어링
  - 리버스 엔지니어링
  - 컨텐트 어시스트
  - 데이터베이스 및 다른 파일의 데이터 모델과 비교 / 합병
  - 스크립트

## 지원 데이터베이스

  - [Oracle](https://ko.wikipedia.org/wiki/Oracle "wikilink") 9i, 10g, 11g, 12c
  - [Microsoft SQL Server](https://ko.wikipedia.org/wiki/Microsoft_SQL_Server "wikilink") 2005, 2008, 2012, 2014
  - [MySQL](https://ko.wikipedia.org/wiki/MySQL "wikilink") 5.1, 5.5, 5.6
  - [DB2](https://ko.wikipedia.org/wiki/DB2 "wikilink") 9.x
  - [PostgreSQL](../Page/PostgreSQL.md "wikilink") 9.x

  - 2010.12.30 eXERD 1.1.7 (Stable) - 정식으로 출판된 버전
  - 2011.08.26 eXERD 1.7.0 (Beta) - Microsoft SQL Server 2005 / 2008 지원
  - 2012.04.16 eXERD 2.0.0 - DB2 9.x 지원, Microsoft SQL Server 2012 지원

  - 2013.09.27 eXERD 2.3.0 - MySQL 5.6 지원
  - 2013.12.19 eXERD 2.3.1 - Oracle 12c 지원
