**이영미의 스튜디오 1039**는 [TBN 대구교통방송](../Page/대구교통방송.md "wikilink")(대구,경주,구미 FM 103.9㎒, 김천 FM 95.9㎒)에서 평일 아침 9시부터 10시 55분까지 방송되는 대구·경북권 지역의 출근길 아침 라디오 프로그램이다.

## 방송내용안내

### 1부

  - (매일)아침을 여는 이야기 : 가볍게 듣고 넘길 수 있는 이야기와 좋은 노래 한곡을 들려 드려요
  - (월)슬기로운 가정생활 : 화목한 가정을 일구어 나가는 유용한 팁을 알려 드립니다(진혜전 소장)
  - (화)1039 초대석 : 우리지역 화제의 인물, 각급 기관장 출연
  - (수)문화가 있는 수요일 : 매거진 월간대구문화 이효영 기자
  - (목)김성삼 교수의 행복시대 : 유쾌하게 사는 생활의 노하우(대구한의대학교 김성삼 교수)
  - (금)날씨와 생활 : 대구기상지청 관측예보과 김병춘 과장
  - 여러분의 일터는 안전한가요? : 안전보건공단 대구광역본부와 함께하는 불량 산업현장 탐사보도

### 2부

  - (월)걸어서 대구한바퀴 : 안영선 시인과 함께 떠나는 현대 대구골목 투어
  - (화)명장의 향기 : 누구보다 자기일에 최선을 다하는 우리지역 명인들을 만나봅니다(정윤화 리포터)
  - (수)라디오 정보통 : 알아두면 도움이되는 생활정보를 제공해 드립니다
      - Happy노후설계프로젝트 잘살아보세 : 국민연금 대구지역본부 이은숙 차장
      - 슬기로운 금융생활 : 금융감독원 대구경북지원 소비자보호팀 서현재 팀장
      - Now 대구경제 : 대구테크노파크 권업 원장
      - 슬기로운 소비생활 : 대구경북소비자연맹 김은주 팀장
  - (목) 슬기로운 건강생활 :대학병원 및 전문병원 전문의로부터 듣는 과목별 최신 의학정보
  - (금) 공간의 발견'아는 만큼 보인다' : 김현진 건축가
