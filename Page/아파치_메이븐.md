[섬네일](https://ko.wikipedia.org/wiki/파일:Maven_CoC.svg "wikilink") **아파치 메이븐**()은 [자바용](https://ko.wikipedia.org/wiki/자바_\(프로그래밍_언어\) "wikilink") 프로젝트 관리 도구이다. [아파치 앤트의](https://ko.wikipedia.org/wiki/아파치_앤트 "wikilink") 대안으로 만들어졌다. [아파치 라이선스로](https://ko.wikipedia.org/wiki/아파치_라이선스 "wikilink") 배포되는 [오픈 소스 소프트웨어이다](https://ko.wikipedia.org/wiki/오픈_소스_소프트웨어 "wikilink").

## 예

``` xml
<project>
  <!-- model version is always 4.0.0 for Maven 2.x POMs -->
  <modelVersion>4.0.0</modelVersion>

  <!-- project coordinates, i.e. a group of values which
       uniquely identify this project -->

  <groupId>com.mycompany.app</groupId>
  <artifactId>my-app</artifactId>
  <version>1.0</version>

  <!-- library dependencies -->

  <dependencies>
    <dependency>

      <!-- coordinates of the required library -->

      <groupId>junit</groupId>
      <artifactId>junit</artifactId>
      <version>3.8.1</version>

      <!-- this dependency is only used for running and compiling tests -->

      <scope>test</scope>

    </dependency>
  </dependencies>
</project>
```
