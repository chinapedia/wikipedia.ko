**일차 부등식**(一次不等式)은 [미지수](https://ko.wikipedia.org/wiki/미지수 "wikilink")의 최고 차항의 [차수](https://ko.wikipedia.org/wiki/차수 "wikilink")가 1인 [부등식](../Page/부등식.md "wikilink")을 뜻한다.

## 일반형

\(x\)에 대한 일차 부등식의 일반형은 다음과 같다.

\[ax>b\]

\[ax<b\](단, \(a\), \(b\)는 [상수](https://ko.wikipedia.org/wiki/상수 "wikilink")이다.)

## 예시

  - \(\ 4x-7y>4x-5(x-34)+9z\)
  - \(\ {x-3 \over 6}>{7x+2 \over 9}\)
  - \(\ 1.2x-0.6>x\)

## 일반적인 해법

\(x\)에 대한 일차 부등식의 일반적인 해법은 다음과 같다.

| \(ax>b\) | \(b \ge 0\)        | \(b<0\)      |
| -------- | ------------------ | ------------ |
| \(a>0\)  | \(x>\frac {b}{a}\) |              |
| \(a=0\)  | 해 없음               | \(x\)는 모든 실수 |
| \(a<0\)  | \(x<\frac {b}{a}\) |              |
