**오세현**(吳世賢, 1968년 6월 15일 \~ )은 대한민국의 정치인이다.지금은 제7대 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [아산시장](../Page/아산시장.md "wikilink")이다.

## 학력

| 연도           | 학교                                                      | 학과     | 비고 |
| ------------ | ------------------------------------------------------- | ------ | -- |
| 1973 \~ 1979 | [배방초등학교](../Page/배방초등학교.md "wikilink")                  | ―      | 졸업 |
| 1979 \~ 1982 | [아산중학교](../Page/아산중학교_\(충남\).md "wikilink")             |        |    |
| 1982 \~ 1985 | [천안중앙고등학교](../Page/천안중앙고등학교.md "wikilink")              |        |    |
| 1985 \~ 1990 | [경희대학교](https://ko.wikipedia.org/wiki/경희대학교 "wikilink") | 정치외교학과 |    |
| 2010 \~ 2012 | 센트럴 미시간대학교                                              | 행정학과   |    |

## 경력

  - 2016.01 \~ 2016.12: 충청남도청 복지보건국 국장
  - 2017.01 \~ 2017.12: 충청남도 아산시 부시장
  - 2018.07 \~ : 제7대 민선 7기 충청남도 아산시장(초선)
