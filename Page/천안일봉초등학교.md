**천안일봉초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [천안시](../Page/천안시.md "wikilink") [동남구](../Page/동남구.md "wikilink") [다가동에](https://ko.wikipedia.org/wiki/다가동_\(천안시\) "wikilink") 있는 공립 초등학교이다.

## 학교 연혁

| 1934년 | 10월 10일              | **환성공립보통학교** 개교(설립인가 [9월 18일](../Page/9월_18일.md "wikilink"), 환성면 용곡리) |
| ----- | -------------------- | --------------------------------------------------------------------- |
| 1938년 | 4월 1일                | **환성공립심상소학교** 개편                                                      |
| 1941년 | 4월 1일                | **환성공립국민학교** 개편                                                       |
| 1956년 | 4월 21일               | **환성국민학교**로 개칭                                                        |
| 1962년 | 11월 2일               | **천안일봉국민학교**로 개칭                                                      |
| 1996년 | 3월 1일                | 천안일봉초등학교로 개칭                                                          |
| 2016년 | 3월 1일                | 21학급 편성(특수1학급 포함)                                                     |
| 2017년 | 2월 10일               | 제77회 91명 졸업(졸업생 총 19,794명)                                            |
| 3월 1일 | 21학급 편성(특수 1학급 포함)   |                                                                       |
| 2018년 | 2월 9일                | 제78회 99명 졸업(졸업생 총 19,893명)                                            |
| 3월 1일 | : 21학급 편성(특수 1학급 포함) |                                                                       |
