**윈도우 스몰 비즈니스 서버**(), 혹은 **윈도우 서버 에센셜** ()은 중소 기업이 쓰기에 최적화된 서버 계열 [운영 체제이다](../Page/운영_체제.md "wikilink"). 2011년 Windows Small Business Server 2011 출시 이후, "윈도우 서버 에센셜"이라는 이름으로 변경하여 현재에 이르고 있다.

## 버전

  - 1997년 10월 27일 – BackOffice Small Business Server 4.0 (코드네임 *Sam*)
  - 1999년 5월 24일 – BackOffice Small Business Server 4.5 (코드네임 *Horton*)
  - 2001년 2월 21일 – Microsoft Small Business Server 2000
  - 2003년 10월 9일 – Windows Small Business Server 2003 (코드네임 *Bobcat*)
  - 2006년 7월 29일 - Windows Small Business Server 2003 R2
  - 2008년 8월 21일 - Windows Small Business Server 2008 (코드네임 *Cougar*)
  - 2010년 12월 13일 - Windows Small Business Server 2011
  - 2012년 10월 10일 - Windows Server 2012 Essentials
  - 2013년 9월 9일 - Windows Server 2012 R2 Essentials
  - 2016년 10월 12일 - Windows Server 2016 Essentials
