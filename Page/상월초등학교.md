**상월초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [논산시](../Page/논산시.md "wikilink") [상월면](../Page/상월면.md "wikilink") 신충리에 있는 공립 초등학교이다.

## 학교 연혁

| 1935년   | 6월 7일              | **상월공립보통학교** 개교(지경리 임시 교사, 설립인가 [4월 30일](../Page/4월_30일.md "wikilink")) |
| ------- | ------------------ | ----------------------------------------------------------------------- |
| 12월 23일 | 신충리 현 교사로 이전       |                                                                         |
| 1938년   | 4월 1일              | **상월공립심상소학교**로 개칭                                                       |
| 1940년   | 8월 30일             | 6년제로 승격                                                                 |
| 1941년   | 4월 1일              | **상월공립국민학교**로 개칭                                                        |
| 1960년   | 11월 24일            | [대명국민학교](../Page/대명초등학교.md "wikilink") 분리                               |
| 1966년   | 3월 4일              | 한천국민학교 분리                                                               |
| 1981년   | 3월 6일              | 병설유치원 개원                                                                |
| 1992년   | 3월 1일              | 한천국민학교 통합                                                               |
| 1996년   | 3월 1일              | 상월초등학교로 개칭                                                              |
| 2009년   | 7월 16일             | 꿈빛관 개관                                                                  |
| 2016년   | 2월 17일             | 제78회 졸업(졸업생 10명, 총 6,146명)                                              |
| 3월 1일   | 초등 6학급, 유치원 1학급 편성 |                                                                         |
| 2017년   | 2월 10일             | 79회 졸업(졸업생 7명, 총6,153명)                                                 |
| 3월 1일   | 초등 6학급, 유치원 1학급 편성 |                                                                         |
