**2012 핫식스 글로벌 스타크래프트 II 리그 시즌 2** (2012 Hot6ix Global Starcraft 2 League Season 2) 는 [곰TV](../Page/곰TV.md "wikilink")가 주최하는 [글로벌 스타크래프트 II 리그의](https://ko.wikipedia.org/wiki/글로벌_스타크래프트_II_리그 "wikilink") 16번째 정규 리그로 2012년에 두 번째로 열리는 정규 리그이다.

## 대회 일정

  - Code A : 4월 17일 \~ 5월 17일
  - Code S : 3월 26일 \~ 5월 19일
