**패럿**(Parrot)은 [동적 언어를](../Page/동적_프로그래밍_언어.md "wikilink") 효율적으로 실행할 수 있도록 설계된 [레지스터 기반](https://ko.wikipedia.org/wiki/레지스터_머신 "wikilink") [프로세스 가상 머신이다](../Page/가상_머신.md "wikilink"). [패럿 어셈블리어와](https://ko.wikipedia.org/wiki/패럿_어셈블리어 "wikilink") PIR (중간 언어)를 패럿 [바이트코드](../Page/바이트코드.md "wikilink")로 컴파일하여 실행할 수 있다. 패럿은 [자유](../Page/자유_소프트웨어.md "wikilink") 및 [오픈 소스](../Page/오픈_소스.md "wikilink") 소프트웨어이다.\[1\]

패럿은 [펄 프로그래밍](../Page/펄.md "wikilink") 커뮤니티에서 비롯하였으며 [오픈 소스 및 자유 소프트웨어 운동으로부터](https://ko.wikipedia.org/wiki/자유_소프트웨어_운동 "wikilink") 도움을 받아 개발되어 있다. 그 결과 펄의 라이선스([아티스틱 라이선스 2.0](../Page/아티스틱_라이선스.md "wikilink"))와 호환된다는데 중점을 두고 있다.

개발을 위한 안정적인 [API](../Page/API.md "wikilink")를 갖춘 버전 1.0은 2009년 3월 17일에 공개되었다.\[2\]

## 예

### 레지스터

패럿은 스택 기반인 대부분의 가상 머신과 달리 대부분의 하드웨어 [CPU와](../Page/중앙_처리_장치.md "wikilink") 같이 레지스터 기반이다. 패럿은 다음의 네 종류의 레지스터를 제공한다.

  - I (정수)
  - N ([부동소수점](../Page/부동소수점.md "wikilink"))
  - S (문자열)
  - P (PMC: *Polymorphic Container*) - 패럿 오브젝트 타입

### 산술 연산자

  - PASM

`   set I1, 4`
`   inc I1        # I1 is now 5`
`   add I1, 2     # I1 is now 7`
`   set N1, 42.0`
`   dec N1        # N1 is now 41.0`
`   sub N1, 2.0   # N1 is now 39.0`
`   print I1`
`   print ', '`
`   print N1`
`   print "\n"`
`   end`

  - PIR

<!-- end list -->

``` perl6
 .sub 'main' :main
    $I1 = 4
    inc $I1     # $I1 is now 5
    $I1 += 2    # $I1 is now 7
    $N1 = 42.0
    dec $N1     # $N1 is now 41.0
    $N1 -= 2.0  # $N1 now 39.0
    print $I1
    print ', '
    print $N1
    print "\n"
 .end
```
