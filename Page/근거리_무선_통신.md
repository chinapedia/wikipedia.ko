**근거리 무선 통신**(, )는 13.56MHz의 대역을 가지며, 아주 가까운 거리의 무선 통신을 하기 위한 기술이다. 현재 지원되는 데이터 통신 속도는 초당 424 킬로비트다. 교통, 티켓, 지불 등 여러 서비스에서 사용할 수 있다.

## 모드

| 모드                   | 동작                              | 응용분야                        |
| -------------------- | ------------------------------- | --------------------------- |
| 카드 모드                | NFC를 탑재한 기기가 기존의 비접촉식 카드와 같이 동작 | 신용카드, 교통카드, 멤버십 카드등 각종 카드   |
| Read/Write 모드        | NFC를 탑재한 기기가 RFID 태그 리더기로 동작    | 스마트 포스터 같은 옥외광고, 작품 설명 등    |
| P2P(Peer-to-Peer) 모드 | NFC 기기 간 데이터 송수신                | 개인 간 데이터 전송, 명함 교환, 개인 송금 등 |

| 212 kbps                                                  | 맨체스터, 10% ASK                                                 | 맨체스터, 10% ASK |
