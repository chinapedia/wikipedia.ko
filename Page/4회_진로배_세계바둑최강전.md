**4회 진로배 세계바둑최강전**은 [대한민국](../Page/대한민국.md "wikilink"), [중국](../Page/중국.md "wikilink"), [일본](../Page/일본.md "wikilink")의 한국이 4연패를 달성했다.

## 대회 일정

| 구분        | 일정                            | 장소            |
| --------- | ----------------------------- | ------------- |
| 개막식       |                               | 일본 도쿄 일본기원    |
| 1차전       | 1995년 12월 5일 \~ 1995년 12월 11일 |               |
| 2차전       | 1996년 1월 9일 \~ 1996년 1월 13일   | 서울 힐튼호텔 국화룸   |
| 3차전       | 1996년 2월 6일 \~ 1996년 2월 10일   | 중국 상하이 신진장 호텔 |
| 시상식 및 폐막식 |                               |               |
|           |                               |               |

## 참가 선수 명단

  - [일본](../Page/일본.md "wikilink"): 모리타 미치히로, 미야자와 고로, [요다 노리모토](../Page/요다_노리모토.md "wikilink"), [다케미야 마사키](../Page/다케미야_마사키.md "wikilink") 九단, 야마히로 히로시
  - [대한민국](../Page/대한민국.md "wikilink"): [최규병](https://ko.wikipedia.org/wiki/최규병 "wikilink") 七단, [양재호](https://ko.wikipedia.org/wiki/양재호 "wikilink") 九단, [유창혁](../Page/유창혁.md "wikilink") 七단, [이창호](https://ko.wikipedia.org/wiki/이창호_\(바둑_기사\) "wikilink") 七단, [조훈현](../Page/조훈현.md "wikilink") 九단
  - [중국](../Page/중국.md "wikilink"): 뤄시허, 천린신, 차오다위안, 녜웨이핑, 마사오춘
