### 2018년 상반기(PKL \#1)

2018년도의 PKL은 한국 지역에서 진행되는 PUBG 공인 인증 대회 중에서 PUBG Tour Point가 부여되는 모든 대회들이 속해 있는 리그 시스템을 뜻한다. PKL 소속 대회는 포인트 부여 등급 별로 Pro Tour, A Tour, B Tour의 세 가지 대회의 형태로 진행되었다. 각 Tour에 소속된 대회는 다음과 같다.

| **2018 PKL \#1**       |
| ---------------------- |
| **Pro Tour**           |
| 펍지 서바이벌 시리즈(PSS)       |
| 펍지 워페어 마스터즈(PWM)       |
| **A Tour**             |
| **B Tour**             |
| OZ Arena PUBG Survival |
| 삼성 SSD 970 PKL B Tour  |
| 레노버 리전 챔피언십 PKL B Tour |

PKL 종료 후 각 팀이 획득한 PUBG Tour Point 합산 순위에 따라 상위 팀은 향후 글로벌 대회가 있을 시 우선 출전자격을 얻게 된다.

2018년 상반기 프로투어에는 아프리카TV가 주관한 ‘아프리카TV 펍지 리그(APL)’ 2회, OGN이 주관한 ‘펍지 서바이벌 시리즈(PSS)’ 2회, SPOTV GAMES가 주관한 ‘펍지 워페어 마스터즈(PWM)’ 1회를 진행했다.
