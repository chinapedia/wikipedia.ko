〈**Danger**〉는 [방탄소년단](../Page/방탄소년단.md "wikilink")의 첫 번째 정규 음반 《[DARK\&WILD](../Page/DARK&WILD.md "wikilink")》의 타이틀 곡이다. 2014년 8월 20일에 발표하였다.

| 가온 주간 다운로드 차트\[2\]                     | 44   |
| 가온 주간 BGM 차트\[3\]                      | 36   |

|                                        |      |
