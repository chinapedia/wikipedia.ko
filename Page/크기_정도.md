[섬네일부터](https://ko.wikipedia.org/wiki/파일:Orders_of_magnitude_\(english_annotations\).png "wikilink") [우주](../Page/우주.md "wikilink")까지의 대상들은 서로 다른 크기 정도를 가진다.\]\]

**크기 정도**(-程度, ) 또는 **크기 자릿수**(-數)는 [상용로그](../Page/상용로그.md "wikilink")의 척도로 근사한 수의 크기의 등급이다. 즉, 각 등급은 그 이전 등급의 10배가 된다.\[1\] 수의 크기 정도는 그 수를 10의 [거듭제곱](../Page/거듭제곱.md "wikilink")으로 나타낼 때의 지수로 나타낼 수도 있다.\[2\] 만약 두 수의 [과학적 표기법의](https://ko.wikipedia.org/wiki/과학적_표기법 "wikilink") 지수가 같다면, 또는 두 수의 비율이 10배 이하라면, 이 두 수의 크기 정도가 같다()고 하며,\[3\]\[4\] 또한 예를 들어 만약 두 수의 과학적 표기법의 지수의 차이가 3이라면, 또는 두 수가 약 천 배 차이라면, 두 수가 세 크기 정도만큼 차이가 난다()고 한다.\[5\] 크기 정도가 같다는 말은 [대문자 O 표기법의](https://ko.wikipedia.org/wiki/대문자_O_표기법 "wikilink") 뜻으로 쓰이기도 한다.\[6\]

## 예시

양의 크기 정도의 예시는 다음과 같다.

| 양                 | 크기 정도              |
| ----------------- | ------------------ |
| 태양의 반지름           | 10<sup>6</sup>킬로미터 |
| 원자핵의 반지름          | 1펨토미터              |
| 쥐라기가 오늘날부터 떨어진 시간 | 1억년                |
| 꼭대기 쿼크의 수명        | 10<sup>-24</sup>초  |

두 양의 크기 정도의 차이를 서술하는 예시는 다음과 같다.

| 문구                                                                                 | 의미                                              |
| ---------------------------------------------------------------------------------- | ----------------------------------------------- |
| [목성](../Page/목성.md "wikilink")과 [토성](../Page/토성.md "wikilink")의 질량은 같은 크기 정도를 갖는다. | 목성과 토성의 질량의 차이는 10배 이하이다.\[7\]                  |
| 플라스크 속 박테리아의 수가 하나의 크기 정도만큼 증가하였다.                                                 | (플라스크 속 박테리아의 수가 몇백에서 몇천으로 늘었을 때 할 수 있는 말)\[8\] |
| 이 가공 기구들의 속도는 최근 들어 두 크기 정도만큼 증가하였다.                                               | 이 가공 기구들의 속도는 최근 들어 약 100배만큼 증가하였다.\[9\]        |
| \(f\)와 \(\phi\)는 같은 크기 정도를 갖는다.                                                    | (일부 문헌에서 \(f=O(\phi)\)임을 뜻할 때 쓰는 말)\[10\]       |
