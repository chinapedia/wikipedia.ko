**키즐랴르** (, , )는 [다게스탄에](https://ko.wikipedia.org/wiki/다게스탄_공화국 "wikilink") 있는 도시이다. [델타 강의](https://ko.wikipedia.org/wiki/델타_강 "wikilink") [테레크 강이](https://ko.wikipedia.org/wiki/테레크_강 "wikilink") 남동부의 [마하치칼라](https://ko.wikipedia.org/wiki/마하치칼라 "wikilink")를 향해 221km뻗어 나간다.

## 인구

인구는 30,000명([1970](https://ko.wikipedia.org/wiki/1970 "wikilink"))이다.

|    |      |      |      |      |      |      |
| -- | ---- | ---- | ---- | ---- | ---- | ---- |
| 연도 | 1959 | 1970 | 1979 | 1989 | 2000 | 2006 |
| 인구 | 25.6 | 29.7 | 31.3 | 39.7 | 45.1 | 48.8 |

이 도시는 [러시아인](../Page/러시아인.md "wikilink"), [아르메니아인](https://ko.wikipedia.org/wiki/아르메니아인 "wikilink"), [체첸인](https://ko.wikipedia.org/wiki/체첸인 "wikilink"), 다게스탄의 여러 민족들이 거주한다.

키즐랴르에는 1970년에서 1989년까지 러시아인들이 많았었다. 러시아인들은 2%를 차지하고, 시간이 지나면서 아르메니아인들과 레즈긴인들이 조금씩 많아졌다. [다르긴인](https://ko.wikipedia.org/wiki/다르긴인 "wikilink")도 3.5%를 차지하고, [라크인](https://ko.wikipedia.org/wiki/라크인 "wikilink")들이 7%를 차지한다.
