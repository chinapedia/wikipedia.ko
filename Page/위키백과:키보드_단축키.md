[위키백과](https://ko.wikipedia.org/wiki/위키백과 "wikilink")의 기본 [스킨인](../Page/스킨_\(컴퓨팅\).md "wikilink") 벡터 스킨에서는 키보드 단축키를 이용할 수 있습니다. 이것을 이용하면 위키백과의 특정 기능을 더 빠르게 이용할 수 있습니다.

## 단축키 목록

브라우저에 따라 누르고 있어야 하는 키가 다릅니다.

  - [윈도우](https://ko.wikipedia.org/wiki/마이크로소프트_윈도우 "wikilink")(Windows)에 설치된 [파이어폭스](https://ko.wikipedia.org/wiki/모질라_파이어폭스 "wikilink") 1.5:  + 접근 키
  - 윈도우와 [리눅스](https://ko.wikipedia.org/wiki/리눅스 "wikilink")에 설치된 파이어폭스 2 이상:  + 접근 키
      - [OS X에](https://ko.wikipedia.org/wiki/OS_X "wikilink") 설치된 파이어폭스:  + 접근 키
  - [윈도우에](https://ko.wikipedia.org/wiki/마이크로소프트_윈도우 "wikilink") 설치된 [인터넷 익스플로러](https://ko.wikipedia.org/wiki/인터넷_익스플로러 "wikilink") 6:  + 접근 키
  - 윈도우에 설치된 인터넷 익스플로러 7:  + 접근 키
  - 윈도우에 설치된 인터넷 익스플로러 8:  + 접근 키 (이후, 를 따로 눌러야 한다.)
  - 모든 운영체제에 설치된 [오페라](https://ko.wikipedia.org/wiki/오페라_\(웹_브라우저\) "wikilink"):  + 접근 키
  - [윈도우에](https://ko.wikipedia.org/wiki/마이크로소프트_윈도우 "wikilink") 설치된 [구글 크롬](https://ko.wikipedia.org/wiki/구글_크롬 "wikilink"):  + 접근 키
      - [리눅스](https://ko.wikipedia.org/wiki/리눅스 "wikilink")에 설치된 구글 크롬:  + 접근 키
      - [OS X에](https://ko.wikipedia.org/wiki/OS_X "wikilink") 설치된 구글 크롬:  + 접근 키
  - [윈도우에](https://ko.wikipedia.org/wiki/마이크로소프트_윈도우 "wikilink") 설치된 [사파리](https://ko.wikipedia.org/wiki/사파리_\(웹_브라우저\) "wikilink"):  + 접근 키
      - [OS X에](https://ko.wikipedia.org/wiki/OS_X "wikilink") 설치된 사파리:  + 접근 키
      - OS X에 설치된 사파리 3과 그 이전 버전:  + 접근 키

## 단축키 표시

[사용자 CSS 파일을](https://ko.wikipedia.org/wiki/특수:Mypage/common.css "wikilink") 수정함으로써 위키백과 문서의 내부 링크 앞에 단축키를 표시할 수 있습니다.

네모 상자가 있는 단축키 모양으로 표시:

``` css
a[accesskey]:before {
    content: " " attr(accesskey) " ";
    text-transform: uppercase;
    white-space: pre;
    border: thin solid;
    font-family: sans-serif;
    text-decoration: underline overline;
    margin-right: 0.5ex;
}
```

네모 상자가 없는 단축키 모양으로 표시:

``` css
a[accesskey]:before {
    content: " " attr(accesskey) " ";
    text-transform: uppercase;
    white-space: pre;
    font-family: sans-serif;
    margin-right: 0.5ex;
}
```

**8.0 미만의 [인터넷 익스플로러](https://ko.wikipedia.org/wiki/인터넷_익스플로러 "wikilink") 버전에서는 CSS 셀렉터를 지원하지 않으므로 동작하지 않습니다.**

**시각편집기나 [2017 위키텍스트 편집기를](https://ko.wikipedia.org/wiki/mw:2017_wikitext_editor "wikilink") 사용할 때 페이지 게시를 위한 <kbd>s</kbd> 등의 단축키들은 처리 상의 차이로 인해 CSS에 의해 표시되지 않습니다.**
