**김정은의 스튜디오 1059**는 [TBN 강원교통방송](../Page/강원교통방송.md "wikilink")(원주 FM 105.9㎒, 춘천 FM 103.7㎒, 강릉 FM 105.5㎒, 양양 FM 95.1㎒, 동해 FM 95.3㎒, 평창 FM 89.3㎒)에서 평일 오전 9시부터 10시 55분까지 방송되는 강원권 지역의 아침 라디오 프로그램이다.

## 매일 코너

  - 오늘의 마음산책 : 드라마와 영화, 소설 속 좋은 글들을 모아 인생에 대한 이야기를 전하는 시간

## 주간 코너

### 월요일

  - 슬기로운 취미생활 : 세상에 있는 모든 취미를 알아보는 시간(출연 : 이유나 리포터)
  - 그.사.세 : 그들이 찍은 사진 속 세상을 들여다보는 시간으로 청취자들의 핸드폰에 저장된 사진과 사연을 받는 코너

### 화요일

  - 불후의 명곡 : 7080 올드팝과 아티스트들의 이야기를 전하는 코너(전화연결 : 박현준 음악 평론가)
  - 우리집아재 : 들었을 땐 황당하지만 집에가서 다시 생각해보면 재미있는 퀴즈를 풀어보는 시간
  - 마.법.사 : 일상생활 속 크고 작은 분쟁들을 법률로 쉽게 풀어보는 시간(출연 : 김민성 변호사)

### 수요일

  - 맛있는 강원도 이야기 : 강원도 곳곳의 맛있는 음식 이야기를 들어보는 시간(전화연결 : 황영철 푸드스토리텔러)
  - 라떼는 말이야 : 청취자들과 나누고 싶은 추억 속 가요와 그때 그시절 이야기

### 목요일

  - 클래식이 있는 아침 : 위대한 음악가들이 남긴 클래식과 그 속에 담긴 이야기를 들어보는 시간(전화연결 : 김강하 음악 평론가)
  - 강원듣기평가 : 강원도 말(사투리)을 퀴즈로 풀어보는 시간
  - 마음처방전 : 인간관계에서 오는 마음 속 고민을 해결 해 줄 마음의 약을 처방해주는 시간(출연 : 김진겸 심리서사분석가)

### 금요일

  - 고씨네 조조영화 : 영화 소개부터 영화 제작까지 영화에 모든 것을 들어보는 시간(출연 : 고승현 고씨네 대표)
  - 노. 골 적 인디 : 노래 골라주는 인디밴드에게 라이브로 들어보는 팝송과 가요(격주 출연 : 몽키타이거라벨, 해비치 밴드)
