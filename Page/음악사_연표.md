<p><a href="../Page/피타고라스.md" title="wikilink">피타고라스</a>, 수학적 원리에 의한 5도음계 창시.</p>

<p>(∼A.D.100) 크리스트교회, 공동체의 상징으로서 유니즌인 찬송가를 부르다.</p>
<p><a href="https://ko.wikipedia.org/wiki/프톨레마이오스" title="wikilink">프톨레마이오스</a>, 3권의 『음악론』을 저술하다.</p>
<p>암브로시우스, 교회에 찬송가의 사용을 촉진하다.</p>
<p><a href="../Page/아우구스티누스.md" title="wikilink">아우구스티누스</a>,『고백』 『음악론』에서 교부(敎父)의 음악관은 고대와 대조적으로 금욕적이며 비감각적이고 성구(聖句)에 종속된다고 기술하다.</p>
<p>보에티우스, 『음악론』에서 그리스·로마의 전통을 기술하고 음악을 '우주' '인간' '기관(器官)'의 3가지 음악으로 분류하다. 이는 중세의 사변적 음악이론의 원천이 되다.</p>
<p>교황 그레고리우스 1세, 로마교회의 성가를 집대성하여 '그레고리오 성가'를 조직화하다.</p>
<p>세비야의 주교 이시도르, 『백과사전』에 그리스의 자료를 기초로 성악과 악기에 대하여 기술하다.</p>
<p>수도사 보니파티우스, 수도원에 가창(歌唱)학교를 설립하다.</p>
<p>카를대제, 로마교회의 정신에 따라 전례(典禮)를 통일하고 영내의 성가(聖歌)를 보호하다.</p>
<p>귀중한 이론서 『음악개요』나오다.</p>

<p><인도> <a href="../Page/베다.md" title="wikilink">베다</a>의 3음선율에 의한 낭창법 발달하다.</p>

<p><중국> (<a href="https://ko.wikipedia.org/wiki/춘추전국시대" title="wikilink">춘추전국시대</a>, B.C.770∼B.C.221), 5성(聲)과 삼분손익법에 의한 12율이 확립되다.</p>
<p>예악(禮樂)사상에 따라 아악이 발달하다.</p>
<p><인도> 최초의 음악 이론가 바라타(Bharata), 음악·무용·연극에 관한 책 『나티아 샤스트라(Natya Shastra)』를 남기다.</p>
<p><인도> 인도의 2대 서사시 『<a href="../Page/마하바라타.md" title="wikilink">마하바라타</a>』와 『<a href="https://ko.wikipedia.org/wiki/라마야나" title="wikilink">라마야나</a>』에 의한 무용극이 발달하다.</p>
<p><중국> (<a href="../Page/한나라.md" title="wikilink">한나라</a> 때, 前漢 B.C.202∼後漢 A.D.220) 아악을 관장하는 대악서(大樂署)를 설치하다.</p>
<p>&lt;신라 <a href="https://ko.wikipedia.org/wiki/유리왕" title="wikilink">유리왕</a> 5&gt; 민속 환락하여 가악의 시초인 <a href="https://ko.wikipedia.org/wiki/도솔가" title="wikilink">도솔가</a>를 짓다.</p>
<p>&lt;신라 유리왕 9&gt; 팔월보름 6부의 여자들이 가위(嘉俳)놀이를 하며 회소곡(會蘇曲)을 부르다.</p>
<p><중국> (後漢, 22∼220) 불교음악과 백희(百戱)음악의 영향을 받다.</p>

<p>&lt;신라 <a href="https://ko.wikipedia.org/wiki/눌지왕" title="wikilink">눌지왕</a> 37&gt; 각종 악인(樂人) 80인을 배 80척을 마련하여 일본에 파견하다.</p>
<p>&lt;신라 <a href="https://ko.wikipedia.org/wiki/자비왕" title="wikilink">자비왕</a> 1&gt; 낭산하에 살던 <a href="https://ko.wikipedia.org/wiki/백결" title="wikilink">백결</a>(百結)선생, 세모에 거문고로 대악(대樂)을 짓다.</p>
<p>&lt;신라 <a href="../Page/진흥왕.md" title="wikilink">진흥왕</a> 12&gt; 왕이 <a href="../Page/우륵.md" title="wikilink">우륵</a>(于勒)과 그 제자 이문(泥文)을 하림궁으로 불러 음악을 듣다.</p>
<p>&lt;백제 <a href="../Page/위덕왕.md" title="wikilink">위덕왕</a> 1&gt; 일본에 파견된 백제 악인 시덕, 삼근, 계덕 등이 교대하기를 청하다.</p>
<p>&lt;고구려 <a href="../Page/평원왕.md" title="wikilink">평원왕</a> 23&gt; 수(隋) 개황(開皇)초의 7부기(七部伎)에 고구려기도 참여하다.</p>
<p>&lt;백제 <a href="../Page/무왕_(백제).md" title="wikilink">무왕</a> 13&gt; 백제인 미마지(味摩之), 일본에 귀화하여 기악무를 가르치다.</p>
<p>&lt;신라 <a href="https://ko.wikipedia.org/wiki/진덕왕" title="wikilink">진덕왕</a> 5&gt; 음악서에 대사(大舍) 2인을 두다.</p>
<p>&lt;신라 <a href="../Page/문무왕.md" title="wikilink">문무왕</a> 4&gt; 성천 구일 등 28인을 부성(府城)에 보내어 당악을 배우게 하다.</p>
<p>&lt;신라 <a href="../Page/신무왕.md" title="wikilink">신무왕</a> 4&gt; 전정에서 소간전무(小墾田舞) 등 고구려·백제·신라의 삼국악을 아뢰다.</p>
<p>&lt;신라 <a href="../Page/경덕왕.md" title="wikilink">경덕왕</a> 18&gt; <a href="https://ko.wikipedia.org/wiki/월명대사" title="wikilink">월명대사</a>(月明大師), 향가 「<a href="https://ko.wikipedia.org/wiki/도솔가" title="wikilink">도솔가</a>」를 지어 부르다.</p>
