**서산초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [서산시](https://ko.wikipedia.org/wiki/서산시 "wikilink") [동문동에](https://ko.wikipedia.org/wiki/동문동_\(서산시\) "wikilink") 있는 공립 초등학교이다.

## 학교 연혁

| 1911년  | 9월 1일                    | **공립서산보통학교** 개교(설립인가 [7월 5일](https://ko.wikipedia.org/wiki/7월_5일 "wikilink"), 서산면 [읍내리](https://ko.wikipedia.org/wiki/읍내동_\(서산시\) "wikilink") 472) |
| ------ | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| 11월 1일 | **서산공립보통학교**로 개칭(교육령 개정) |                                                                                                                                                    |
| 1933년  | 11월 27일                  | 현 위치로 교사 이전                                                                                                                                        |
| 1938년  | 4월 1일                    | **서산영공립심상소학교**로 개칭                                                                                                                                 |
| 1941년  | 4월 1일                    | **서산영공립국민학교**로 개칭                                                                                                                                  |
| 1946년  | 3월 1일                    | **서산공립국민학교**로 개칭                                                                                                                                   |
| 1949년  | 12월 31일                  | **서산국민학교**로 개칭\[1\]                                                                                                                                |
| 1996년  | 3월 1일                    | **서산초등학교**로 개칭\[2\]                                                                                                                                |
| 2015년  | 9월 1일                    | 제41대 이은미 교장 부임                                                                                                                                     |
| 2016년  | 2월 12일                   | 제104회 졸업식(66명 졸업) 총 27,056명                                                                                                                        |
