**렝겔의 상수**(Lengyel's Constant)\[1\]또는 렌겔 상수의 기호는 \(\Lambda\)이다.

## 유도과정

\[Z_{(n)} =  \sum_{m=1}^{n-1}s(n,m)Z_{(m)}\]

\[s(n,m) =\][제2종 스털링 수](https://ko.wikipedia.org/wiki/스털링_수#제2종_스털링_수 "wikilink")

\[p_{(n)}={ {(n!)^2}  \over {(2 ln 2)^n \; n^{1+{{ln2} \over{3}} } } }\]

\[q_{(n)} = { {Z_{(n)} } \over {  p_{(n)} }    }\]

\[q_{(n)} = { {Z_{(n)} } \over {\left(  { {(n!)^2}  \over {(2 ln 2)^n \; n^{1+{{ln2} \over{3}} } } }  \right)}    }\]\[2\]

\[\;\;\;\;\; = {{Z_{(n)}(2 ln 2)^n \; n^{1+{{ln2} \over{3}}}} \over {(n!)^2}}\]

\[\Lambda= \lim_{n\to \infty}q_{(n)}=1.0986858055....\]\[3\]
