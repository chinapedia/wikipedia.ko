**남제주화력발전소**(南濟州火力發電所)는 [제주특별자치도](https://ko.wikipedia.org/wiki/제주특별자치도 "wikilink") [서귀포시](https://ko.wikipedia.org/wiki/서귀포시 "wikilink") [안덕면](https://ko.wikipedia.org/wiki/안덕면_\(서귀포시\) "wikilink") 화순리에 위치한 화력발전소로, [한국남부발전](https://ko.wikipedia.org/wiki/한국남부발전 "wikilink")에서 운영하고 있다. 한림복합화력, 한경풍력, 성산풍력도 함께 운영하고 있다.

## 발전설비 현황\[1\]

### 남제주화력

위치: 제주특별자치도 서귀포시 안덕면 화순리

| 발전방식    | 발전원       | 설비용량         | 소계 (MW)    |
| ------- | --------- | ------------ | ---------- |
| 기력      | 중유        | 1, 2호기       | 100MW × 2기 |
| 신재생에너지  | 태양광       | 0.196MW × 1기 | 0.196      |
| 설비용량 총계 | 200.196MW |              |            |

### 한림복합화력

위치: 제주특별자치도 [제주시](https://ko.wikipedia.org/wiki/제주시 "wikilink") [한림읍](https://ko.wikipedia.org/wiki/한림읍 "wikilink") 한림리

| 발전방식    | 발전원   | 설비용량                                   | 소계 (MW) |
| ------- | ----- | -------------------------------------- | ------- |
| 복합화력    | 등유    | **G/T:** 35MW × 2기, **S/T:** 35MW × 1기 | 105     |
| 설비용량 총계 | 105MW |                                        |         |

### 한경풍력

위치: 제주특별자치도 제주시 [한경면](https://ko.wikipedia.org/wiki/한경면 "wikilink") 신창리

| 발전방식    | 발전원  | 설비용량                 | 소계 (MW) |
| ------- | ---- | -------------------- | ------- |
| 신재생에너지  | 풍력   | 1.5MW × 4기, 3MW × 5기 | 21      |
| 설비용량 총계 | 21MW |                      |         |

### 성산풍력

위치: 제주특별자치도 서귀포시 [성산읍](https://ko.wikipedia.org/wiki/성산읍 "wikilink") 수산리

| 발전방식    | 발전원  | 설비용량      | 소계 (MW) |
| ------- | ---- | --------- | ------- |
| 신재생에너지  | 풍력   | 2MW × 10기 | 20      |
| 설비용량 총계 | 20MW |           |         |
