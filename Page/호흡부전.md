**호흡 부전**(呼吸不全, respiratory failure)은 [의학](../Page/의학.md "wikilink")에서 [호흡계통](../Page/호흡계통.md "wikilink")에 의한 부적절한 [가스 교환을](https://ko.wikipedia.org/wiki/가스_교환 "wikilink") 가리키는 용어이다. 이 증상이 일어나면 동맥 산소 및 이산화탄소 수준이 통상 범위 안에서 유지할 수 없다. 피의 산소화가 떨어지는 현상을 [저산소증](../Page/저산소증.md "wikilink")이라고 한다. 동맥의 [이산화탄소](https://ko.wikipedia.org/wiki/이산화탄소 "wikilink") 수준이 증가하는 현상을 [과탄산혈증](https://ko.wikipedia.org/wiki/과탄산혈증 "wikilink")이라고 한다. 일반적인 참조값은 산소 [Pa](https://ko.wikipedia.org/wiki/부분압력 "wikilink")[O<sub>2</sub>이](../Page/산소.md "wikilink") 80 mmHg (11 kPa)보다 크고 이산화탄소 Pa[CO<sub>2</sub>가](https://ko.wikipedia.org/wiki/이산화탄소 "wikilink") 45 mmHg(6.0 kPa) 보다 낮은 것과 같다. 이 증상은 타입 I과 타입 II로 나눌 수 있는데 탄소 I은 탄소과잉의 부재와, 탄소 II는 탄소과잉의 존재와 관련이 있다.

## 종류

### 제1형

|                              |                           |
| ---------------------------- | ------------------------- |
| P<sub>a</sub>O<sub>2</sub>   | 낮음 (\< 60 mmHg (8.0 kPa)) |
| P<sub>a</sub>CO<sub>2</sub>  | 일반 또는 낮음                  |
| P<sub>A-a</sub>O<sub>2</sub> | 증가                        |

### 제2형

|                              |                |
| ---------------------------- | -------------- |
| P<sub>a</sub>O<sub>2</sub>   | 감소             |
| P<sub>a</sub>CO<sub>2</sub>  | 증가 (\>50 mmHg) |
| P<sub>A-a</sub>O<sub>2</sub> | 일반             |
| pH                           | 감소             |
