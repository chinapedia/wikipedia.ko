**성거초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [천안시](https://ko.wikipedia.org/wiki/천안시 "wikilink") [서북구](https://ko.wikipedia.org/wiki/서북구 "wikilink") [성거읍](https://ko.wikipedia.org/wiki/성거읍 "wikilink") 송남리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1936년 | 6월 1일              | 성거공립보통학교 개교(설립인가 [5월 11일](https://ko.wikipedia.org/wiki/5월_11일 "wikilink")) |
| ----- | ------------------ | --------------------------------------------------------------------------- |
| 1938년 | 4월 1일              | 성거공립심상소학교로 개칭                                                               |
| 1941년 | 4월 1일              | 성거공립국민학교로 개칭                                                                |
| 1950년 | 6월 1일              | 성거국민학교로 개칭                                                                  |
| 1981년 | 3월 1일              | 병설유치원 개원                                                                    |
| 1982년 | 3월 1일              | 특수학급 개설                                                                     |
| 1996년 | 3월 1일              | 성거초등학교로 개칭                                                                  |
| 2016년 | 3월 1일              | 36학급 편성(특수학급 2 포함)                                                          |
| 2017년 | 2월 14일             | 제76회 졸업(총 10,403명)                                                          |
| 3월 1일 | 36학급 편성(특수학급 2 포함) |                                                                             |
| 2018년 | 2월 9일              | 제77회 졸업(총 10,559명)                                                          |
| 2019년 | 2월 15일             | 제78회 졸업(총 10,705명)                                                          |
| 3월 1일 | 34학급 편성(특수학급 2 포함) |                                                                             |
