**나눗셈**()은 [수학](../Page/수학.md "wikilink")에 [곱셈](../Page/곱셈.md "wikilink")의 [역연산](https://ko.wikipedia.org/wiki/역연산 "wikilink")인 산술 연산이다. 때로는 [빼기를](../Page/뺄셈.md "wikilink") 반복하는 것으로도 생각할 수 있다.

수식으로 표현하면, 만일 \(b\)가 \(0\)이 아닐 때

\[c \times b = a\] 이면,

\[\frac a b = c\] 또는 \(a \div b = c\) 즉, \(a\) 나누기 \(b\) 는 \(c\)이다. 예를 들면, \(\frac 6 3 = 2\) 이므로 \(2 \times 3 = 6\)이다.

위와 같은 식에서, \(a\) 는 **피제수**(dividend)라고 부르고, \(b\) 는 **제수**(divisor) 그리고 \(c\)는 **몫**(quotient)이라고 한다.

제수가 \(0\)일 때의 나눗셈, 즉 [0으로 나누기는](../Page/0으로_나누기.md "wikilink") 보통 정의되지 않는다.

## 쉽게 하는 방법

10÷2=5에서 10은 나누어지는 수, 2는 나누는 수, 5는 몫이다.

## 유리수의 나눗셈

\(b, c, d\) 가 \(0\) 이 아니라는 조건 하에 유리수 \(\frac{a}{b}\) 를 \(\frac{c}{d}\) 로 나누는 식은 \(\frac{a}{b}\) \(\div\) \(\frac{c}{d}\) 와 같이 나타낼 수 있으며
나누는 방법은 제수 \(\frac{c}{d}\) 의 분모와 분자의 위치를 각각 바꾸어 주어서 곱해주면 된다.
즉 \(\frac{a}{b}\) \(\div\) \(\frac{c}{d}\) \(=\) \(\frac{a}{b}\) \(\times\) \(\frac{d}{c}\) 이 식을 계산해주면 된다.

## 나눗셈의 연산

직관적인 나눗셈 연산

\[\;\; 246 \div 2\]

\[\;\;   \begin{array}{r }        1      \end{array} \;\;\]

\[2   \begin{array}{|r } \hline   246      \end{array} \;\;\]

\[\begin{array}{r }    \;\;        2      \end{array} \;\;\]

\[\;\;   \begin{array}{r }        12      \end{array} \;\;\]

\[2   \begin{array}{|r } \hline   246      \end{array} \;\;\]

\[\begin{array}{r }    \;\;        24      \end{array} \;\;\]

\[\;\;   \begin{array}{r }        123      \end{array} \;\;\]

\[2   \begin{array}{|r } \hline   246      \end{array} \;\;\]

\[\begin{array}{r }    \;\;        246      \end{array} \;\;\]

\[\therefore \;\; 246 \div 2 = 123 \;\;\]

## 성질

나눗셈 연산은 분수의 성질을 갖는다.

\[a \div b\]

\[= a / b\]

\[={ a \over b}\] 나눗셈은 몫과 나머지 성질을 갖는다.

## 나머지

나눗셈의 연산에 의해서 즉, 피제수와 제수와의 나눗셈의 결과로 항상 몫과 [나머지](../Page/나머지.md "wikilink")가 생겨난다. 그러나 나머지 값이 인 경우에는 이를 생략한다.

이처럼 나눗셈연산은 나머지에 대한 중요한 성질을 갖는다. 일반적인 나눗셈의 연산의 결과값이 몫인 경우와는 다르게 나머지를 그 결과값으로 하는 나눗셈의 연산을 [나머지](../Page/나머지.md "wikilink")연산이라고 했을때 기호  를 사용한다.

\[9 \bmod 3 = 0 \;,\; 10 \bmod 3 = 1\]
