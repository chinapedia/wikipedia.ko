**ARTBEAT**는 대한민국 대구광역시를 중심으로 활동하는 댄스 커버 팀이다. 현재 멤버는 30명이며, 남자 14명 여자 16명이다.

현재 Youtube 채널 [ARTBEAT](https://www.youtube.com/channel/UCgZlBRLRB1-0l-qL9BkecLQ)를 2016년 3월 7일 개설하여 운영하고 있으며, 2019년 4월 21일 100만 구독자를 달성하였다.

## AB PROJECT

\- ARTBEAT 소속의 전속 댄스팀

\- 첫 영상은 빨간맛 ( 조회수 : 356208회 )

현재 Youtube 채널 [AB PROJECT](https://www.youtube.com/channel/UCaZ2NO5ZQTh1qZBuN9i4xkw/featured)를 운영하고 있다.

원래 멤버 모집은 비정기적으로 모집영상을 통해 신청을 받았으나, ( 하\*영 / 윤\*은 / 정\*주 / 김\*림 / 박\*호 / 김\*희 / 이\*훈 ) 2019년 5월 이후 부터 상시 오디션으로 변경하였으며 이는 6월 16일로 종료되었다. ( 오\*민 )

## 공연

### 콘서트

#### 2018 ARTBEAT CONCERT

2018년 12월 1일(토) 대구와 2018년 12월 2일(일) 서울에서 총 2회에 걸쳐 진행되었다.

추첨을 통해 대구 300명, 서울 400명의 구독자들에게 콘서트 티켓을 보내주는 이벤트도 진행하였으며, 신청서를 받은지 4일만에 신청자가 3,000명을 돌파했다.

당시 콘서트 영상들은 Youtube 채널 AB PROJECT '2018 AB CONCERT' 재생목록에서 확인가능하다.

특별 이벤트로 '감독님'이 방탄소년단(BTS)의 IDOL에서 정국 파트의 대타를 맡아 댄스 커버를 하기도 했다.

#### 미니콘서트 (ARTBEAT MINI CONCERT)

2019년 5월 12일 대구 동성로 중앙무대에서 약 1시간 동안 ARTBEAT MINI CONCERT가 진행되었다.

'AB직캠'이라는 명칭으로 ARTBEAT Youtube 채널에서 공연의 직캠을 볼 수 있다.

#### 2019 ARTBEAT CONCERT

2019년 7월 20일 건국대학교 새천년관 대공연장에서 진행되었다. 티켓비는 콘서트 관람(A) 20000원 , 콘서트 관람 + 하이터치회 35000원 이였고 , 콘서트 당일 날 콘서트 로비에서 여기서요 ? 촬영을 했다. 굿즈스토어에서 포카 팩 ( 10000원 ) / 핀버튼 ( 2000원 ) / 티셔츠 ( 20000원 ) / 하이터치 입장권 ( 15000원 ) 을 팔았다. 팬딩 이벤트로는 응원 글을 남기면 종이 슬로건을 주었고, 멤버쉽을 가입하면 종이 슬로건 + 야광 팔찌를 주었다.

### Riyadh of Saudi Arabia

AB PROJECT가 사우디아라비아 대사관들의 초청을 받아 처음으로 사우디아라비아에서 해외공연을 하였다.

사우디아라비아에서는 여자가 무대에 서는 것이 법으로 금지되어있어, 남자 멤버들만 공연을 진행하였다.
