**윈클론(WinClon)** 은 [클로닉스](https://ko.wikipedia.org/wiki/클로닉스 "wikilink") 에서 개발한 시스템 백업/복원 소프트웨어 제품이다. 정상 동작시 시스템 데이터를 이미지 파일 형태로 백업하여 시스템 이상 또는 디스크의 물리적 손상 시 백업한 파일을 이용하여 시스템을 정상 작동 시킨다.

## 주요 기능

  - Windows O/S 기반의 다양한 컴퓨팅 시스템 백업/복구용
  - Windows XP, Vista, 7, 8, 8.1, 10 지원
  - 시스템 영역과 중요한 데이터를 이미지 파일 형태로 보관하여 시스템 이상 시 백업한 시점으로 안전하게 복구
  - 안전한 복원 영역 지원 : 하드디스크 내에 OS에서 접근이 불가능한 별도 파티션을 복원 영역으로 생성
  - 백업 이미지를 안전하게 보관하여 시스템 오류 시 빠르게 복원
  - 스케줄링 백업/복원 : 매일/매주/매월 사용자가 설정한 주기에 맞추어 자동 백업/복원
  - 부팅 미디어 생성 : 부팅 미디어를 통해 부팅과 복원 가능

## 제품 활용

  - 개인/기업/공공기관 PC의 안전한 관리 & 유지보수
  - 시스템 장애 발생시 IT 전담 부서에서 빠른 복구가 가능
  - 반도체/휴대폰/자동차 등 산업공정용 탑재 컴퓨터 시스템의 안전한 백업/복구

  - PC, POS, ATM, DID 등 IT단말기 유지보수
  - 컴퓨터 및 IT기기 사용에 익숙하지 않은 점주/직원/아르바이트 인원도 쉽게 사용 가능
