**소금물** 또는 **염수**(塩水)는 [용해](../Page/용해.md "wikilink")된 [소금](../Page/소금.md "wikilink") ([염화나트륨](https://ko.wikipedia.org/wiki/염화나트륨 "wikilink"))이 상당히 녹아든 [물](https://ko.wikipedia.org/wiki/물 "wikilink")을 가리킨다.

일반적으로 중간 정도로나 높은 정도로 소금이 녹아 있는 물은 사람에게 거의 쓰이지 않는다. 사람은 소금물을 직접 마시기 어렵고 농작물에 물을 대는 데에도 적합하지 않다. 다만 열전기나 광업과 같은 일부 산업은 소금물을 이용한다.

## 고농도 염수

염수(소금물)은 염분의 [농도](../Page/농도.md "wikilink")가 올라갈수록 [어는점](https://ko.wikipedia.org/wiki/어는점 "wikilink")이 낮아진다. 한편 소금(염분)은 물에 녹아드나 물100ml기준(20도)에서 소금은 약34%(34g)를 전후해서 더 이상 [용해](../Page/용해.md "wikilink")되기 어렵다.

| 염분농도(%) | 동결점(C) |
| ------- | ------ |
| 0       | 0      |
| 1       | \-0.6  |
| 3       | \-1.8  |
| 5       | \-3    |
| 10      | \-6.56 |
| 16      | \-11.8 |
| 20      | \-16.4 |
| 26      | \-19.1 |
