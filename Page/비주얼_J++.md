**마이크로소프트 비주얼 J++**(Microsoft Visual J++)은 [마이크로소프트](../Page/마이크로소프트.md "wikilink")사에서 제공한 [자바](../Page/자바_\(프로그래밍_언어\).md "wikilink") 언어 개발 툴이다. [사용자 인터페이스는](../Page/사용자_인터페이스.md "wikilink") 기존에 MS에서 제공한 개발자 환경과 동일하다. 출시된 지 얼마 지나지 않아 [선 마이크로시스템에서](https://ko.wikipedia.org/wiki/선_마이크로시스템 "wikilink") 상표권 및 저작권 침해를 문제 삼아 현재는 판매되지 않는다. 이후 MS는 비주얼 스튜디오 2005까지 [Visual J\#이라는](https://ko.wikipedia.org/wiki/J_Sharp "wikilink") 이름으로 계속 [닷넷](https://ko.wikipedia.org/wiki/닷넷 "wikilink") 기반 자바 개발 환경을 지원해왔다.

## Visual J++

Visual J++은 자바와 동일하다.

``` java
import com.ms.lang.*;

public class Class1
{
    public void Main()
    {
        System.out.Println("Hello, World!\n");
    }
}
```

## Visual J\#

Visual J\#의 문법은 자바와 유사하지만 [컴파일](https://ko.wikipedia.org/wiki/컴파일 "wikilink")된 [바이너리](https://ko.wikipedia.org/wiki/바이너리 "wikilink")는 [닷넷 프레임 워크](../Page/닷넷_프레임워크.md "wikilink")(.NET Framework)에서 작동된다.

``` csharp
public class Class1
{
    public static void Main()
    {
        System.Console.WriteLine("Hello, World!");
    }
}
```

## Visual C\#

[c\#](../Page/C_샤프.md "wikilink") 은 JAVA를 본따 만든 언어로서 JAVA와 문법이 비슷하다.(고로 J\#과도 비슷하다)

``` csharp
public class Class1
{
    public static void Main()
    {
        System.Console.WriteLine("Hello, World!");
    }
}
```
