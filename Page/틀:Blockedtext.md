<td><p><strong><a href="../Page/위키백과:차단_정책.md" title="wikilink">차단</a></strong>이란 위키백과의 정책과 지침을 위반하는 부적절한 편집으로부터 위키백과를 보호하기 위해, 특정 계정이나 IP의 편집을 막는 것을 말합니다. 차단은 문제가 있는 행동에 책임이 있거나 관계 있는 계정, IP 주소, 또는 넓은 IP 주소 대역에 대해 이루어집니다. 차단에 대한 위키백과의 정책은 <a href="../Page/위키백과:차단_정책.md" title="wikilink">위키백과:차단 정책에서</a> 보실 수 있습니다.</p>
<p>차단에 대한 이의는 <strong><a href="../Page/위키백과:차단_재검토.md" title="wikilink">차단 재검토</a></strong> 절차에 따라 다양한 방법으로 제기하실 수 있습니다.</p>
<p>우선 <strong></strong>을 이용하는 방법이 있습니다. 따로 명시되지 않은 경우, 기본적으로 차단을 당하더라도 <a href="https://ko.wikipedia.org/wiki/특수기능:내사용자토론" title="wikilink">자신의 사용자 토론은</a> 편집이 가능합니다. 자신의 사용자 토론 문서 맨 아래에 를 쓰고 그 밑에 자세한 이유를 쓰면 사용자 토론 문서가 <a href="https://ko.wikipedia.org/wiki/:분류:차단_재검토_요청" title="wikilink">:분류:차단 재검토 요청에</a> 분류되고, 다른 사용자가 이 차단이 정당한지에 대해 토론하게 됩니다. 단 사용자 토론 문서에서도 위키백과의 지침이나 정책을 위반하는 행동을 하면 사용자 토론 문서의 편집마저 차단될 수 있으니, 이의 제기를 하실 때에는 신중히 해 주세요.</p>
<div style="text-align:center;">
</div>
<p>사용자 토론의 편집도 차단되어 위 방법을 사용할 수 없을 경우, <strong><a href="../Page/위키백과:IRC_대화방.md" title="wikilink">IRC 대화방</a></strong>이나 <strong>이메일</strong>을 통해 관리자에게 이의 제기를 할 수도 있습니다. IRC 대화방은 Freenode의 <a href="irc://irc.freenode.net/wikipedia-ko">#wikipedia-ko</a>에 개설되어 있습니다. IRC 클라이언트를 사용하고 있지 않으시다면 <a href="http://webchat.freenode.net/?channels=#wikipedia-ko">프리노드 웹 IRC</a> 등을 사용하여 IRC 대화방에 접속하실 수 있습니다. 이메일은 이메일을 받을 사용자의 사용자 문서나 사용자 토론에 들어가면 도구 모음에 생기는 이메일 보내기 링크를 사용해서 보내시거나, 아래 입력란에 이메일을 받는 사용자의 계정 이름을 입력해서 보내실 수 있습니다. 이메일을 누구에게 보내야 할 지 모르겠다면 <a href="../Page/위키백과:관리자_목록.md" title="wikilink">위키백과:관리자 목록을</a> 참조하세요. 이메일을 부적절하게 사용하는 경우 이메일 기능도 차단될 수 있으니 주의하세요.</p>

<p>다른 사용자의 행위 때문에 시행된 차단이 선의의 사용자의 편집을 막는 경우도 있을 수 있습니다. 이런 일은 차단된 사용자가 사용했던 IP 주소에서 접속했기 때문에 발생합니다. 이런 경우, 차단 이의 제기와 같은 방법으로 자신이 선의의 사용자임을 관리자에게 알려 주세요. 위키백과를 편집하실 수 있도록 관리자가 필요한 조치를 해 드릴 것입니다.</p></td>
