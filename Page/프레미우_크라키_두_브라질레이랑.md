**프레미우 크라키 두 브라질레이랑**()는 [브라질 축구 협회에서](https://ko.wikipedia.org/wiki/브라질_축구_협회 "wikilink") 한 시즌동안 [세리에 A에서](https://ko.wikipedia.org/wiki/캄페오나투_브라질레이루_세리이_A "wikilink") 활약한 선수, 감독 및 심판들에게 시상하는 브라질의 시상식이다.

## 시상 부분

| **베스트 11**     | 2005-현재   |
| **득점상**        | 2005-현재   |
| **감독상**        | 2005-현재   |
| **심판상**        | 2005-현재   |
| 특별상            | 2005-2007 |
| **팬선정 최우수선수상** | 2006-현재   |
| **신인상**        | 2007-현재   |
| 최우수 팬상         | 2007-2010 |
| **페어플레이상**     | 2013-현재   |
| **외국인선수상**     | 2015-현재   |
