**성연초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [서산시](https://ko.wikipedia.org/wiki/서산시 "wikilink") [성연면](https://ko.wikipedia.org/wiki/성연면 "wikilink") 평리에 있는 공립 초등학교이다.

## 학교 연혁

| 1931년    | 7월 18일                        | [성연공립보통학교](https://ko.wikipedia.org/wiki/성연공립보통학교 "wikilink") 개교                                                               |
| -------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| 1938년    | 4월 1일                         | 성연공립심상소학교로 개칭                                                                                                                  |
| 1941년    | 4월 1일                         | 성연공립국민학교로 개칭                                                                                                                   |
| 1947년    | 9월 1일                         | 성연국민학교로 개칭                                                                                                                     |
| 1962년    | 5월 31일                        | 일광분교장, 명봉분교장 설치                                                                                                                |
| 1967년    | 3월 1일                         | 일광국민학교, 명봉국민학교 분리                                                                                                              |
| 1981년    | 3월 1일                         | 급식학교 운영                                                                                                                        |
| 병설유치원 개원 |                               |                                                                                                                                |
| 1993년    | 3월 1일                         | 일광분교장 편입                                                                                                                       |
| 1996년    | 3월 1일                         | 성연초등학교로 개칭                                                                                                                     |
| 1998년    | 9월 1일                         | [일광분교장](https://ko.wikipedia.org/wiki/성연초등학교_일광분교 "wikilink"), [명봉초등학교](https://ko.wikipedia.org/wiki/명봉초등학교 "wikilink") 본교 통합 |
| 2001년    | 3월 1일                         | 초등 7학급 편성                                                                                                                      |
| 2002년    | 3월 1일                         | [성연중학교](../Page/성연중학교.md "wikilink")와 통합 개교                                                                                    |
| 2014년    | 3월 1일                         | 초등 6학급 편성                                                                                                                      |
| 2015년    | 2월 14일                        | 제81회 졸업식(11명)                                                                                                                  |
| 3월 1일    | 초등 6학급, 유치원 1학급 편성            |                                                                                                                                |
| 2016년    | 3월 1일                         | 초등 12학급 편성                                                                                                                     |
| 2017년    | 2월 14일                        | 제83회 졸업식(23명)                                                                                                                  |
| 3월 2일    | 초등 14학급, 유치원 4학급 편성, 현 위치로 이전 |                                                                                                                                |
