<td><div class="description ko">
<p><em>이 파일은 독일 <a href="https://ko.wikipedia.org/wiki/:de:Körperschaft_des_öffentlichen_Rechts" title="wikilink">공법상 사단법인</a>(Körperschaft des öffentlichen Rechts)의 <a href="https://ko.wikipedia.org/wiki/:de:Siegel" title="wikilink">직인</a>(Siegel)이나 로고를 표현하고 있습니다. 독일 <a href="https://ko.wikipedia.org/wiki/:de:Gesetz_über_Urheberrecht_und_verwandte_Schutzrechte" title="wikilink">저작권법</a>(UrhG) 5조 1항에 따라, 직인과 같은 <a href="https://ko.wikipedia.org/wiki/:de:Amtliches_Werk" title="wikilink">공적인 저작</a>(amtliche Werke)은 <strong><a href="https://ko.wikipedia.org/wiki/:de:Gemeinfreiheit" title="wikilink">퍼블릭 도메인</a></strong>(gemeinfrei)입니다.</em></p>
<p>직인은 저작권의 상태와 관계 없이, <strong>이용에 법적인 제약</strong>을 받습니다. 이것은 <a href="https://ko.wikipedia.org/wiki/:de:Namenrechts" title="wikilink">성명권</a>(Namenrechts)이 있을 때에만 사용할 수 있으며 공공 단체의 <a href="https://ko.wikipedia.org/wiki/:de:Hoheitszeichen" title="wikilink">문장</a>(Hoheitszeichen)의 역할을 합니다. 자세한 정보는 <a href="https://ko.wikipedia.org/wiki/:de:Dienstsiegel" title="wikilink">관인</a>(Dienstsiegel)문서를 참조하십시오.</p>
</div>
<hr />
<div class="description de">
<p><em>Dieses Bild stellt das <a href="https://ko.wikipedia.org/wiki/:de:Siegel" title="wikilink">Siegel</a> oder Logo einer <a href="https://ko.wikipedia.org/wiki/:de:Deutschland" title="wikilink">deutschen</a> <a href="https://ko.wikipedia.org/wiki/:de:Körperschaft_des_öffentlichen_Rechts" title="wikilink">Körperschaft des öffentlichen Rechts</a> dar. Nach § 5 Abs. 1 <a href="https://ko.wikipedia.org/wiki/:de:Gesetz_über_Urheberrecht_und_verwandte_Schutzrechte" title="wikilink">UrhG</a> (Deutschland) sind <a href="https://ko.wikipedia.org/wiki/:de:Amtliches_Werk" title="wikilink">amtliche Werke</a> wie Siegel <strong><a href="https://ko.wikipedia.org/wiki/:de:Gemeinfreiheit" title="wikilink">gemeinfrei</a></strong> ("public domain").</em></p>
<p>Siegel sind allgemein unabhängig von ihrem urheberrechtlichen Status in ihrer <strong>Nutzung gesetzlich beschränkt</strong>. Ihre Verwendung unterliegt dem <a href="https://ko.wikipedia.org/wiki/:de:Namensrecht" title="wikilink">Namensrecht</a> und den öffentlichen Körperschaften dienen sie darüberhinaus als <a href="https://ko.wikipedia.org/wiki/:de:Hoheitszeichen" title="wikilink">Hoheitszeichen</a>. Weitere Informationen dazu gibt es unter <a href="https://ko.wikipedia.org/wiki/:de:Dienstsiegel" title="wikilink">Dienstsiegel</a>.</p>
