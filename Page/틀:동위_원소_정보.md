## 변수

| 변수명       | 설명                                                                                                                  |
| --------- | ------------------------------------------------------------------------------------------------------------------- |
| 배경색       | 틀 제목부에 사용될 배경색을 넣습니다. 지정하지 않으면 기본 색상이 사용됩니다.                                                                        |
| 글자색       | 틀 제목부에 사용될 글자색을 넣습니다. 지정하지 않으면 기본 색상이 사용됩니다.                                                                        |
| 그림        | 그림이 있다면 파일명을 적습니다.                                                                                                  |
| 그림설명      | 그림이 있을 때 그림의 설명을 적습니다.                                                                                              |
| 이름        | 틀에 사용될 이름을 적습니다. 기본 값은 {{PAGENAME}}입니다. (예: 수소-3)                                                                   |
| 이름추가      | 대체 이름을 적습니다. 제목에는 표시되지 않고, 이름 항목에만 표시됩니다. (예: 삼중수소, 트리튬)                                                            |
| 질량수       | 동위 원소의 **[질량수](https://ko.wikipedia.org/wiki/질량수 "wikilink")**를 적습니다. 질량과는 다릅니다. (필수) (예: 3)                        |
| 원소기호      | 동위 원소의 원소기호를 적습니다. (필수) (예: H)                                                                                      |
| 중성자수      | 동위 원소에 포함된 중성자 개수를 적습니다. (필수) (예: 2)                                                                                |
| 양성자수      | 동위 원소에 포함된 양성자 개수를 적습니다. (필수) (예: 1)                                                                                |
| 자연존재비     | 자연에 존재하는 비율을 적습니다. 단위는 붙지 않습니다. (예: 미량)                                                                             |
| 반감기       | 동위 원소의 반감기를 적습니다. 단위는 붙지 않습니다. (예: 4500)                                                                            |
| 반감기오차     | 동위 원소의 반감기 수치의 오차를 적습니다. 자동으로 ±이 붙지만, 단위는 붙지 않습니다. (예: 8[일](https://ko.wikipedia.org/wiki/날 "wikilink"))            |
| 부모원소      | 붕괴되기 전의 원소들을 적습니다. 를 사용합니다. (틀 참조)                                                                                  |
| 붕괴생성물     | 방사성 감쇠로 생성되는 물질의 이름을 적습니다. (예: 헬륨)                                                                                  |
| 붕괴질량수     | 방사성 감쇠로 생성되는 물질의 질량수를 적습니다. (예: 3)                                                                                  |
| 붕괴기호      | 방사성 감쇠로 생성되는 물질의 원소기호를 적습니다. (예: He)                                                                                |
| 질량        | 동위 원소의 **[질량](https://ko.wikipedia.org/wiki/질량 "wikilink")**을 적습니다. 질량수와는 다릅니다. 뒤에는 자동으로 단위 u가 붙습니다. (예: 3.0160492) |
| 스핀        | 동위 원소의 [스핀을](../Page/스핀_\(물리학\).md "wikilink") 적습니다. (예: 1/2+)                                                      |
| 질량 과잉     | 동위 원소의 [질량 과잉을](https://ko.wikipedia.org/wiki/질량_과잉 "wikilink") 적습니다. 뒤에는 자동으로 단위 KeV가 붙습니다. (예: 14,949.794)        |
| 질량과잉오차    | 질량 과잉의 오차를 적습니다. 자동으로 ±이 붙습니다. (예: 0.001)                                                                           |
| 결합에너지     | 동위 원소의 [결합 에너지를](https://ko.wikipedia.org/wiki/결합_에너지 "wikilink") 적습니다. 뒤에는 자동으로 단위 KeV가 붙습니다. (예: 8,481.821)       |
| 결합에너지오차   | 결합 에너지의 오차를 적습니다. 자동으로 ±이 붙습니다. (예: 0.004)                                                                          |
| 붕괴과정1\~4  | 동위 원소의 붕괴 과정을 적습니다. 최대 네 개까지 지원합니다. (예: [베타 붕괴](../Page/베타_붕괴.md "wikilink"))                                       |
| 붕괴에너지1\~4 | 동위 원소의 붕괴 에너지를 적습니다. 최대 네 개까지 지원하며, 뒤에는 자동으로 단위 MeV가 붙습니다. (예: 0.018590)                                            |
