**훈민정음의 치두음과 정치음**은 중국어의 다른 두 [잇소리](https://ko.wikipedia.org/wiki/잇소리 "wikilink")를 표기하기 위해 ㅅ, ㅆ, ㅈ, ㅉ, ㅊ의 왼쪽과 오른쪽 획의 길이를 다르게 한 글자를 말한다. [훈민정음](../Page/훈민정음.md "wikilink")에는 없지만, 훈민정음보다 나중에 펴낸 [훈민정음 언해에서](../Page/훈민정음_언해.md "wikilink") 처음 도입되었다.

## 종류

|     |   |   |   |   |   |
| --- | - | - | - | - | - |
| 치음  | ㅅ | ㅆ | ㅈ | ㅉ | ㅊ |
| 치두음 | ᄼ | ᄽ | ᅎ | ᅏ | ᅔ |
| 정치음 | ᄾ | ᄿ | ᅐ | ᅑ | ᅕ |

치두음과 정치음은 각각 현대 음성학의 [치음](https://ko.wikipedia.org/wiki/치음 "wikilink"), [전 경구개 음에](https://ko.wikipedia.org/wiki/경구개음 "wikilink") 해당한다. 닿소리의 이름은 각각 **치두음시옷**, **치두음쌍시옷**, **치두음지읒**, **치두음쌍지읒**, **치두음치읓**, **정치음시옷**, **정치음쌍시옷**, **정치음지읒**, **정치음쌍지읒**, **정치음치읓**이다.
