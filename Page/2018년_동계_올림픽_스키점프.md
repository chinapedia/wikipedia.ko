**[제23회 동계 올림픽](https://ko.wikipedia.org/wiki/제23회_동계_올림픽 "wikilink") [스키 점프](../Page/올림픽_스키점프.md "wikilink")** 경기는 2018년 2월 8일부터 19일까지 [대한민국](../Page/대한민국.md "wikilink") [평창의](https://ko.wikipedia.org/wiki/평창군 "wikilink") [알펜시아 스키점프 경기장에서](https://ko.wikipedia.org/wiki/알펜시아_스키점프_경기장 "wikilink") 4개 세부 종목으로 나뉘어 진행되었다.

## 경기 일정

모든 시간은 [한국 표준시](../Page/한국_표준시.md "wikilink") (UTC +09:00) 기준이다.\[1\]

| 날짜     | 시간          | 종목               | 장소                                                                      |
| ------ | ----------- | ---------------- | ----------------------------------------------------------------------- |
| 2월 8일  | 21:30-22:40 | 노멀힐 남자 개인 예선     | [알펜시아 스키점프 경기장](https://ko.wikipedia.org/wiki/알펜시아_스키점프_경기장 "wikilink") |
| 2월 10일 | 21:35-23:20 | **노멀힐 남자 개인 결승** |                                                                         |
| 2월 12일 | 21:50-23:20 | **노멀힐 여자 개인 결승** |                                                                         |
| 2월 16일 | 21:30-22:40 | 라지힐 남자 개인 예선     |                                                                         |
| 2월 17일 | 21:30-23:15 | **라지힐 남자 개인 결승** |                                                                         |
| 2월 19일 | 21:30-23:15 | **라지힐 남자 단체 결승** |                                                                         |
