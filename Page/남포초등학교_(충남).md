**남포초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [보령시](../Page/보령시.md "wikilink") [남포면](../Page/남포면.md "wikilink") 읍내리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1924년 | 4월 17일            | 남포공립보통학교 4년제 개교                              |
| ----- | ----------------- | -------------------------------------------- |
| 1926년 | 4월 1일             | 6년제 개편 인가                                    |
| 1938년 | 4월 1일             | 남포공립심상소학교 개칭                                 |
| 1941년 | 4월 1일             | 남포공립국민학교 개칭                                  |
| 1947년 | 4월 1일             | [월전국민학교](../Page/월전초등학교.md "wikilink") 승격 분리 |
| 8월 1일 | 남포국민학교로 개칭        |                                              |
| 1981년 | 3월 9일             | 병설유치원 개원                                     |
| 1996년 | 3월 1일             | 남포초등학교로 개칭                                   |
| 2017년 | 2월 10일            | 제90회 졸업식(11명 졸업, 총 7,611명)                   |
| 3월 1일 | 7학급 편성(병설유치원 1학급) |                                              |
