**메타폰트**(Metafont)는 [벡터 글꼴](../Page/외곽선_글꼴.md "wikilink") 을 정의하는 데 사용되는 설명 언어 중 하나이다. 메타폰트 코드를 실행하는 [인터프리터](../Page/인터프리터.md "wikilink")의 이름이기도 한데, 예를 들어 [포스트스크립트](../Page/포스트스크립트.md "wikilink")(PostScript)에 [임베디드할](../Page/임베디드_시스템.md "wikilink") 수 있는 [비트맵](https://ko.wikipedia.org/wiki/비트맵 "wikilink") [글꼴](../Page/글꼴.md "wikilink")을 생성한다. 메타폰트는 [도널드 커누스](https://ko.wikipedia.org/wiki/도널드_커누스 "wikilink")(Donald Knuth)에 의해 [TeX](../Page/TeX.md "wikilink") 조판 시스템 상에서 작동되도록 고안되었다.

메타폰트의 특징 중 하나는 글리프([자체](../Page/자체.md "wikilink"))의 모든 모양이 기하학적 방정식으로 정의된다는 것이다. 특히, 주어진 점을 선분과 [베지어 곡선의](https://ko.wikipedia.org/wiki/베지어_곡선 "wikilink") 교점으로 정의할 수 있다.

## 예

다음 예제는 글꼴의 문자 "B"에 대해 눕혀지고 닫힌 [콩](../Page/콩.md "wikilink")모양을 만든다.

``` postscript
%file name: beta.mf
%mode_setup;
% Define a beanlike shape for the character B
beginchar("B",11pt#,11pt#,0);
  % Setup coordinates as an equation system
  y1=y2=y3=0;
  y4=y5=y6=h;
  x1=x4=0;
  x2=x5=w;
  x3=x6=2*w;

  % Define pen
  pickup pencircle xscaled 0.2w yscaled 0.04w rotated 45;

  % Draw the character curve
  draw z1..z3..z6{z2-z6}..z5..{z4-z2}z4..cycle;
endchar;

end
```

그러면 다음과 같은 글리프가 생성된다.

[파일:metafont-example-output.svg](https://ko.wikipedia.org/wiki/파일:metafont-example-output.svg "wikilink")

위의 예제는 다음과 같은 명령 행을 사용하여 처리된다.\[1\]

    mf '\mode=ljfour; mode_setup; input beta.mf'; gftopk beta.600gf beta.600pk

그런 다음 다음과 같은 [LaTeX](../Page/LaTeX.md "wikilink") 파일에서 사용할 수 있다 .(모든 파일은 같은 디렉토리에 있어야한다. 또는 적절한 방법을 사용하여 TeX 시스템이 이 정보를 참조할수있게해야 한다)

``` latex
\documentclass{article}

\newfont{\letterbeta}{beta}
\newcommand{\otherbeta}{{\letterbeta B}}

\begin{document}
Let’s try having a strange \otherbeta\ here.
\end{document}
```
