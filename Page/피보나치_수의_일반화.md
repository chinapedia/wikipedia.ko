**피보나치 수**는 다양한 형태로 **일반화**될 수 있다.

## 비슷한 수열들

### 뤼카 수

피보나치 수의 일반화인 [뤼카 수는](https://ko.wikipedia.org/wiki/뤼카_수 "wikilink") 다음과 같다.

  -
    *U*(0) = 0
    *U*(1) = 1
    *U*(*n* + 2) = *PU*(*n* + 1) − *QU*(*n*)

피보나치 수열은 *P* = 1 이고 *Q* = −1인 특수한 경우이다.

### 트리보나치 수

**트리보나치\[1\] 수**는 다음의 점화식으로 정의된다.

\[T_n :=
  \begin{cases}
    0             & \mbox{if } n = 0; \\
    0             & \mbox{if } n = 1; \\
    1             & \mbox{if } n = 2; \\
    T_{n-1}+T_{n-2}+T_{n-3} & \mbox{if } n > 2. \\
   \end{cases}\] 일반적으로 트리보나치 수는 0, 0, 1로 시작하며, 다음 트리보나치 수는 바로 앞의 세 트리보나치 수의 합이 된다.

n=0, 0, 1로 시작하는 트리보나치 수는\[2\]

  -
    [0](../Page/0.md "wikilink"), [0](../Page/0.md "wikilink"), [1](../Page/1.md "wikilink"), [1](../Page/1.md "wikilink"), [2](../Page/2.md "wikilink"), [4](../Page/4.md "wikilink"), [7](../Page/7.md "wikilink"), [13](../Page/13.md "wikilink"), [24](../Page/24.md "wikilink"), [44](https://ko.wikipedia.org/wiki/44 "wikilink"), [81](https://ko.wikipedia.org/wiki/81 "wikilink"), [149](https://ko.wikipedia.org/wiki/149 "wikilink"), [274](https://ko.wikipedia.org/wiki/274 "wikilink"), [504](https://ko.wikipedia.org/wiki/504 "wikilink"), [927](https://ko.wikipedia.org/wiki/927 "wikilink"), 1705, 3136, 5768, 10609, ...

이다.

### 테트라나치 수

**테트라나치\[3\] 수**는 다음의 전개식으로 정의된다.

\[T_n :=
  \begin{cases}
    0             & \mbox{if } n = 0; \\
    0             & \mbox{if } n = 1; \\
    0             & \mbox{if } n = 2; \\
    1             & \mbox{if } n = 3; \\
    T_{n-1}+T_{n-2}+T_{n-3}+T_{n-4} & \mbox{if } n > 3. \\
   \end{cases}\] 일반적으로 테트라나치 수는 0, 0, 0, 1로 시작되며, 다음 테트라나치 수는 바로 앞의 네 테트라나치 수의 합이 된다.

n=0, 0, 0, 1,...로 시작하는 테트라나치 수는\[4\]

  -
    [0](../Page/0.md "wikilink"), [0](../Page/0.md "wikilink"), [0](../Page/0.md "wikilink"), [1](../Page/1.md "wikilink"), [1](../Page/1.md "wikilink"), [2](../Page/2.md "wikilink"), [4](../Page/4.md "wikilink"), [8](../Page/8.md "wikilink"), [15](../Page/15.md "wikilink"), [29](https://ko.wikipedia.org/wiki/29 "wikilink"), [56](https://ko.wikipedia.org/wiki/56 "wikilink"), [108](../Page/108.md "wikilink"), [208](https://ko.wikipedia.org/wiki/208 "wikilink"), [401](https://ko.wikipedia.org/wiki/401 "wikilink"), [773](https://ko.wikipedia.org/wiki/773 "wikilink"), 1490, 2872, 5536, 10671, 20569, ...

이다.

### 펜타나치 수

**펜타나치 수**는 [수학](../Page/수학.md "wikilink")에서 아래 [점화식](../Page/점화식.md "wikilink")으로 정의되는 수열로, [피보나치 수의](../Page/피보나치_수.md "wikilink") 확장이다.

\[P_n :=
  \begin{cases}
    0             & \mbox{if } n = 0; \\
    0             & \mbox{if } n = 1; \\
    0             & \mbox{if } n = 2; \\
    0             & \mbox{if } n = 3; \\
    1             & \mbox{if } n = 4; \\
    P_{n-1}+P_{n-2}+P_{n-3}+P_{n-4}+P_{n-5} & \mbox{if } n > 4. \\
   \end{cases}\]

### 헥사나치 수

헥사나치 수(Hexanacci numbers)는 수학에서 다음의 점화식으로 정의되는 수열로, 피보나치 수의 확장이다.

\[T_n :=
\begin{cases}
 0        &\mbox{if } n=0;\\
 0        &\mbox{if } n=1;\\
 0        &\mbox{if } n=2;\\
 0        &\mbox{if } n=3;\\
 0        &\mbox{if } n=4;\\
 1        &\mbox{if } n=5;\\
 T_{n-1}+T_{n-2}+T_{n-3}+T_{n-4}+T_{n-5}+T_{n-6}&\mbox{if } n>5.
\end{cases}\] 헥사나치 수는 0, 0, 0, 0, 0, 1로 시작하며, 다음 헥사나치 수는 바로 앞의 여섯 헥사나치 수의 합이 된다. n=0, 0, 0, 0, 0, 1...로 시작하는 헥사나치 수는 (OEIS의 수열 A001592) 0, 0, 0, 0, 0, 1, 1, 2, 4, 8, 16, 32, 63, 125, 248, 492, 976, 1936... 이다.

### 그 외의 n-나치 수

이 외에도 펜타나치 수, 헥사나치 수, 헵타나치 수 등이 있다. n-나치 수들의 공통적인 특징은

  - \(n-1\)개의 0과 1개의 1로 시작한다.
  - 다음 수는 바로 앞의 n개의 수의 합이 된다.

또한 \(n+2\)번째 수부터 \(2n\)번째 수까지는 2의 거듭제곱인 수들이다.
