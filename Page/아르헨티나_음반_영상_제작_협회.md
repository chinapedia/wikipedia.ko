**아르헨티나음반영상제작협회**(영어:Argentine Chamber of Phonograms and Videograms Producers, 스페인어:Cámara Argentina de Productores de Fonogramas y Videogramas, CAPIF)는 아르헨티나의 음악산업을 대표하는 [국제음반산업협회](https://ko.wikipedia.org/wiki/국제음반산업협회 "wikilink") 기구이다. 이 협회는 다국적 회사와 독립 음반 회사들이 합쳐진 [비영리 단체이다](https://ko.wikipedia.org/wiki/비영리_단체 "wikilink").\[1\]

## 판매량 기준

CAPIF는 1980년에 골드와 플래티넘 인증 기준을 마련했다. 초기에 앨범 판매량 기준은 30,000장 골드, 50,000장 플래티넘이였다. 하지만 2001년 음악 산업이 쇠퇴함에 따라 인증 기준을 낮췄다.\[2\]

**앨범**

| 인증    | 2001년 1월 1일 이전\[3\] | 2001년 1월 1일 이후\[4\] |
| ----- | ------------------- | ------------------- |
| 골드    | 30,000              | 20,000              |
| 플래티넘  | 60,000              | 40,000              |
| 다이아몬드 | 500,000             | 250,000             |

**싱글**

| 인증   | 2001년 1월 1일 이전 피지컬 싱글\[5\] | 디지털 다운로드: 2009년 이후\[6\] |
| ---- | -------------------------- | ----------------------- |
| 골드   | 50,000                     | 10,000                  |
| 플래티넘 | 100,000                    | 20,000                  |
|      |                            |                         |

**비디오**

| 인증      | 초기\[7\] | 2011년 이후\[8\] |
| ------- | ------- | ------------- |
| 골드      | 4,000   | 7,500         |
| 플래티넘    | 8,000   | 15,000        |
| Diamond |         | 75,000        |

**컴필레이션 음반**

| 인증   | 초기\[9\] | 2001년 1월 1일 이후\[10\] |
| ---- | ------- | -------------------- |

|      |         |                      |
