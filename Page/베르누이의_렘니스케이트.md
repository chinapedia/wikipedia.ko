[기하학](../Page/기하학.md "wikilink")에서, **베르누이의 렘니스케이트**()는 거리가 *2a*인 두 초점*F*<sub>1</sub> , *F*<sub>2</sub>가 주어졌을 때 곡선상의 각각의 점 *P*에 대해 *PF*<sub>1</sub>·*PF*<sub>2</sub> = *a*<sup>2</sup>을 만족하는 평면곡선으로 정의된다. 이 곡선의 모양은 숫자 8 또는 기호 [∞와](https://ko.wikipedia.org/wiki/무한대 "wikilink") 유사하며 그 이름은 에서 유래했는데 이는 “펜던트 리본”이라는 뜻이다. 이 곡선은 [카시니의 난형선의](../Page/카시니의_난형선.md "wikilink") 특수한 경우이며 유리곡선이자 [4차 대](https://ko.wikipedia.org/wiki/4차_곡선 "wikilink")[수 곡선이다](../Page/대수_곡선.md "wikilink").

  - 이것의 [직교좌표계](https://ko.wikipedia.org/wiki/직교좌표계 "wikilink")상의 [방정식](../Page/방정식.md "wikilink")은 :

\((x^2 + y^2)^2 = 2a^2 (x^2 - y^2).\,\)

  - [극좌표](https://ko.wikipedia.org/wiki/극좌표 "wikilink")상에서는 : \(r = a \sqrt{2 \cos(2\varphi)}\)

<!-- end list -->

  - [매개변수](../Page/매개변수.md "wikilink")[방정식](../Page/방정식.md "wikilink")으로는 : \(x = \frac{a\sqrt{2}\cos(t)}{\sin(t)^2 + 1}; \qquad y = \frac{a\sqrt{2}\cos(t)\sin(t)}{\sin(t)^2 + 1}\)

렘니스케이트는 [타원](../Page/타원.md "wikilink")의 변형으로서 1694년 [야코프 베르누이에](../Page/야코프_베르누이.md "wikilink") 의해 처음 고안되었다. [타원](../Page/타원.md "wikilink")은 두 초점으로부터 거리의 *합*이 일정한 곡선이다. 반면에, [카시니의 난형선은](../Page/카시니의_난형선.md "wikilink") 두 초점으로부터 거리의 *곱*이 일정한 곡선이다. 이때 이 곡선이 두 초점의 중점을 지나는 경우가 바로 베르누이의 렘니스케이트이다.

이 [렘니스케이트](https://ko.wikipedia.org/wiki/렘니스케이트 "wikilink")는 중심이 쌍곡선의 중심과 일치하는 반전원에 대한 쌍곡선의 [반전형으로도](../Page/반전기하학.md "wikilink") 얻을 수 있다.

## 다른 방정식 표현들

렘니스케이트는 아래의 [극좌표](https://ko.wikipedia.org/wiki/극좌표 "wikilink") 방정식으로도 표현이 가능하다.

\[r^2 = 2a^2 \cos 2\theta\,\]

또는 아래의 [쌍극좌표계](https://ko.wikipedia.org/wiki/쌍극좌표계 "wikilink") 방정식으로도 표현된다.

\[rr' = \frac{a^2}{2}\]

## 미분

### \(x\)에 대한 \(y\)의 함수로서

\[\frac{dy}{dx} = \begin{cases}
\mbox{unbounded} & \mbox{if } y = 0 \mbox{ and } x \ne 0 \\
\pm1 & \mbox{if } y = 0 \mbox{ and } x = 0 \\
\frac{x(a^2 - x^2 - y^2)}{y(a^2 + x^2 + y^2)}  & \mbox{if } y \ne 0
\end{cases}\]

\[\frac{d^2y}{dx^2} = \begin{cases}
\mbox{unbounded} & \mbox{if } y = 0 \mbox{ and } x \ne 0 \\
0 & \mbox{if } y = 0 \mbox{ and } x = 0 \\
\frac{3a^6(y^2 - x^2)}{y^3(a^2 + 2x^2 + 2y^2)^3}  & \mbox{if } y \ne 0
\end{cases}\]

### \(y\)에 대한 \(x\)의 함수로서

\[\frac{dx}{dy} = \begin{cases}
\mbox{unbounded} & \mbox{if } 2x^2 + 2y^2 = a^2 \\
\pm1 & \mbox{if } x = 0 \mbox{ and } y = 0 \\
\frac{y(a^2 + 2x^2 + 2y^2)}{x(a^2 - 2x^2 - 2y^2)}   & \mbox{else }
\end{cases}\]

\[\frac{d^2x}{dy^2} = \begin{cases}
\mbox{unbounded} & \mbox{if } 2x^2 + 2y^2 = a^2 \\
0 & \mbox{if } x = 0 \mbox{ and } y = 0 \\
\frac{3a^6(x^2 - y^2)}{x^3(a^2 - 2x^2 - 2y^2)^3}  & \mbox{else }
\end{cases}\]
