**9회 TV 바둑 아시아 선수권대회**는 [대한민국](https://ko.wikipedia.org/wiki/대한민국 "wikilink"), [중국](https://ko.wikipedia.org/wiki/중국 "wikilink"), [일본](https://ko.wikipedia.org/wiki/일본 "wikilink")의 속기전 상위 입상자가 참가한 바둑 기전이다. 경상북도 경주시 [호텔현대 경주에서](https://ko.wikipedia.org/wiki/현대호텔 "wikilink") 개최되었으며 결승에서는 [중국](https://ko.wikipedia.org/wiki/중국 "wikilink")의 [위빈](https://ko.wikipedia.org/wiki/위빈 "wikilink") 九단이 [일본](https://ko.wikipedia.org/wiki/일본 "wikilink")의 [왕리청](../Page/왕리청.md "wikilink") 九단을 꺾고 중국이 최초로 우승했다.

## 대회 일정

| 구분             | 일정     | 장소                                                                |
| -------------- | ------ | ----------------------------------------------------------------- |
| 개막식            | 5월 11일 | 경상북도 경주시 [호텔현대 경주](https://ko.wikipedia.org/wiki/현대호텔 "wikilink") |
| 1회전 1\~2국      | 5월 12일 |                                                                   |
| 1회전 3국, 준결승 1국 | 5월 13일 |                                                                   |
| 준결승 2국         | 5월 14일 |                                                                   |
| 결승전, 시상식 및 폐막식 | 5월 15일 |                                                                   |
|                |        |                                                                   |

## 상금

  - 우승 250만엔(한화 약 1,900만원), 준우승 50만엔(한화 약 380만원)

  - [중국](https://ko.wikipedia.org/wiki/중국 "wikilink"): [녜웨이핑](https://ko.wikipedia.org/wiki/녜웨이핑 "wikilink")(11기 [CCTV배](https://ko.wikipedia.org/wiki/중신은행배 "wikilink") 우승자), [위빈](https://ko.wikipedia.org/wiki/위빈_\(바둑_기사\) "wikilink")(11기 [CCTV배](https://ko.wikipedia.org/wiki/중신은행배 "wikilink") 준우승자)
