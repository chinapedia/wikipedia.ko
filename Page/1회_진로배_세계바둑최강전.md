**1회 진로배 세계바둑최강전**은 [대한민국](../Page/대한민국.md "wikilink"), [중국](../Page/중국.md "wikilink"), [일본](../Page/일본.md "wikilink")의 한국이 우승했다.

## 대회 일정

| 구분        | 일정                             | 장소          |
| --------- | ------------------------------ | ----------- |
| 개막식       |                                | 서울 힐튼호텔 국화룸 |
| 1차전       | 1992년 12월 12일 \~ 1992년 12월 15일 |             |
| 2차전       | 1993년 1월 12일 \~ 1992년 1월 18일   |             |
| 3차전       | 1993년 2월 25일 \~ 1992년 2월 26일   |             |
| 시상식 및 폐막식 |                                |             |
|           |                                |             |

## 참가 선수 명단

  - [일본](../Page/일본.md "wikilink"): 미야자와 고로, 요다 노리모토, 아와지 슈조, 린하이펑, 다케미야 마사키
  - [대한민국](../Page/대한민국.md "wikilink"): [유창혁](../Page/유창혁.md "wikilink"), [장수영](../Page/장수영.md "wikilink"), [이창호](../Page/이창호.md "wikilink"), [서봉수](../Page/서봉수.md "wikilink"), [조훈현](../Page/조훈현.md "wikilink")
  - [중국](../Page/중국.md "wikilink"): 샤오 웨이강, 위빈, 차오다위안, 류사오광, 마샤오춘
