**란다우 토션트 상수**(Landau's totient constant)\[1\]

\[C_{Lt} = \prod_{p} \left(1 + {{1}\over{p(p-1)} } \right) = {{315}\over{2\pi^4} }\zeta(3) = 1.943596436820759205057070...\]

\[C_{Lt} = {{315}\over{2\pi^4} }\zeta(3)  = {{1}\over{{\zeta(2) \zeta(3)}\over{\zeta(6)}} }= {{\zeta(2) \zeta(3)}\over{\zeta(6)}} = \sum_{s=1}^{\infty} {{(\mu(s))^2}\over{s \phi(s)}} \;\;\;\]\[2\]\[3\]\[4\]

\[\zeta(p)\][리만 제타 함수](../Page/리만_제타_함수.md "wikilink")\(, \;\; \mu(s)\) [뫼비우스 함수](../Page/뫼비우스_함수.md "wikilink")\(, \;\;\phi(s)\) [오일러 피 함수](../Page/오일러_피_함수.md "wikilink")

## 알틴 상수와 란다우 토션트 상수

\[C_{Lt} =\] 란다우 토션트 상수

\[C_{Lt} = \prod_{p} \left(1 + {{1}\over{p(p-1)} } \right)\]

\[\;\;\; = \prod_{p} \left({{p^2-p+1}\over{p^2-p} } \right)\]

\[\;\;\; = \prod_{p} \left({{p^2-p}\over{p^2-p} } \right) + \left({{1}\over{p^2-p} } \right)\]

\[C_{A} =\] [알틴 상수](../Page/알틴_상수.md "wikilink")

\[C_{A}= \prod_{p}^{}  \left( 1-{{1}\over{p(p -1)}}\right)\]

\[\;\;\;= \prod_{p}^{} \left({{p^2 -p-1}\over{p^2 -p}}\right)\]

\[\;\;\; = \prod_{p} \left({{p^2-p}\over{p^2-p} } \right) - \left({{1}\over{p^2-p} } \right)\]

\[C_{Lt} =  C_{A} +  \left({{1}\over{p^2-p} } \right) +  \left({{1}\over{p^2-p} } \right)\]

\[C_{A} =  C_{Lt} -  \left({{1}\over{p^2-p} } \right) -  \left({{1}\over{p^2-p} } \right)\]
