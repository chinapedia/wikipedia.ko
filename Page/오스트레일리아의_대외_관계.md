<td><p>이 나라는 1949년 8월 14일, 대한민국을 승인하고,[1] <a href="../Page/한국_전쟁.md" title="wikilink">한국 전쟁에</a> 참전하였다. <a href="../Page/베트남_전쟁.md" title="wikilink">베트남 전쟁</a> 기간 동안에도 양국이 같이 참전했다. 오스트레일리아군은 <a href="../Page/한국_전쟁.md" title="wikilink">한국 전쟁</a> 때 339명이 전사했고, 베트남 전쟁 때 520명이 전사했다. 한국인은 대사관 가족을 포함하여 10만 5558명이 오스트레일리아에 거주하고 있으며 이들 중 대부분은 <a href="../Page/뉴사우스웨일스주.md" title="wikilink">뉴사우스웨일스주</a>에 거주한다. <a href="https://ko.wikipedia.org/wiki/인천국제공항" title="wikilink">인천국제공항</a>에서 호주까지 가는 노선은 <a href="https://ko.wikipedia.org/wiki/대한항공" title="wikilink">대한항공</a>이 매일 논스톱 직항편으로 운행되고 <a href="https://ko.wikipedia.org/wiki/아시아나항공" title="wikilink">아시아나항공</a>과 <a href="../Page/콴타스_항공.md" title="wikilink">콴타스 항공도</a> 한국으로 논스톱 직항편을 운항한다. 현재는 <a href="../Page/조선민주주의인민공화국.md" title="wikilink">북한과도</a> 수교하고 있는 상태이다. <a href="https://ko.wikipedia.org/wiki/상상원정대" title="wikilink">상상원정대</a>가 오스트레일리아를 방문한 적이 있으며, 또한 오스트레일리아 국적자인 <a href="../Page/샘_해밍턴.md" title="wikilink">샘 해밍턴이</a> MBC <a href="https://ko.wikipedia.org/wiki/진짜_사나이" title="wikilink">진짜 사나이</a>,<a href="https://ko.wikipedia.org/wiki/슈돌" title="wikilink">슈돌</a> 에 출연하고 있다. 샘 해밍턴은 한국으로 이주하여 지금까지 살고있으며 한국 연예인 생활을 하고 있다.</p></td>

<td><p>미국과는 1940년[2]에 수교하였다. 오스트레일리아와 미국은 친밀한 동맹국이다.[3]오스트레일리아는 전통적으로 영국연방 가맹국들과 유대관계를 유지하여 왔다. 그러나, 영국의 아시아에 대한 영향력이 약화되자 1942년 이후, 오스트레일리아는 미국과의 관계를 강화하고 현재와 같이 미국의 동맹국으로서의 지위를 확보하였다. 정부 차원에서 미국과 오스트레일리아는 ANZUS 조약과 AUSFTA를 맺고 있다.</p></td>
