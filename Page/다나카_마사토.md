[오른쪽](https://ko.wikipedia.org/wiki/파일:Masato_Tanaka.jpg "wikilink") **다나카 마사토**(/Masato Tanaka, 1973년 2월 28일 \~ )는 [일본](../Page/일본.md "wikilink")의 [프로레슬링선수](https://ko.wikipedia.org/wiki/프로레슬링선수 "wikilink")다. 키는 181cm(5ft 11in) 몸무게는 90kg(198lb)이다. 1993년 프로레슬링 입문으로 밝혀진다.

## 프로페셔널 레슬링 하이라이트스 / Professional wrestling highlights

  - **피니싱 무브 / Finishing moves**
      - *컴플릿 더스트* (싯아웃 인버티드 사이드 슬램) / *Complete Dust* (Sitout inverted side slam)
      - 단간 밤 - 이너베이터 / Dangan Bomb - innovator
      - *단간 스피닝 엘보우* / *마사토 엘보우* (스피닝 섬머솔트 엘보우 스매쉬) / *Dangan Spinning Somersault Elbow* / *Masato Elbow* (Spinning somersault elbow smash)
      - *다이아몬드 더스트* (인버티드 커터) - 이너베이터 / *Diamond Dust* (inverted cutter) - innovator
      - *썬더 파이어 파워밤* (원 숄더 파워밤) - 어답터드 바이 [온이타 아츠시](https://ko.wikipedia.org/wiki/온이타_아츠시 "wikilink") / *Thunder Fire Powerbomb* (One shoulder powerbomb) - adopted by [Atsushi Onita](https://ko.wikipedia.org/wiki/Atsushi_Onita "wikilink")
      - 런닝 인버티드 토네이도 디디티 섬타임즈 인 어 더벌 언더훅 버전 / Running inverted tornado DDT sometimes in a double underhook version
  - **시그니처 무브** / **Signature moves**
      - 체어 샷 / Chair shot
      - *다이몬드 토네이도 사이드 슬램* (파이어맨즈 캐리 사이드 슬램) / *Diamond Tornado Side slam* (Fireman's carry side slam)
      - *라 터페이셔* (모디파이드 옥스퍼드 홀드) / *La Tapatia* (Modified octopus hold)
      - 멀티플 수플렉스 베리에이션 / Multiple suplex variations
          - 벨리 투 백 / Belly to back
          - 풀 넬슨 / Full nelson
          - 저먼 / German
          - 슈퍼 / Super
      - 런닝 인버티드 데스 벨리 드라이버 Running inverted death Valley driver
      - 싯아웃 버티컬 수플렉스 파워슬램 / Sitout vertical suplex powerslam
      - 섬머솔트 래리어트 / Somersault lariat
      - 스피닝 빅 붓 어 브레인버스터 / Spinning big boot a brainbuster
      - *슈퍼 플라이* (다이빙 프로그 스플래쉬) / *Super Fly* (Diving frog splash)
