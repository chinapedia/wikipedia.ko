**할당자**(割當子) 또는 **얼로케이터**()는 [C++](https://ko.wikipedia.org/wiki/C++ "wikilink")의 [표준 라이브러리에서](../Page/표준_라이브러리.md "wikilink") 메모리 할당을 관리하는 객체이다. C++표준 라이브러리에서 이용되는 [리스트](../Page/리스트_\(컴퓨팅\).md "wikilink"), [셋](https://ko.wikipedia.org/wiki/셋 "wikilink") 등의 다양한 [컨테이너](../Page/컨테이너.md "wikilink") [자료구조](https://ko.wikipedia.org/wiki/자료구조 "wikilink")들은 용도에 따라 [동적 메모리 할당을](../Page/동적_메모리_할당.md "wikilink") 요구한다. **할당자**는 동적 메모리 할당을 통합 관리하는데 필요한 기능이 정의된 객체이다.

#include <iostream>

using namespace __gnu_cxx;

class RequiredAllocation
{
public:
    RequiredAllocation ();
    ~RequiredAllocation ();
    std::basic_string<char> s = "hello world!\n";
};

RequiredAllocation::RequiredAllocation ()
{
    cout << "RequiredAllocation::RequiredAllocation()" << endl;
}
RequiredAllocation::~RequiredAllocation ()
{
    cout << "RequiredAllocation::~RequiredAllocation()" << endl;
}





void alloc(__gnu_cxx ::new_allocator<RequiredAllocation>* all, unsigned int size, void* pt, RequiredAllocation* t){
    try
        {
            all->allocate (size, pt);
            cout << all->max_size () << endl;
            for (auto& e : t->s)
                {
                    cout << e;
                }
        }
    catch (std::bad_alloc& e)
        {
            cout << e.what () << endl;
        }
}

int
main ()
{

            new __gnu_cxx ::new_allocator<RequiredAllocation> ();

    return 0;
