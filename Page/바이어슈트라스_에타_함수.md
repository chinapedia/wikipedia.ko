**바이어슈트라스 에타 함수**(Weierstrass Eta Function) \(\eta \;\)는

[홀함수](../Page/홀함수와_짝함수.md "wikilink")(odd function)의 성질을 갖는 [바이어슈트라스 제타 함수](../Page/바이어슈트라스_제타_함수.md "wikilink")(Weierstrass Eta Function)\(\zeta(z)\)와

  -
    [짝함수](../Page/홀함수와_짝함수.md "wikilink")(even function)의 성질을 갖는[바이어슈트라스 타원 함수를](https://ko.wikipedia.org/wiki/바이어슈트라스_타원_함수 "wikilink") 연관 시킬때,
    \(\wp (z+ 2 \omega_1)= \wp(z)\)로 부터
    \(\zeta (z+ 2 \omega_1)= \zeta(z) + 2 \eta_1\) 에서,

보여지는 [특수 함수이다](../Page/특수_함수.md "wikilink").

\[z =  -\omega_1\]를 예약하면,

\[\zeta(z) + 2 \eta_1 = \zeta(-\omega_1) + 2 \eta_1\]

\[\zeta(-\omega_1) + 2\eta_1 = -\zeta(\omega_1)+2 \eta_1\] 이고, 그리고,

\[\zeta (z+ 2 \omega_1)= \zeta (z)+ 2 \zeta ( \omega_1)= \zeta (-\omega_1)+ 2 \zeta ( \omega_1)\] 따라서,

\[\zeta (-\omega_1)+ 2 \zeta ( \omega_1)   = -\zeta(\omega_1)+2 \eta_1\]

\[- \zeta (\omega_1)+ 2 \zeta ( \omega_1)   = -\zeta(\omega_1)+2 \eta_1\]

\[- \zeta (\omega_1) +\zeta(\omega_1)+ 2 \zeta ( \omega_1)   =2 \eta_1\]

\[2 \zeta ( \omega_1)   =2 \eta_1\]

\[\zeta ( \omega_1)   = \eta_1\] 따라서,

\[\eta_1 = \zeta(\omega_1)\] 그리고 또한,

\[\zeta (z+ 2 \omega_2)= \zeta(z) + 2 \eta_2\] 에서,

\[z =  -\omega_2\]를 예약하면,

\[\zeta(z) + 2 \eta_2 = \zeta(-\omega_2) + 2 \eta_2\]

\[\zeta(-\omega_2) + 2\eta_2 = -\zeta(\omega_2)+2 \eta_2\] 이고, 그리고,

\[\zeta (z+ 2 \omega_2)= \zeta (z)+ 2 \zeta ( \omega_2)= \zeta (-\omega_2)+ 2 \zeta ( \omega_2)\] 따라서,

\[\zeta (-\omega_2)+ 2 \zeta ( \omega_2)   = -\zeta(\omega_2)+2 \eta_2\]

\[- \zeta (\omega_2)+ 2 \zeta ( \omega_2)   = -\zeta(\omega_2)+2 \eta_2\]

\[- \zeta (\omega_2) +\zeta(\omega_2)+ 2 \zeta ( \omega_2)   =2 \eta_2\]

\[2 \zeta ( \omega_2)   =2 \eta_2\]

\[\zeta ( \omega_2)   = \eta_2\] 따라서,

\[\eta_2 = \zeta(\omega_2)\]

바이어슈트라스 에타 함수는 [데데킨트 에타 함수](../Page/데데킨트_에타_함수.md "wikilink"), [디리클레 에타 함수와](../Page/디리클레_에타_함수.md "wikilink") 다른 함수이므로 혼동하지 않게 주의해야 한다.

## 바이어슈트라스 에타 함수와 오메가2 상수

바이어슈트라스 에타 함수와 오메가2 상수\[1\]

\[\eta_1 \omega_2 - \eta_2 \omega_1 = {1 \over 2} \pi i  \qquad\]

\[\omega_2 = {{\Gamma^3 \left( {1 \over 3} \right)}\over{4 \pi}}\]

\[\qquad = 1.529954037.... \qquad (OEIS A064582) \qquad \Gamma(z)\]는 [감마 함수](../Page/감마_함수.md "wikilink")

\[\omega_1 = {1\over2} \omega_2 \left( 1 + i \sqrt{3} \right)\]

\[\qquad = { {  \left(1 + i \sqrt{3} \right)  \Gamma^3 \left( {1 \over 3} \right)}\over{8 \pi}}\]

\[\qquad = 0.764977....+ (1.32497903....) i\]\(\qquad (OEIS A094961, A094962)\)
