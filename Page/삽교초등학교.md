**삽교초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [예산군](../Page/예산군.md "wikilink") [삽교읍](../Page/삽교읍.md "wikilink") 두리에 있는 공립 초등학교다.

## 학교 연혁

| 1929년           | 6월 10일                    | 삽교공립보통학교 개교                                                   |
| --------------- | ------------------------- | ------------------------------------------------------------- |
| 1938년           | 4월 1일                     | 삽교공립심상소학교 개칭                                                  |
| 1941년           | 4월 1일                     | 삽교공립국민학교 개칭                                                   |
| 1949년           | 9월 20일                    | 안치국민학교 분리                                                     |
| 1969년           | 4월 25일                    | [용동국민학교](../Page/용동초등학교_\(충남\).md "wikilink") 분리              |
| 1991년           | 3월 1일                     | 성리분교장 통폐합                                                     |
| 1996년           | 3월 1일                     | 삽교초등학교로 개칭                                                    |
| 1999년           | 3월 1일                     | [안치초등학교](https://ko.wikipedia.org/wiki/안치초등학교 "wikilink") 통폐합 |
| 12월 22일         | 다목적교실 성덕관 개관              |                                                               |
| 2010년           | 1월 22일                    | 인조잔디 운동장 개장                                                   |
| 2015년           | 2월 13일                    | 제83회 졸업식(졸업생 14,772명)                                         |

| 유치원 1학급(20명) 편성 |                           |                                                               |
| 2016년           | 2월 4일                     | 제84회 졸업식 (졸업생 14,803명)                                        |
| 3월 1일           | 7학급 편성(특수학급 1학급 포함)(142명) |                                                               |
| 유치원 1학급(18명) 편성 |                           |                                                               |
| 2017년           | 2월 9일                     | 제85회 졸업식 (졸업생 14,829명)                                        |
