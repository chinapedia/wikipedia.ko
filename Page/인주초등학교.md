**인주초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [아산시](../Page/아산시.md "wikilink") [인주면](../Page/인주면.md "wikilink") 신성리에 있는 공립 초등학교다.

## 학교 연혁

| 1930년 | 5월 27일                   | 인주공립보통학교 개교(설립인가 [4월 29일](../Page/4월_29일.md "wikilink")) |
| ----- | ------------------------ | -------------------------------------------------------- |
| 1938년 | 4월 1일                    | 인주공립심상소학교로 개칭                                            |
| 1941년 | 4월 1일                    | 인주공립국민학교로 개칭                                             |
| 1949년 | 12월 31일                  | 인주국민학교로 개칭                                               |
| 1996년 | 3월 1일                    | 인주초등학교로 개칭                                               |
| 2013년 | 3월 1일                    | 충청남도교육청 지정 다문화 예비학교 연구학교 운영(2년)                          |
| 2016년 | 2월 5일                    | 제83회 졸업식(27명 졸업, 누계 7,848명)                              |
| 3월 1일 | 11학급 편성(일반 10학급, 특수 1학급) |                                                          |
| 2017년 | 2월 13일                   | 제84회 졸업 (총 7,875명)                                       |
| 3월 1일 | 11학급 편성(일반10, 특수1)       |                                                          |
| 2018년 | 2월 13일                   | 제85회 졸업 (총 7,904명)                                       |
| 3월 1일 | 11학급 편성(일반 10학급, 특수 1학급) |                                                          |
