**whoami**는 [유닉스 계열의](../Page/유닉스_계열.md "wikilink") 운영체제나 [윈도 비스타](https://ko.wikipedia.org/wiki/윈도_비스타 "wikilink"), [윈도 7](https://ko.wikipedia.org/wiki/윈도_7 "wikilink"), [윈도 서버 2003](https://ko.wikipedia.org/wiki/윈도_서버_2003 "wikilink"), [윈도 서버 2008에서](https://ko.wikipedia.org/wiki/윈도_서버_2008 "wikilink") 현재 사용자의 이름을 출력하는 명령어이다. 명령어의 이름은 [영어](../Page/영어.md "wikilink") 문장 "Who am I?"에서 비롯된 것이다.

## 예시

### 유닉스, 유닉스 계열

``` console
# whoami
root
```

### 인텔 iRMX 86

``` console
--WHOAMI
USER ID: 5
ACCESS ID'S: 5, WORLD
```

### 윈도우, ReactOS

``` doscon
C:\Users\admin>whoami
workgroup\admin
```
