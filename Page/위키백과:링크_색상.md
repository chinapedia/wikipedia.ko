#### 외부 링크

같은 방법으로 스타일이 적용된 span 태그를 외부 링크 안에서 사용할 수 있습니다.

### 모든 링크의 모양 지정

[스킨 보조문서에서](https://ko.wikipedia.org/wiki/특수:사용자_문서 "wikilink") CSS를 편집하여 링크 색상을 사용자 정의에 따라 지정할수 있습니다. 이 변경 사항은 사이트 전체의 *모든* 링크에 적용되지만, 사용자에게만 표시됩니다

표준 링크 선택기는 다음과 같습니다:

  - a:link - 일반적인 방문하지 않은 링크의 모양을 정의합니다.
  - a:visited - 방문한 링크의 모양을 정의합니다.
  - a:active - 활성 링크의 모양을 지정합니다. 링크를 클릭하면 링크가 활성화됩니다.
  - a:hover - 올려놓은 링크의 모양을 정의합니다. 링크가 마우스 위로 이동하면 링크가 가져옵니다.

색상은 16진수로 정의됩니다: [웹 색상을](https://ko.wikipedia.org/wiki/웹_색상 "wikilink") 참조하세요.

``` css
/* 표준 링크 색상 */
.mw-body-content a:link { color: #0000FF; } /* 일반적인 방문하지 않은 링크 */
.mw-body-content a:link:visited { color: #0B0080; } /* 방문한 링크 */
.mw-body-content a:link:active { color: #FF0000; } /* 활성 링크 */
.mw-body-content a:link.new { color: #FF0000; } /* 새 링크 */
.mw-body-content a:link.extiw { color: #3366BB; } /* 인터위키 링크 */
.mw-body-content a:link.external { color: #3366BB; } /* 외부 링크 */
.mw-body-content a:link.stub { color: #772233; } /* 공중 선회 링크 */

.mw-body-content a:link {color: #FF0000}
.mw-body-content a:visited {color: #00FF00}
.mw-body-content a:hover {color: #FF00FF}
.mw-body-content a:active {color: #0000FF}
```

참고:

  - a:hover는 a:link와 a:visited 뒤에 와야합니다.
  - a:active는 a:hover 뒤에 와야합니다.
  - 이 웹사이트의 기본 정의가 정의한 것보다 우선 순위가 높기 때문에, 표준 링크 정의 앞에 ".my-body-content"가 있어야합니다.

#### 넘겨주기

넘겨주기를 녹색 링크로 표시합니다:

``` css
.mw-body-content a.mw-redirect {color:#308050}
.mw-body-content a.mw-redirect:visited {color:#3070A0}
```

#### 외부 링크

외부 링크를 녹색 링크로 표시합니다:

``` css
.mw-body-content a.external {color: #008000}
.mw-body-content a.external:visited {color: #008000}
```

#### 본문 꾸미기

이렇게하면 밑줄과 같은 서식을 지정할수 있습니다. 예:

``` css
.mw-body-content a:link {color: #000000; text-decoration: underline; }
```

사용 가능한 값은 다음과 같습니다:

  - none
  - underline
  - overline
  - line-through

#### 글꼴 모음

이렇게하면 링크 글꼴이 변경됩니다:

``` css
.mw-body-content a:link {font-family: monospace}
```

#### 환경 설정

[환경 설정 메뉴의 "가젯" 탭](https://ko.wikipedia.org/wiki/특수:환경설정 "wikilink") ("보이기" 문단 아래)에 있는 "의 동음이의어 문서 링크 표시"기능을 사용하면 [동음이의어 문서에](https://ko.wikipedia.org/wiki/위키백과:동음이의어_문서 "wikilink") 대한 모든 링크가 주황색으로 표시됩니다.

#### 스크립트

[사용자:Anomie/링크분류](https://ko.wikipedia.org/wiki/사용자:Anomie/링크분류 "wikilink")는 삭제할 문서, 비자유 미디어, 넘겨주기, 보호된 문서와 같은 링크를 나타내기 위해 색상을 사용자가 맞춤 설정하는 인기 스크립트입니다.
