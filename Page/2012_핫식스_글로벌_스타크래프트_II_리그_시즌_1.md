**2012 핫식스 글로벌 스타크래프트 II 리그 시즌 1**(2012 Hot6ix Global Starcraft 2 League Season 1) 은 [곰TV](../Page/곰TV.md "wikilink")가 주최하는 [글로벌 스타크래프트 II 리그의](https://ko.wikipedia.org/wiki/글로벌_스타크래프트_II_리그 "wikilink") 14번째 정규 리그로 2012년에 열리는 첫 번째 정규 리그이다.

## 리그 일정

  - Code A 예선 : 1월 5일 \~ 1월 6일
  - Code S 16강 조지명식 : 1월 20일
  - Code S : 1월 9일 \~ 3월 3일
  - Code A : 1월 31일 \~ 3월 1일
  - 승격강등전 : 3월 12일 \~ 3월 16일

## 리그 방식 변경점

[2011 GSL Nov에서](../Page/소니_에릭슨_글로벌_스타크래프트_II_리그_November.md "wikilink") 크게 개편된 방식에서 약간 수정되었다. 우선 32강, 16강전 조별 풀리그에서 단판제로 진행되었던 것이 3전 2선승제로 변경되었고, Code S 조 지명식이 16강 이전으로 옮겨졌으며, 이에 따라 32강 대진표는 추첨 형식으로 변경되었다.
