**리그 오브 레전드 월드 챔피언십 2017** (League of Legends World Championship 2017)(**2017 롤드컵**)는 라이엇 게임즈가 주최, 주관하는 [리그 오브 레전드 월드 챔피언십의](../Page/리그_오브_레전드_월드_챔피언십.md "wikilink") 7번째 대회이다.

## 개요

  - 주최 : 라이엇 게임즈
  - 주관 : 라이엇 게임즈
  - 주제가 : Legends never die
  - 기념 스킨 : 챔피언십 애쉬
  - 우승팀 기념 스킨 : 나르, 자르반 4세, 이즈리얼, 탈리야, 자야, 라칸

### 방식

  - 플레이-인 스테이지 : 12팀이 참가
      - 1라운드 : 4개 조로 나누어 풀 리그 방식으로 진행, 조별 상위 2팀이 2라운드에 진출
      - 2라운드 : 5전 3선승제
  - 그룹 스테이지 : 4개 조로 나누어 풀 리그 방식으로 진행, 조별 상위 2팀이 8강에 진출
  - 결승 토너먼트 : 5전 3선승제

### 일정

  - 플레이-인 스테이지
      - 1라운드 : 9월 23일 (토) \~ 26일 (화)
      - 2라운드 : 9월 28일 (목) \~ 29일 (금)
  - 그룹 스테이지
      - 1주차 : 10월 5일 (목) \~ 8일 (일)
      - 2주차 : 10월 12일 (목) \~ 15일 (일)
  - 8강 : 10월 19일 (목) \~ 22일 (일)
  - 준결승 : 10월 28일 (토) \~ 29일 (일)
  - 결승 : 11월 4일 (토)

### 경기장

  - 플레이-인/그룹 스테이지 : 우한 스포츠 센터 (중국 우한)
  - 8강 : 광저우 체육관 (중국 광저우)
  - 준결승 : 상하이 오리엔탈 스포츠센터 (중국 상하이)
  - 결승 : 베이징 국립 경기장 (중국 베이징)

### 상금

  - 총 상금 : $ 4,600,000
  - 우승 : $ 1,725,000 (37.5%)
  - 준우승 : $ 621,000 (13.5%)
  - 3-4위 : $ 322,000 (7%)
  - 5-8위 : $ 184,000 (4%)
