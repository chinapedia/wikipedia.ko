**성읍초등학교**는 [제주특별자치도](https://ko.wikipedia.org/wiki/제주특별자치도 "wikilink") [서귀포시](https://ko.wikipedia.org/wiki/서귀포시 "wikilink") [표선면](../Page/표선면.md "wikilink") 성읍리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

  -

|           |                                  |                              |
| --------- | -------------------------------- | ---------------------------- |
| **1937년** | 08월 21일                          | 성읍공립심상소학교 설립 인가              |
| 09월 01일   | 개교, 4년제(성읍리 875번지, 표선공립보통학교장 겸직) |                              |
| **1941년** | 06월 07일                          | 성읍공립국민학교로 개명                 |

| **1997년** | 03월 01일                          | 제주도교육청 지정 전통음악교육시범학교         |
| **2000년** | 04월 08일                          | 전통예술모범학교 선정(문화관광부 지정)        |
| 10월 28일   | 학교 교기 여자 테니스부 창단                 |                              |

| **2004년** | 12월 10일                          | 전국 전통예술모범학교 발표회 우수상 수상       |
| **2009년** | 09월 01일                          | 제주특별자치도교육청 지정 농산어촌 전원학교 지정   |
| **2011년** | 01월 14일                          | 2010학년도 학교평가 우수학교 선정         |
| **2012년** | 01월 31일                          | 본관 대수선 공사 완공                 |
| 04월 10일   | 푸른 운동장 조성(천연잔디 및 트랙) 공사 완공       |                              |
| **2013년** | 01월 28일                          | 제4기 제주형 자율학교 운영(6년간)         |
| **2016년** | 04월 29일                          | 제50회 도민체전 테니스대회 여자단체전 우승     |
| 10월 08일   | 제55회 탐라문화제 학생민속예술축제 한라상(대회장상) 수상 |                              |

| **2017년** | 01월 19일                          | 학교 교문 교체 및 테니스장 정비           |

| **2018년** | 09월 01일                          | 제37대 윤성희 교장 부임               |

| 12월 21일   | 학교 후문 제주돌담 울타리 공사 완공             |                              |
