**2000년 하계 올림픽 남자 장대높이뛰기** 경기는 2004년 8월 25일부터 8월 27일까지 [아테네 올림픽 스타디움에서](../Page/아테네_올림픽_스타디움.md "wikilink") 열렸다.

## 경기 일정

<small>모든 경기시간은 그리스 표준시이다 ([UTC+2](https://ko.wikipedia.org/wiki/UTC+2 "wikilink"))</small>

| 날짜               | 시간        | 라운드    |
| ---------------- | --------- | ------ |
| 2004년 8월 25일 수요일 | **19:15** | **예선** |
| 2004년 8월 27일 금요일 | **20:00** | **결선** |
|                  |           |        |
