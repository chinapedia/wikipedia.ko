**앰퍼샌드**(, **&**, 앤드 기호)는 ‘\~와(과)’를 의미하는 [기호](https://ko.wikipedia.org/wiki/기호 "wikilink")이다. [영어](../Page/영어.md "wikilink")의 에 해당하는 [라틴어](../Page/라틴어.md "wikilink")의 의 [합자](https://ko.wikipedia.org/wiki/합자 "wikilink")로, 를 로 쓰기도 했다. [Z](../Page/Z.md "wikilink") 다음에 해당하는 27번째 [라틴 문자](https://ko.wikipedia.org/wiki/라틴_문자 "wikilink") [알파벳](https://ko.wikipedia.org/wiki/알파벳 "wikilink")으로 여겨졌던 시기도 있다. 또한 AND의 의미를 나타내기도 한다

## 사용

한국과 일본에서는 [가운뎃점](../Page/가운뎃점.md "wikilink")(·)과 흡사하게, 중국에서는 [모점](https://ko.wikipedia.org/wiki/반점과_모점 "wikilink")(、)과 흡사하게 사용된다. 어쩔 때는 [슬래시](https://ko.wikipedia.org/wiki/슬래시 "wikilink")(/)처럼 사용되기도 한다.

## 프로그래밍 언어

C 등 여러 프로그래밍 언어에서는 AND 연산자로 사용된다. 다음은 C의 예이다.

``` c
x = a && b;
```

위와 같이 & 2개를 붙여쓴 것은 논리연산자 AND를 나타낸다. 이 경우 A, B가 모두 참이라면 X도 참이 되고, 그 외의 경우는 거짓이 된다.

``` c
x = 0x12345678 & 0x0f0f0f0f;
```

위와 같이 &가 1개이면 비트연산자 AND를 나타낸다. 이 경우의 결과는 0x02040608이다.

BASIC 계열의 언어로는 문자열의 연결 연산자로서 사용된다.

C 언어에서 단항연산자로 쓰는 경우 변수의 주소를 나타낸다.

``` c
int v = 15;
int* ad = &v;
```

위의 소스 코드에서 변수 ad에는 변수 v의 주소가 담겨 있다.

``` qbasic
A$ = "foo" & "bar"
```

위의 경우에는 "foobar"를 돌려준다. 또, 주로 마이크로소프트 계열에서는 정수의 16진표기에 \&h를 이용해 \&h0F (10진으로 15)와 같이 표현한다.

SGML, XML, HTML에서는, 앰퍼샌드(&)를 사용해 SGML 실체를 참조한다.
