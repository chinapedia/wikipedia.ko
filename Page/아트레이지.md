**아트레이지**(ArtRage)는 Ambient Design Ltd이 개발한 [디지털 페인팅용](https://ko.wikipedia.org/wiki/디지털_페인팅 "wikilink") [비트맵 그래픽스 편집기이다](https://ko.wikipedia.org/wiki/비트맵_그래픽스_편집기 "wikilink"). 현재 최신 버전은 5.0.3이며, [윈도우](../Page/마이크로소프트_윈도우.md "wikilink"), [macOS](https://ko.wikipedia.org/wiki/macOS "wikilink"), 모바일 앱 및 안드로이드 장치들을 지원하며 여러 언어로 사용이 가능하다. 어린이에서부터 전문 아티스트에 이르기까지 다양한 연령대와 스킬 수준에 적합하다. 아트레이지 5는 2017년 1월에 발표되었으며\[1\], 출시는 2017년 2월에 이루어졌다.\[2\]

### 지원되는 파일 유형

아트레이지는 "painting"을 의미하는 ".ptg"라는 사유 파일 확장자를 사용한다.\[13\] 파일|열기 명령을 사용하여 PTG로만 저장이 가능하고 PSD, PTG 파일만 열 수 있다. 그러나 이미지의 경우 다음의 포맷으로 내보내기가 가능하다: [PNG](../Page/PNG.md "wikilink"), [JPEG](../Page/JPEG.md "wikilink"), [GIF](../Page/GIF.md "wikilink"), [BMP](https://ko.wikipedia.org/wiki/BMP_파일_포맷 "wikilink"), [TIFF](https://ko.wikipedia.org/wiki/TIFF "wikilink"), [어도비 포토샵의](../Page/어도비_포토샵.md "wikilink") [.psd](https://ko.wikipedia.org/wiki/.psd "wikilink") 포맷. 아트레이지는 파일 \> 그림 가져오기 또는 레이어 명령으로서 그림 가져오기를 사용하여 이러한 파일 유형을 가져올 수 있다. PTG 파일을 가져오면 섬네일 이미지를 위해 사용되는 PNG를 여는 과정이 대신 수행된다. (손상된 PTG 파일로부터 이미지를 복구하기 위해 사용할 수 있음)\[14\]

| 내보내기 파일 유형 | 레이어 | 투명도 |
| ---------- | --- | --- |
| BMP        | 아니오 | 아니오 |
| JPEG       | 아니오 | 아니오 |
| GIF        | 아니오 | 아니오 |
| PNG        | 아니오 | 예   |
| TIFF       | 아니오 | 예\* |
| PSD        | 예   | 예   |
|            |     |     |

\*개개의 레이어로 내보낼 때에만

**투명도**
아트레이지는 "가져온" 파일에 대해 투명도를 지원하지만 GIF와 TIFF 등의 파일을 내보내는 경우 투명도를 지원하지 않는다. 개개의 레이어로 투명 이미지를 내보내는 것이 더 쉬운 경우가 있다.
