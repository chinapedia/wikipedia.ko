**프랜차이즈 서울**(International Franchise Seoul)은 1996년에 시작된 **대한민국 최초의 창업 박람회**이다. 이 박람회는 [코엑스](../Page/코엑스.md "wikilink")가 주최한다. 전시회는 매 년 상반기에 서울커피엑스포와 함께 서울 코엑스에서 개최된다. 2016년에는 3월 3일부터 5일까지 150개사 350부스 규모로 코엑스 1층 B홀에서 개최된다.

## 지난 전시회

  - 2012년 이전 전시회는 생략함.

### 2012 프랜차이즈 서울 Spring

  - 개최기간 : 2012. 4. 26(목) \~ 4. 28(토)
  - 장소 : 코엑스 3층 C홀 3, 4실
  - 규모 : 114개사 252부스

### 2012 프랜차이즈 서울 Fall

  - 개최기간 : 2012. 9. 6(목) \~ 9. 8(토)
  - 장소 : 코엑스 3층 C홀 1, 2실
  - 규모 : 87개사 201부스

### 2013 프랜차이즈 서울

  - 개최기간 : 2013. 4. 11(목) \~ 4. 14(일)
  - 장소 : 코엑스 C홀
  - 규모 : 87개사 250부스
  - 참관객 : 17,252명
  - 주요 참가사 : 국순당, 놀부, 다이소, 달콤커피, 스테프핫도그, 파파존스피자, 커피마마, 스노우스푼, 릴렉스샵, 레드아이, 인토외식산업, 세븐일레븐, 튼튼영어 등

### 2014 프랜차이즈 서울

  - 개최기간 : 2014. 4. 10(목) \~ 4. 12(토)
  - 장소 : 코엑스 1층 B홀
  - 규모 : 109개사 254부스
  - 참관객 : 18,761명
  - 주요 참가사 : 맥도날드, 다이소, 365플러스, 초록마을, iFace, 써브웨이, 더후리아팬, 달콤커피, 디초콜릿커피, 커피베이, 셀렉토커피, 오드리햅번까페, 카페드롭탑, 놀부, 하루엔소쿠, 도니도니도니버거, 페리카나, 서래갈매기, 못된고양이 등

### 2015 프랜차이즈 서울

  - 개최기간 : 2015. 3. 12(목) \~ 3. 14(토)
  - 장소 : 코엑스 1층 B홀
  - 규모 : 2개국 129개사 324부스
  - 참관객 : 17,182명
  - 주요 참가사 : 달.콤커피, 드롭탑, 셀렉토커피, 주커피, 커피베이, 해피레몬플러스, 훌랄라, 마시그래이, 유가네, 미트 프레쉬, 땅땅치킨, 초록마을, 드림디포, 다이소, 압구정 봉구비어, 청담동 말자싸롱, 맥도날드, 스테프 핫도그, 도니버거, 시즌아이PC방, 워시엔조이, 장보는 강아지와 고양이, 호미빙, 브알라, 티니비, 바푸리, 율리어스마이늘, 불소식당, 투존치킨, 스트릿팬케익, 아이스캡, 멀티박스, 벨로스 등
