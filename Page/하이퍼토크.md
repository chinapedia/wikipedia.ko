**하이퍼토크**(HyperTalk)는 1987년에 Dan Winkler가 개발한 높은 수준의 [절차적](https://ko.wikipedia.org/wiki/절차적_프로그래밍 "wikilink") [프로그래밍 언어이다](https://ko.wikipedia.org/wiki/프로그래밍_언어 "wikilink"). [빌 앳킨슨의](https://ko.wikipedia.org/wiki/빌_앳킨슨 "wikilink") [애플](https://ko.wikipedia.org/wiki/애플_\(기업\) "wikilink") [하이퍼카드](../Page/하이퍼카드.md "wikilink") 하이퍼미디어 프로그램과 결합하는 데 사용된다.

## 예제

` on mouseUp`
`   put "100,100" into pos`
`   repeat with x = 1 to the number of card buttons`
`     set the location of card button x to pos`
`     add 15 to item 1 of pos`
`   end repeat`
` end mouseUp`

` on mouseDown`
`   put "Disk:Folder:MyFile" into filePath -- no need to declare variables`
`   if there is a file filePath then`
`     open file filePath`
`     read from file filePath until return`
`     put it into cd fld "some field"`
`     close file filePath`
`     set the textStyle of character 1 to 10 of card field "some field" to bold`
`   end if`
` end mouseDown`

` function replaceStr pattern,newStr,inStr`
`   repeat while pattern is in inStr`
`     put offset(pattern,inStr) into pos`
`     put newStr into character pos to (pos +the length of pattern)-1 of inStr`
`   end repeat`
`   return inStr`
` end replaceStr`
