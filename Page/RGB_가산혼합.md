[섬네일](https://ko.wikipedia.org/wiki/파일:Synthese+.svg "wikilink") [섬네일](https://ko.wikipedia.org/wiki/파일:RGBCube_b.svg "wikilink") **RGB 가산혼합**(RGB 加算混合)은 빛의 삼원색을 이용하여 [색](https://ko.wikipedia.org/wiki/색 "wikilink")을 표현하는 방식이다. [빨강](../Page/빨강.md "wikilink")(RED), [초록](../Page/초록.md "wikilink")(GREEN), [파랑](../Page/파랑.md "wikilink")(BLUE) 세 종류의 광원(光源)을 이용하여 색을 혼합하며 색을 섞을수록 밝아지기 때문에 '가산혼합'이라고 한다. 디지털 이미지에서 사용되는 RGB 가산혼합의 종류로는 [sRGB](https://ko.wikipedia.org/wiki/sRGB "wikilink"), [어도비 RGB](https://ko.wikipedia.org/wiki/어도비_RGB "wikilink") 등이 있다.

## 가산 혼합의 원리

  - 알기 쉬운 실험

<!-- end list -->

  -
    세개의 손전등에 각각 빨강, 초록, 파랑의 [셀로판지](https://ko.wikipedia.org/wiki/셀로판지 "wikilink")를 겹쳐놓고 불을 비추면 오른쪽의 그림 '가산혼합의 원리'와 비슷한 결과를 얻을 수 있다.

<!-- end list -->

  - RGB 큐브 모델과 표현 원리

<!-- end list -->

  -
    오른쪽의 그림 'RGB의 큐브 모델'과 같이 하나의 색은 삼원색을 기준선으로 하는 삼차원 [직교 좌표계의](https://ko.wikipedia.org/wiki/직교_좌표계 "wikilink") 한 점으로 나타낼 수 있다.
    예를 들어 [검정](https://ko.wikipedia.org/wiki/검정 "wikilink")은 세 광원 모두 빛이 없는 상태이며 [좌표](https://ko.wikipedia.org/wiki/좌표 "wikilink")(0,0,0)으로 표현할 수 있다.

|          |            |
| -------- | ---------- |
| 채널별 값    | 혼합결과와 좌표   |
| 빨강 (RED) | 초록 (GREEN) |
| 0        | 0          |

  -
    각 광원의 최댓값을 255라고 한다면 빨강은 좌표(255,0,0)과 같이 표현되며 RED 광원만이 최댓값을 갖고 GREEN 광원과 BLUE광원은 빛이 없는 상태를 의미한다.

|          |            |
| -------- | ---------- |
| 채널별 값    | 혼합 결과와 좌표  |
| 빨강 (RED) | 초록 (GREEN) |
| 255      | 0          |

  -
    좌표(80,0,0)과 같이 GREEN 광원과 BLUE 광원이 여전히 빛이 없는 상태에서 RED 광원이 약하게 비춰진다면 [고동색](https://ko.wikipedia.org/wiki/고동색 "wikilink")이 될 것이다.

|          |            |
| -------- | ---------- |
| 채널별 값    | 혼합결과와 좌표   |
| 빨강 (RED) | 초록 (GREEN) |
| 80       | 0          |

  -
    [주황색](https://ko.wikipedia.org/wiki/주황색 "wikilink")은 좌표(255,127,0)으로 나타낼 수 있으며 RED 광원이 최댓값을 갖는 상태에서 GREEN 광원이 중간 정도로 섞인 상태라고 할 수 있다.

|          |            |
| -------- | ---------- |
| 채널별 값    | 혼합결과와 좌표   |
| 빨강 (RED) | 초록 (GREEN) |
| 255      | 127        |

  -
    세 광원이 모두 같은 값을 갖으면 [무채색](https://ko.wikipedia.org/wiki/무채색 "wikilink")이 된다.

|          |            |
| -------- | ---------- |
| 채널별 값    | 혼합결과와 좌표   |
| 빨강 (RED) | 초록 (GREEN) |
| 192      | 192        |

  - 감산혼합과의 관계

<!-- end list -->

  -
    RGB 가산혼합과 달리 [CMYK 감산혼합은](https://ko.wikipedia.org/wiki/CMYK_감산혼합 "wikilink") 물감이나 잉크같은 안료가 서로 섞일 때 일어나는 색의 혼합을 표현하는 방식으로 색을 섞을수록 어두워지기 때문에 '감산혼합'이라 한다. 가산혼합과 감산혼합은 서로 반대되는 개념으로 [보색](https://ko.wikipedia.org/wiki/보색 "wikilink") 관계에 있다.

## RGB 채널

[섬네일](https://ko.wikipedia.org/wiki/파일:Barn_grand_tetons_rgb_separation.jpg "wikilink") [포토샵](https://ko.wikipedia.org/wiki/포토샵 "wikilink") 등에서 지원하는 기본 채널이다. 이 채널들은 각각 RGB채널, Red 채널, Green 채널, Blue 채널로 나뉘며 RGB채널을 선택하면 Red, Green, Blue 채널이 모두 선택된다.

RGB 가산혼합에서, 빨강은 [하이톤](https://ko.wikipedia.org/wiki/하이톤 "wikilink") 라임은 [미드톤](https://ko.wikipedia.org/wiki/미드톤 "wikilink") 파랑은 [섀도톤](https://ko.wikipedia.org/wiki/섀도톤 "wikilink")을 각각 맡으며, 보기엔 [그레이스케일](https://ko.wikipedia.org/wiki/그레이스케일 "wikilink")처럼 느껴진다.

## RGB와 디스플레이

RGB 가산혼합을 사용하고 있는 일반적인 사례로는 [음극선관](https://ko.wikipedia.org/wiki/음극선관 "wikilink")(브라운관), [액정 디스플레이](https://ko.wikipedia.org/wiki/액정_디스플레이 "wikilink"), [플라스마 디스플레이와](../Page/플라스마_디스플레이.md "wikilink") 같이 [컴퓨터](../Page/컴퓨터.md "wikilink"), [텔레비전](https://ko.wikipedia.org/wiki/텔레비전 "wikilink")의 영상 표시에 사용되는 [디스플레이](https://ko.wikipedia.org/wiki/디스플레이 "wikilink")이다. 화면을 구성하는 각 픽셀은 컴퓨터와 [그래픽 카드](https://ko.wikipedia.org/wiki/그래픽_카드 "wikilink") 등으로 빨강, 초록, 파랑의 명도(Value)를 표현한다. 이들 수치는 [감마 보정으로](https://ko.wikipedia.org/wiki/감마_보정 "wikilink") 표현하고 싶은 휘도에서 디스플레이에 표시되는 휘도(intensity)와 전압으로 전환되고 있다.

빨강, 초록, 파랑 세 종류의 광원의 조합을 통해 여러 가지 색을 표현한다. 2017년 현재 전형적인 디스플레이는 하나의 픽셀에 [24비트](https://ko.wikipedia.org/wiki/24비트 "wikilink") 규모의 정보를 사용하고 있다. 이는 [8비트](https://ko.wikipedia.org/wiki/8비트 "wikilink")를 각각 빨강, 초록, 파랑에 각각 할당함으로써 각 색상(hue)마다 256가지의 명도, 선명도를 결정한다. 이 시스템으로는 16,777,216가지(256<sup>3</sup>혹은 2<sup>24</sup>)의 색상(hue), 채도(saturation), 명도(value)를 표현할 수 있다.
