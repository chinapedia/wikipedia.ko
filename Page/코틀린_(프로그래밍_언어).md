**코틀린**(Kotlin)은 [JVM](https://ko.wikipedia.org/wiki/JVM "wikilink")에서 동작하는 프로그래밍 언어이다. 2011년 7월, [젯브레인](https://ko.wikipedia.org/wiki/젯브레인 "wikilink")사가 공개하였다.

캇린으로 읽어야 한다. 파일 확장자는 **.kt** 또는 **.kts**를 사용한다.

코틀린은 2019년 5월 7일 이후로 구글의 안드로이드 앱 개발에서 선호하는 언어가 되었다.\[1\]

## 문법

### 기본 진입점

``` kotlin numberLines

// Hello, World! example
fun main(args: Array<String>) {
  val scope = "World"
  println("Hello, $scope!")
}
```
