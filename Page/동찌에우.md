**동찌에우 시사**(\[1\])는 [베트남](https://ko.wikipedia.org/wiki/베트남 "wikilink") [꽝닌성](https://ko.wikipedia.org/wiki/꽝닌성 "wikilink")의 [티싸](https://ko.wikipedia.org/wiki/티싸 "wikilink")(시사)이다.

## 행정 구역

동찌에우 [시사에는](https://ko.wikipedia.org/wiki/티싸 "wikilink") 10개 [방](https://ko.wikipedia.org/wiki/프엉 "wikilink")(坊)과 11개 [사](https://ko.wikipedia.org/wiki/싸_\(행정_단위\) "wikilink")(社)가 있다.

  - 방
      - 동찌에우 (Đông Triều / 東潮)
      - 마오케 (Mạo Khê / 帽溪)
      - 쑤언선 (Xuân Sơn / 春山)
      - 흥다오 (Hưng Đạo / 興道)
      - 낌선 (Kim Sơn / 金山)
      - 득찐 (Đức Chính / 德政)
      - 홍퐁 (Hồng Phong / 鴻豐)
      - 짱안 (Tràng An / 長安)
      - 옌토 (Yên Thọ / 安壽)
      - 호앙꾸에 (Hoàng Quế / 黃桂)
  - 사
      - 투이안 (Thuỷ An / 水安)
      - 응우옌후에 (Nguyễn Huệ / 阮惠)
      - 비엣전 (Việt Dân / 越民)
      - 떤비엣 (Tân Việt / 新越)
      - 빈즈엉 (Bình Dương / 平陽)
      - 옌득 (Yên Đức / 安德)
      - 짱르엉 (Tràng Lương / 長良)
      - 빈케 (Bình Khê / 平溪)
      - 홍타이동 (Hồng Thái Đông / 鴻泰東)
      - 홍타이떠이 (Hồng Thái Tây / 鴻泰西)
      - 안신 (An Sinh / 安生)
