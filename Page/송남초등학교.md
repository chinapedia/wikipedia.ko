**송남초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [아산시](../Page/아산시.md "wikilink") [송악면](../Page/송악면.md "wikilink") 역촌리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1924년 | 7월 15일  | 송남공립보통학교 개칭(설립인가 5월 9일)                   |
| ----- | ------- | ----------------------------------------- |
| 1938년 | 4월 1일   | 송남공립심상소학교로 개칭                             |
| 1941년 | 4월 1일   | 송남공립국민학교로 개칭                              |
| 1949년 | 9월 30일  | 거산국민학교 분리                                 |
| 1950년 | 4월 1일   | 송남국민학교로 개칭                                |
| 1954년 | 4월 1일   | 동화국민학교 분리                                 |
| 1981년 | 3월 10일  | 병설유치원 설립인가                                |
| 1991년 | 3월 1일   | 동화분교장 편입                                  |
| 1992년 | 3월 1일   | 거산분교장 편입                                  |
| 1994년 | 3월 1일   | 동화분교장 본교로 통폐합                             |
| 1996년 | 3월 1일   | 송남초등학교로 개칭                                |
| 2005년 | 3월 1일   | [거산초등학교](../Page/거산초등학교.md "wikilink") 분리 |
| 2010년 | 2월 11일  | 샛별관 개관                                    |
| 2016년 | 2월 4일   | 제90회 졸업(졸업생 5,790명)                       |
| 3월 1일 | 12학급 편성 |                                           |
| 2017년 | 3월 1일   | 12학급 편성                                   |
