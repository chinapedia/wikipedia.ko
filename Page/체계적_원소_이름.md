[화학](../Page/화학.md "wikilink")에서 **체계적 원소 이름**은 아직 발견되지 않았거나 발견되었으나 아직 이름이 결정되지 않은 원소를 논문 등에서 가리키기 위하여 임시로 붙이는 이름이다. 이 체계는 원소의 [원자 번호를](../Page/원자_번호.md "wikilink") 바탕으로 하며, [국제순수·응용화학연합](https://ko.wikipedia.org/wiki/국제순수·응용화학연합 "wikilink")(IUPAC)에서 결정하였다.

## 규칙

| 자리 | 이름 토막 | 기호 토막 |
| -- | ----- | ----- |
| 0  | nil   | n     |
| 1  | un    | u     |
| 2  | b(i)  | b     |
| 3  | tr(i) | t     |
| 4  | quad  | q     |
| 5  | pent  | p     |
| 6  | hex   | h     |
| 7  | sept  | s     |
| 8  | oct   | o     |
| 9  | en(n) | e     |

어떤 원소의 체계적 원소 이름은 원자 번호를 [십진법](https://ko.wikipedia.org/wiki/십진법 "wikilink")으로 표기한 각 자리를 표에 따라 해당하는 이름 토막으로 바꾸고 뒤에 -ium을 붙여 이은 것이다. 이름 토막의 어원은 [라틴어](../Page/라틴어.md "wikilink")와 [그리스어](../Page/그리스어.md "wikilink")에서 왔는데, 두 언어를 함께 쓴 이유는 모든 이름 토막이 서로 다른 글자로 시작되게 (따라서 기호 토막이 중복되지 않게) 하기 위함이다. 다만 글자가 중복되어 이상하게 보이는 이름을 피하기 위하여 다음 두 법칙이 적용된다.

  - 마지막 자리가 2(-bi-)거나 3(-tri-)이면 원소 이름은 -biium이나 -triium이 아닌 -bium이나 -trium으로 끝난다.
  - 숫자에서 -90-이라는 자리들이 연달아 있으면 원소 이름은 -ennnil-이 아닌 -ennil-이 된다.

해당 원소 이름에 대응되는 원소 기호는 단순히 각 토막의 맨 첫 글자를 딴 뒤 그 중 맨 첫 글자를 대문자로 바꾼 것이다. 이 규칙이 제정되던 당시에는 110번까지 ([2019년](../Page/2019년.md "wikilink") 현재는 118번까지) 의 모든 원소가 이름을 부여받았기 때문에, 체계적 원소 이름은 사실상 세 글자로 이루어져 있다고 생각할 수 있다.

하지만 137번과 155번 원소는 각각 파인마늄과 카자늄으로 불리기도한다.

즉, 이 규칙을 적용하면 다음과 같이 체계적 원소 이름을 적용할 수 있다.

1.  119번 원소: un + un + enn + ium = [우누넨늄](../Page/우누넨늄.md "wikilink")(Ununennium, Uue)
2.  123번 원소: un + bi + tri + ium = [운비트륨](https://ko.wikipedia.org/wiki/운비트륨 "wikilink")(Unbitrium, Ubt)
3.  208번 원소: bi + nil + oct + ium = 비닐옥튬(Biniloctium, Bno)
4.  457번 원소: quad + pent + sept + ium = 쿼드펜트셉튬(Quadpentseptium, Qps)
5.  888번 원소: oct + oct + oct + ium = 옥톡톡튬(Octoctoctium)
