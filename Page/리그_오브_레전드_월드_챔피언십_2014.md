**리그 오브 레전드 월드 챔피언십 2014** (League of Legends World Championship 2014)(**2014롤드컵**)는 라이엇 게임즈가 주최하고 라이엇 게임즈, [온게임넷](https://ko.wikipedia.org/wiki/온게임넷 "wikilink"), [한국e스포츠협회](https://ko.wikipedia.org/wiki/한국e스포츠협회 "wikilink"), 가레나가 주관한 [리그 오브 레전드 월드 챔피언십의](https://ko.wikipedia.org/wiki/리그_오브_레전드_월드_챔피언십 "wikilink") 4번째 대회이다.

## 개요

  - 주최 : 라이엇 게임즈
  - 주관 : 라이엇 게임즈, 온게임넷, 한국e스포츠협회, 가레나
  - 협찬 : 코카콜라, 대한항공, 엔비디아
  - 주제가 : Warriors
  - 기념 스킨 : 챔피언십 쉬바나
  - 우승팀 기념 스킨 : 신지드, 렝가, 탈론, 트위치, 쓰레쉬

### 방식

  - 그룹 스테이지 : 각 조 4팀씩 4개 조로 나누어 더블 라운드 로빈 방식으로 진행, 각 조 상위 두 팀은 결승 토너먼트에 진출한다.
  - 결승 토너먼트 : 5전 3선승제

### 변경된 점

  - 14팀에서 16팀으로 증가
  - 결승 토너먼트 직행을 폐지하고 모든 팀이 그룹 스테이지에 참가하게 된다.

### 일정

  - 그룹 스테이지
      - A/B조 : 9월 18일 (목) \~ 9월 21일 (일)
      - C/D조 : 9월 25일 (목) \~ 9월 28일 (일)
  - 8강 : 10월 3일 (금) \~ 10월 6일 (월)
  - 준결승 : 10월 11일 (토) \~ 10월 12일 (일)
  - 결승 : 10월 19일 (일)

### 경기장

  - 그룹 스테이지
      - A/B조 :  대만 타이페이 국립 대만 대학 스포츠 센터
      - C/D조 :  싱가포르 싱가포르 엑스포
  - 8강 :  대한민국 부산광역시 BEXCO
  - 준결승 :  대한민국 서울특별시 잠실실내체육관
  - 결승 :  대한민국 서울특별시 서울월드컵경기장

### 상금

  - 총 상금 : $ 2,130,000
  - 우승 : $ 1,000,000
  - 준우승 : $ 250,000
  - 준결승 : $ 150,000
  - 8강 : $ 75,000
  - 그룹 스테이지 각 조 3위 : $ 45,000
  - *그룹 스테이지 공동 조 3위 : $ 35,000 (동률 발생시)*
  - 그룹 스테이지 각 조 4위 : $ 25,000
