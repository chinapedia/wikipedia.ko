[대수기하학](https://ko.wikipedia.org/wiki/대수기하학 "wikilink")에서, **델 페초 곡면**(del Pezzo曲面, )은 [사영 평면의](../Page/사영_평면.md "wikilink") 점들을 [부풀려](https://ko.wikipedia.org/wiki/부풀리기 "wikilink") 얻을 수 있는 [대수 곡면의](../Page/대수_곡면.md "wikilink") 한 종류다.

## 정의

[대수적으로 닫힌 체](../Page/대수적으로_닫힌_체.md "wikilink") \(K\)에 대하여, **델 페초 곡면**은 다음 두 조건을 만족시키는 \(K\)-[대수다양체](https://ko.wikipedia.org/wiki/대수다양체 "wikilink") \(X\)이다.

  - [대수 곡면이다](../Page/대수_곡면.md "wikilink"). 즉, [크룰 차원이](https://ko.wikipedia.org/wiki/크룰_차원 "wikilink") 2차원이다.
  - [파노 다양체이다](https://ko.wikipedia.org/wiki/파노_다양체 "wikilink"). 즉, [반표준 인자](https://ko.wikipedia.org/wiki/반표준_인자 "wikilink") \(-K_X\)가 [풍부한 인자이며](https://ko.wikipedia.org/wiki/풍부한_선다발 "wikilink"), [완비 대수다양체이다](https://ko.wikipedia.org/wiki/완비_대수다양체 "wikilink").

## 분류

델 페초 곡면은 [유리 곡면이다](https://ko.wikipedia.org/wiki/유리_곡면 "wikilink"). 즉, [사영 평면과](../Page/사영_평면.md "wikilink") [쌍유리 동치이다](https://ko.wikipedia.org/wiki/쌍유리_동치 "wikilink"). 델 페초 곡면 *X*의 **차수**() *d*는 [반표준 인자](https://ko.wikipedia.org/wiki/반표준_인자 "wikilink")(또는 [표준 인자](https://ko.wikipedia.org/wiki/표준_인자 "wikilink"))의 제곱이며, 표준 인자의 자기 [교차수](../Page/교차수.md "wikilink")와 같다.

\[d = (-K_X)^2 = K_X^2\]

모든 델 페초 곡면은 다음 가운데 정확히 하나와 동형이다.

  - [사영 평면의](../Page/사영_평면.md "wikilink") \(k\)개의 점에서의 [부풀리기](https://ko.wikipedia.org/wiki/부풀리기 "wikilink") (\(k=0,1,2,\dots,8\)). 보통 \(\operatorname{dP}_k\)로 쓴다. 이 경우, 차수가 \(9-k\)이며, [피카르 군은](../Page/피카르_군.md "wikilink") 홀 [유니모듈러 격자](../Page/유니모듈러_격자.md "wikilink") \(\operatorname I_{1,k}\)이다.
  - \(\mathbb P^1\times\mathbb P^1\). 이 경우 차수가 8이며, [피카르 군은](../Page/피카르_군.md "wikilink") 짝 [유니모듈러 격자](../Page/유니모듈러_격자.md "wikilink") \(\operatorname{II}_{1,1}\)이다.

사영 평면에서 9개 이상의 점들을 부풀리면 더 이상 반표준 인자가 풍부하지 않다. 예를 들어, 9개의 점을 부풀리면 반표준 인자의 자기 교차수가 0이다.

### (−1)-곡선

델 페초 곡면에서 **(−1)-곡선**()은 [자기 교차수가](https://ko.wikipedia.org/wiki/자기_교차수 "wikilink") −1이며, [표준 인자와의](https://ko.wikipedia.org/wiki/표준_인자 "wikilink") 교차수 역시 −1인 (기약) 곡선이다. 이들은 다음과 같다.

\[\pi\colon\operatorname{dP}_n\twoheadrightarrow\mathbb P^2\] 에서, \(\mathbb P^2\)의 [세르 뒤틀림 층](https://ko.wikipedia.org/wiki/세르_뒤틀림_층 "wikilink") \(O(1)\)의 [당김을](https://ko.wikipedia.org/wiki/당김_\(미분기하학\) "wikilink") \(H=\pi^*O(1)\)라고 하고, [부풀리기](https://ko.wikipedia.org/wiki/부풀리기 "wikilink")로 발생하는 예외적 인자들을 \(E_1,\dots,E_n\)이라고 하자. 이는 델 페초 곡면의 [피카르 군](../Page/피카르_군.md "wikilink")

\[\operatorname{Pic}(\operatorname{dP}_n)=\operatorname{Span}_{\mathbb Z}\{H,E_1,\dots,E_n\}\cong\mathbb Z^{n+1}\] 의 [기저를](../Page/기저_\(선형대수학\).md "wikilink") 이룬다. 이 경우, [교차수](../Page/교차수.md "wikilink")는 다음과 같다.

\[E_i.E_j=-\delta_{ij}\]

\[E_i.H=0\]

\[H.H=1\] [표준 인자는](https://ko.wikipedia.org/wiki/표준_인자 "wikilink") 다음과 같다.

\[K=-3H+\sum_iE_i\] 인자

\[C=aH-\sum_{i=1}^nb_iE_i\] 가 (−1)-곡선을 이루려면, 다음 연립 [디오판토스 방정식을](https://ko.wikipedia.org/wiki/디오판토스_방정식 "wikilink") 만족시켜야 한다.\[1\]

\[a^2-\sum_{i=1}^nb_i^2=-3a+\sum_{i=1}^nb_i=-1\] 이 디오판토스 방정식의 해는 \(n<9\)인 경우 다음과 같다.

| \(a\) | 0이 아닌 \(b_i\)          | 개수             | 해석                                                                    |
| ----- | ---------------------- | -------------- | --------------------------------------------------------------------- |
| 0     | −1                     | \(n\)          | 예외 곡선                                                                 |
| 1     | 1, 1                   | \(\binom n2\)  | 부풀려진 두 점을 지나는 [사영 직선](https://ko.wikipedia.org/wiki/사영_직선 "wikilink") |
| 2     | 1, 1, 1, 1, 1          | \(\binom n5\)  | 부풀려진 5개의 점을 지나는 [원뿔 곡선](../Page/원뿔_곡선.md "wikilink")                  |
| 3     | 2, 1, 1, 1, 1, 1, 1    | \(7\binom n7\) | 하나의 2중점을 갖는 3차 유리 곡선. \(n=7,8\)인 경우에만 가능                              |
| 4     | 2, 2, 2, 1, 1, 1, 1, 1 | 56             | 세 개의 2중점을 갖는 4차 유리 곡선. \(n=8\)인 경우에만 가능                               |
| 5     | 2, 2, 2, 2, 2, 2, 1, 1 | 28             | 6개의 2중점을 갖는 5차 유리 곡선. \(n=8\)인 경우에만 가능                                |
| 6     | 3, 2, 2, 2, 2, 2, 2, 2 | 8              | 7개의 2중점과 하나의 3중점을 갖는 6차 유리 곡선. \(n=8\)인 경우에만 가능                       |

### 호지 수

델 페초 곡면 \(\operatorname{dP}_n\)은 [사영 평면을](../Page/사영_평면.md "wikilink") \(n\)번 [부풀려](https://ko.wikipedia.org/wiki/부풀리기 "wikilink") 얻은 곡면이다. 따라서, [호지 수](https://ko.wikipedia.org/wiki/호지_수 "wikilink") 가운데 \(h^{1,1}\)만이 \(n\)만큼 증가하고, 나머지 호지 수들은 사영 평면의 호지 수와 같다. 즉, \(\operatorname{dP}_n\)의 호지 수는 다음과 같다.

|   |   |       |
| - | - | ----- |
|   |   | 1     |
|   | 0 |       |
| 0 |   | 1+*n* |
|   | 0 |       |
|   |   | 1     |

\(\mathbb P^1\times\mathbb P^1\)의 경우 두 사영 직선의 곱이므로 호지 수를 쉽게 계산할 수 있으며, 다음과 같다.

|   |   |   |
| - | - | - |
|   |   | 1 |
|   | 0 |   |
| 0 |   | 2 |
|   | 0 |   |
|   |   | 1 |

## 역사

[나폴리](https://ko.wikipedia.org/wiki/나폴리 "wikilink")의 수학자 [파스콸레 델 페초](https://ko.wikipedia.org/wiki/파스콸레_델_페초 "wikilink")()가 1880년대에 연구하였다.\[2\]\[3\]

## 응용

델 페초 곡면은 이론 물리학에서 다양하게 등장한다. 델 페초 곡면은 [거울 대칭의](https://ko.wikipedia.org/wiki/거울_대칭 "wikilink") 중요한 예이다.\[4\] 델 페초 곡면은 또한 [자이베르그 이중성에](../Page/자이베르그_이중성.md "wikilink") 사용되며,\[5\] [M이론](https://ko.wikipedia.org/wiki/M이론 "wikilink")의 **신비로운 이중성**()에 등장한다.\[6\]
