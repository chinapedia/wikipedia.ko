**TBN 차차차**는 [TBN 부산교통방송](../Page/부산교통방송.md "wikilink")(FM 94.9㎒)에서 매일 낮 12시 5분부터 2시까지 방송되는 부산권 지역의 라디오 음악 프로그램이다.

## 코너소개

### 매일

  - '1대 1 후다닭 퀴즈' : 청취자 퀴즈 대결
  - '차차차 총알 배송' : 신청곡 리퀘스트

### 월요일

  - 노래 속, 사연 하나 : 청취자의 마음에 간직한 음악 (취재컷)
  - 이번 주는 여기\! : 부산,울산,경남지역 공연문화,축제소식 (방송인 한태양)

### 화요일

  - 명가수 라이벌 열전 : 명가수의 라이벌 비교, 분석 (방송인 이병호)
  - 한낮의 데이트 : 인기가수를 초대해 근황과 노래를 듣는 시간

### 수요일

  - 최대호의 가요산책 : 추억의 명곡과 가수 소개 (대중가요 평론가 최대호)
  - 오늘은 내가 선곡왕\! : 특정 주제를 놓고 청취자가 선곡

### 목요일

  - 차차차 노래교실 : 노래 한 곡을 선정해 잘 부르는 꿀팁 전수 (통기타 가수 박상운)
  - 김강수의 한국기행 : 월별, 계절별 가볼만한 국내 여행지 소개 (여행작가 김강수)

### 금요일

  - 금주의 가요계 소식 (뉴데일리 문화부 신세인 기자)
  - 노래 불러\~ 호캉스\! : 바쁜 일상을 잠시 내려놓고 가수가 되어보는 시간, 노래방 대결 (심사위원 방송인 이경)

### 토요일

  - <건널목 초자할매> : 건널목 교통봉사대 초자할매와 함께하는 엉뚱·발랄 퀴즈쇼(풍자 꽁트)
  - <나는 갈매기> : 롯데 자이언츠 BJ 콘대와 함께하는 부산 야구 이야기
  - <안녕하세요> : 인기가수의 근황을 들어보는 시간
  - <초희와 한판승부> : 청취자와 MC가 펼치는 막간 게임쇼
  - <내가쏜다> : 통닭 12마리로 한턱 내는 시간

### 일요일

  - <건널목 초자할매> : 건널목 교통봉사대 초자할매와 함께하는 엉뚱·발랄 퀴즈쇼(풍자 꽁트)
  - <운동합시다> : 부산시 체육회와 함께하는 부산 생활체육 이야기
  - <노래찾아삼만리> : 다양한 힌트로 노래 제목을 맞춰보는 시간
  - <내가쏜다> : 통닭 12마리로 한턱 내는 시간
