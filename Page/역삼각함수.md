[수학](../Page/수학.md "wikilink")에서, **역삼각 함수**(逆三角函數, )는 [삼각 함수의](https://ko.wikipedia.org/wiki/삼각_함수 "wikilink") [역함수](../Page/역함수.md "wikilink")이다. [삼각 함수는](https://ko.wikipedia.org/wiki/삼각_함수 "wikilink") [단사 함수](../Page/단사_함수.md "wikilink")(또는 일대일 함수)가 아니기 때문에 이의 역함수를 정의하려면 [정의역](https://ko.wikipedia.org/wiki/정의역 "wikilink")을 제한하는 것이 필요하다.

아래는 역삼각함수들의 정의와 표기법, 정의역과 치역들을 나타낸 표이다.

| 이름     | 표기법               | 정의                          | 정의역            | 치역 ([라디안](https://ko.wikipedia.org/wiki/라디안 "wikilink")) |
| ------ | ----------------- | --------------------------- | -------------- | -------------------------------------------------------- |
| 아크사인   | *y* = arcsin(*x*) | *y* = sin<sup>-1</sup>(*x*) | *x* = sin(*y*) | −1 ≤ *x* ≤ 1                                             |
| 아크코사인  | *y* = arccos(*x*) | *y* = cos<sup>-1</sup>(*x*) | *x* = cos(*y*) | −1 ≤ *x* ≤ +1                                            |
| 아크탄젠트  | *y* = arctan(*x*) | *y* = tan<sup>-1</sup>(*x*) | *x* = tan(*y*) | 모든 [실수](https://ko.wikipedia.org/wiki/실수 "wikilink")     |
| 아크코탄젠트 | *y* = arccot(*x*) | *y* = cot<sup>-1</sup>(*x*) | *x* = cot(*y*) | 모든 [실수](https://ko.wikipedia.org/wiki/실수 "wikilink")     |
| 아크시컨트  | *y* = arcsec(*x*) | *y* = sec<sup>-1</sup>(*x*) | *x* = sec(*y*) | *x* ≤ −1 또는 1 ≤ *x*                                      |
| 아크코시컨트 | *y* = arccsc(*x*) | *y* = csc<sup>-1</sup>(*x*) | *x* = csc(*y*) | *x* ≤ −1 또는 1 ≤ *x*                                      |
|        |                   |                             |                |                                                          |

정의역을 [복소수](../Page/복소수.md "wikilink")로 두게 되면 위에서 치역의 범위는 [실수부](https://ko.wikipedia.org/wiki/실수부 "wikilink")의 범위가 된다.
