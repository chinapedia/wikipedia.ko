[수학](../Page/수학.md "wikilink")에서, **역삼각 함수**(逆三角函數, )는 [삼각 함수의](https://ko.wikipedia.org/wiki/삼각_함수 "wikilink") [역함수](../Page/역함수.md "wikilink")이다. [삼각 함수는](https://ko.wikipedia.org/wiki/삼각_함수 "wikilink") [단사 함수가](../Page/단사_함수.md "wikilink") 아니기 때문에 이의 역함수를 정의하려면 [정의역](https://ko.wikipedia.org/wiki/정의역 "wikilink")을 제한하는 것이 필요하다.

아래는 역삼각함수들의 정의와 표기법, 정의역과 치역들을 나타낸 표이다.

| 이름          | 표기법                                            | 정의            | 정의역                                                  | 치역                                |
| ----------- | ---------------------------------------------- | ------------- | ---------------------------------------------------- | --------------------------------- |
| 아크사인        | *y* = arcsin *x* 또는 *y* = sin<sup>-1</sup> *x* | *x* = sin *y* | −1부터 +1                                              | −π/2 ≤ *y* ≤ π/2                  |
| 아크코사인       | *y* = arccos *x* 또는 *y* = cos<sup>-1</sup> *x* | *x* = cos *y* | −1부터 +1                                              | 0 ≤ *y* ≤ π                       |
| 아크탄젠트(역탄젠트) | *y* = arctan *x* 또는 *y* = tan<sup>-1</sup> *x* | *x* = tan *y* | 모든 [실수](https://ko.wikipedia.org/wiki/실수 "wikilink") | −π/2 \< *y* \< π/2                |
| 아크코탄젠트      | *y* = arccot *x* 또는 *y* = cot<sup>-1</sup> *x* | *x* = cot *y* | 모든 [실수](https://ko.wikipedia.org/wiki/실수 "wikilink") | 0 \< *y* \< π                     |
| 아크시컨트       | *y* = arcsec *x* 또는 *y* = sec<sup>-1</sup> *x* | *x* = sec *y* | −∞부터 −1과 1부터 ∞                                       | 0 ≤ *y* \< π/2 or π/2 \< *y* ≤ π  |
| 아크코시컨트      | *y* = arccsc *x* 또는 *y* = csc<sup>-1</sup> *x* | *x* = csc *y* | −∞부터 −1과 1부터 ∞                                       | −π/2 ≤ *y* \< 0 or 0 \< *y* ≤ π/2 |
|             |                                                |               |                                                      |                                   |

정의역을 [복소수](../Page/복소수.md "wikilink")로 두게 되면 위에서 치역의 범위는 [실수부](https://ko.wikipedia.org/wiki/실수부 "wikilink")의 범위가 된다.
