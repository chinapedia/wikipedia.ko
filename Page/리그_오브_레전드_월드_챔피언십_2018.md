**리그 오브 레전드 월드 챔피언십 2018** (League of Legends World Championship 2018)(**2018 롤드컵**)는 라이엇 게임즈가 주최, 주관하는 [리그 오브 레전드 월드 챔피언십의](../Page/리그_오브_레전드_월드_챔피언십.md "wikilink") 8번째 대회이다.

## 개요

  - 주최 : 라이엇 게임즈
  - 주관 : 라이엇 게임즈
  - 후원 : 마스터카드, 에이서, 로지텍, 스테이트팜
  - 주제가 : RISE
  - 기념 스킨 : 챔피언십 카직스
  - 우승팀 기념 스킨 : 피오라, 이렐리아, 카밀, 르블랑, 카이사, 라칸

### 방식

  - 플레이-인 스테이지 : 메이저 상위 4지역 3번 시드 4팀, 마이너 지역 우승팀 8팀이 참가
      - 1라운드 : 4개 조로 나누어 더블 라운드 로빈 방식으로 진행, 각 조 상위 2팀이 2라운드에 진출
      - 2라운드 : 5전 3선승제
  - 그룹 스테이지 : 메이저 지역 12팀, 플레이-인 스테이지 통과한 4팀이 참가하여 4개 조로 나누어 더블 라운드 로빈 방식으로 진행, 각 조 상위 2팀이 결승 토너먼트에 진출
  - 결승 토너먼트 : 5전 3선승제

### 변경된 점

  - 이번 대회부터 베트남 지역(VCS)이 동남아시아 지역(GPL)과 분리가 되어 VCS에서는 그룹 스테이지 1팀, GPL에서는 플레이-인 스테이지 1팀으로 편성된다.
  - 가레나 프리미어 리그(GPL)는 동남아시아 투어(SEA Tour)로 변경되었다.

### 일정

  - 플레이-인 스테이지
      - 1라운드 : 10월 1일(월) \~ 4일(목)
      - 2라운드 : 10월 6일(토) \~ 7일(일)
  - 그룹 스테이지
      - 1주차 : 10월 10일(수) \~ 13일(토)
      - 2주차 : 10월 14일(일) \~ 17일(수)
  - 8강 : 10월 20일(토) \~ 21일(일)
  - 준결승 : 10월 27일(토) \~ 28일(일)
  - 결승 : 11월 3일(토)

### 경기장

  - 플레이-인 스테이지 : 서울 LoL 파크
  - 그룹 스테이지 \~ 8강전 : 부산 [BEXCO](https://ko.wikipedia.org/wiki/BEXCO "wikilink") 오디토리움
  - 4강 : [광주여자대학교](../Page/광주여자대학교.md "wikilink") 유니버시아드관
  - 결승 : [인천문학경기장](../Page/인천문학경기장.md "wikilink")

### 상금

  - 총 상금 : 기본 상금 ($ 2,250,000) + α (크라우드 펀딩 전체 판매 수익의 25%)
  - 우승 : 총 상금의 37.5%
  - 준우승 : 총 상금의 13.5%
  - 준결승 : 총 상금의 7%
  - 8강 : 총 상금의 4%
  - 그룹 스테이지 각 조 3위 : 총 상금의 2.25%
  - 그룹 스테이지 조 공동 3위 : 총 상금의 1.75% (순위 결정전 없음)
  - 그룹 스테이지 각 조 4위 : 총 상금의 1.25%
  - 플레이-인 스테이지 2라운드 : 총 상금의 0.75%
  - 플레이-인 스테이지 1라운드 : 총 상금의 0.5%
