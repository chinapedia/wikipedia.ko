[기하학](https://ko.wikipedia.org/wiki/기하학 "wikilink")에서 **수족삼각형**이란 한 [점](https://ko.wikipedia.org/wiki/점 "wikilink")에서 세 변에 내린 수선의 발들로 이루어진 [삼각형](https://ko.wikipedia.org/wiki/삼각형 "wikilink")이다.

구체적인 설명을 위해 삼각형 *ABC* 와 점 *A, B, C* 가 아닌 한 점 *P* 를 생각해 보자. 그리고 *P* 에서 삼각형의 세 변에 수선의 발을 내리자. (경우에 따라서 삼각형의 변을 연장해야 할 수도 있다.) 세 수선이 변 *BC*, *AC*, *AB* 와 만나는 세 점을 각각 *L*, *M*, *N* 이라고 하면 수족삼각형은 삼각형 *LMN* 이 된다.

점 *P* 와 삼각형 *ABC* 의 상대적인 위치에 따라 몇 가지 특별한 경우가 있다.

  - *P* 가 [수심](https://ko.wikipedia.org/wiki/수심 "wikilink")일 때, *LMN* 은 수심삼각형이다.
  - *P* 가 [내심](https://ko.wikipedia.org/wiki/내심 "wikilink")일 때, *LMN* 은 내접삼각형(intouch triangle)이다.

[right](https://ko.wikipedia.org/wiki/파일:Pedal_Line.svg "wikilink")

만약 *P* 가 삼각형의 외접원 위에 놓여 있으면 *LMN* 은 직선이 된다. 이 직선을 **수족 직선**, 또는 [로버트 심슨](https://ko.wikipedia.org/wiki/로버트_심슨 "wikilink")(Robert Simson)의 이름을 따서 **심슨 직선**이라 부른다.

수족삼각형의 세 꼭짓점은 기존 삼각형의 변을 나눔에 있어서 다음의 식을 만족한다\[1\]

\[AN^2+BL^2+CM^2=NB^2+LC^2+MA^2\]

## 삼선좌표

만약 *P* 의 [삼선좌표](https://ko.wikipedia.org/wiki/삼선좌표 "wikilink")가 *p* : *q* : *r* 이라면 수족삼각형의 세 꼭짓점 *L,M,N*의 삼선좌표는 다음과 같이 주어진다.

  - *L = 0 : q + p* cos C'' : r + p *cos* B''
  - ''M = p + q *cos* C : 0 : r + q *cos* A''
  - ''N = p + r *cos* B : q + r *cos* A : 0''

## 반(反)수족삼각형

반(反)수족삼각형의 한 점 *L*'은 *B* 를 지나고 *BP* 에 수직인 직선과 *C* 를 지나고 *CP* 에 수직인 직선으로 결정된다. 반(反)수족삼각형의 다른 두 점 *M* 과 *N* 도 같은 방식으로 결정된다. 삼선 좌표는 다음과 같이 주어진다.

  - *L*' *= − (q + p* cos'' C)(r + p'' cos'' B) : (r + p'' cos'' B)(p + q'' cos'' C) : (q + p'' cos'' C)(p + r'' cos'' B)''
  - *M*' *= (r + q* cos'' A)(q + p'' cos'' C) : − (r + q'' cos'' A)(p + q'' cos'' C) : (p + q'' cos'' C)(q + r'' cos'' A)''
  - *N*' *= (q + r* cos'' A)(r + p'' cos'' B) : (p + r'' cos'' B)(r + q'' cos'' A) : − (p + r'' cos'' B)(q + r'' cos'' A)''

예를 들어, 내심에 대한 반(反)수족삼각형은 세 [방심](https://ko.wikipedia.org/wiki/방심 "wikilink")을 연결한 삼각형이다.

*P* 가 선분 *BC, CA, AB* 를 연장한 직선 위에 있지 않고, *P*<sup>−1</sup> 가 *P* 의 [등각켤레점](https://ko.wikipedia.org/wiki/등각켤레점 "wikilink")을 가리킨다고 하자. 그러면 *P* 에 대한 수족삼각형은 *P*<sup>−1</sup> 에 대한 수족삼각형과 [중심 닮음이다](https://ko.wikipedia.org/wiki/중심_닮음 "wikilink"). 또한 [닮음 중심은](https://ko.wikipedia.org/wiki/닮음_중심 "wikilink") 다음과 같이 주어진다.

  -
    *ap(p + q* cos'' C)(p + r'' cos'' B) : bq(q + r'' cos'' A)(q + p'' cos'' C) : cr(r + p'' cos'' B)(r + q'' cos'' A)''.

*P* 에 대한 수족삼각형과 *P*<sup>-1</sup> 에 대한 수족삼각형의 넓이의 곱은 삼각형 *ABC* 의 넓이의 제곱과 같다.
