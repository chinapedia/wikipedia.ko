[수학](../Page/수학.md "wikilink")에서, **르장드르 연관 함수** 란, 다음 **연관 르장드르 미분방정식** 의 답으로 얻어지는 함수이다.

\[(1-x^2)\,y'' -2xy' + \left(\ell[\ell+1] - \frac{m^2}{1-x^2}\right)\,y = 0,\,\]

다른 표현:

\[([1-x^2]\,y')' + \left(\ell[\ell+1] - \frac{m^2}{1-x^2}\right)\,y = 0,\,\]

여기에서 기호 \(\ell\) 과 *m* (이들은 일반적으로 복소수까지 확장 할 수 있다.) 은 각각 연관 르장드르 함수의 degree 와 order 로 부른다. 이 방정식들은 \[−1, 1\] 에서 \(\ell\,\) 과 *m* 이 정수이면서 0 ≤ *m* ≤ \(\ell\)일 때만 무한대로 발산하지 않는다.
