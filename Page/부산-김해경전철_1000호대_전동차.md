**부산-김해경전철 1000호대 전동차**는 [부산-김해경전철](https://ko.wikipedia.org/wiki/부산-김해경전철 "wikilink")에서 운행하는 [전동차](https://ko.wikipedia.org/wiki/전동차 "wikilink")이다. 현재 총 2량 25개 편성(50량)이 운행하고 있다.

## 편성

2량 1편성으로 운행한다. [부산-김해경전철](https://ko.wikipedia.org/wiki/부산-김해경전철 "wikilink") 승강장은 4량 1편성을 수용할 수 있도록 [스크린도어](../Page/스크린도어.md "wikilink")가 설계되어 있다.

|  |                                                        |      |  |
|  | ------------------------------------------------------ | ---- |  |
|  | ◇◇                                                     | ◇◇   |  |
|  | 11XX                                                   | 12XX |  |
|  | M1                                                     | M2   |  |
|  | 차량 앞뒤는 차량기지의 루프선으로 인해 사상역 주박 열차가 아니면 다음날 앞뒤가 바뀌어 운행된다. |      |  |
|  | <small>※XX는 편성 번호</small>                              |      |  |
