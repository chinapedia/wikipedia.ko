**럭스**(lux, 기호 **㏓**)는 빛의 [조명도](https://ko.wikipedia.org/wiki/조명도 "wikilink")를 나타내는 [SI 단위이다](../Page/국제단위계.md "wikilink"). 럭스는 [루멘](../Page/루멘.md "wikilink")에서 유도되었고, 루멘은 [칸델라](../Page/칸델라.md "wikilink")에서 유도된 단위이다.

  -
    1 lx = 1 lm/m<sup>2</sup> = 1 cd·sr·m<sup>-2</sup>

## 럭스의 예

| I밝기차                | 예                                                                                             |
| ------------------- | --------------------------------------------------------------------------------------------- |
| 10<sup>-5</sup> lux | 가장 밝은 별([시리우스](../Page/시리우스.md "wikilink"))의 빛\[1\]                                           |
| 10<sup>-4</sup> lux | 하늘을 덮은 완전한 [별빛](https://ko.wikipedia.org/wiki/별빛 "wikilink")\[2\]                             |
| 0.002 lux           | [대기광](https://ko.wikipedia.org/wiki/대기광 "wikilink")이 있는 달 없는 맑은 밤 하늘\[3\]                     |
| 0.01 lux            | 초승달                                                                                           |
| 0.27 lux            | 맑은 밤의 보름달\[4\]\[5\]                                                                           |
| 1 lux               | 열대 [위도](../Page/위도.md "wikilink")를 덮은 보름달\[6\]                                                |
| 3.4 lux             | 맑은 하늘 아래의 어두운 [황혼](https://ko.wikipedia.org/wiki/황혼 "wikilink")\[7\]                          |
| 50 lux              | 거실\[8\]                                                                                       |
| 80 lux              | 복도/화장실\[9\]                                                                                   |
| 100 lux             | 매우 어두운 낮\[10\]                                                                                |
| 320 lux             | 권장 오피스 조명 (오스트레일리아)\[11\]                                                                     |
| 400 lux             | 맑은 날의 [해돋이](../Page/해돋이.md "wikilink") 또는 [해넘이](https://ko.wikipedia.org/wiki/해넘이 "wikilink") |
| 1000 lux            | 인공 조명\[12\]; 일반적인 [TV 스튜디오](https://ko.wikipedia.org/wiki/TV_스튜디오 "wikilink") 조명              |
| 10,000–25,000 lux   | [낮](https://ko.wikipedia.org/wiki/낮 "wikilink") (직사광선이 없을 때)\[13\]                            |
| 32,000–130,000 lux  | 직사광선                                                                                          |
