**2회 진로배 세계바둑최강전**은 [대한민국](../Page/대한민국.md "wikilink"), [중국](../Page/중국.md "wikilink"), [일본](../Page/일본.md "wikilink")의 한국이 우승했다.

## 대회 일정

| 구분        | 일정                            | 장소            |
| --------- | ----------------------------- | ------------- |
| 개막식       |                               | 서울 힐튼호텔 국화룸   |
| 1차전       | 1993년 12월 7일 \~ 1992년 12월 13일 |               |
| 2차전       | 1994년 1월 12일 \~ 1994년 1월 17일  | 중국 베이징 켐핀스키호텔 |
| 3차전       | 1994년 2월 22일 \~ 1994년 2월 23일  | 서울 힐튼호텔 국화룸   |
| 시상식 및 폐막식 |                               |               |
|           |                               |               |

## 참가 선수 명단

  - [일본](../Page/일본.md "wikilink"): 야마히로 히로시, 이시다 요시오, 요다 노리모토, 고마스 히데키, 다케미야 마사키
  - [대한민국](../Page/대한민국.md "wikilink"): [서봉수](../Page/서봉수.md "wikilink") 九단, [정수현](../Page/정수현.md "wikilink") 七단, [유창혁](../Page/유창혁.md "wikilink") 六단, [조훈현](../Page/조훈현.md "wikilink") 九단, [이창호](https://ko.wikipedia.org/wiki/이창호_\(바둑_기사\) "wikilink") 六단
  - [중국](../Page/중국.md "wikilink"): 차오 다위안, 위빈, 마샤오춘, 류사오광, 녜웨이핑
