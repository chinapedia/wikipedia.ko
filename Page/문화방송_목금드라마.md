**문화방송 목금드라마**는 [문화방송](../Page/문화방송.md "wikilink")에서 [목요일](../Page/목요일.md "wikilink")과 [금요일](../Page/금요일.md "wikilink") 밤 10시대에 방송되었던 [텔레비전 드라마이다](../Page/텔레비전_드라마.md "wikilink").

## 방송 시간

| 방송 채널                                  | 방송 기간                       | 방송 시간         | 방송 분량 |
| -------------------------------------- | --------------------------- | ------------- | ----- |
| [MBC TV](../Page/MBC_TV.md "wikilink") | 1981년 4월 2일 \~ 1982년 2월 11일 | 매주 목 밤 10시 5분 | 85분   |
| 1982년 2월 18일 \~ 1983년 3월 11일           | 매주 목,금 밤 9시 50분             | 45분           |       |
| 1983년 3월 31일 \~ 1983년 10월 28일          | 매주 목,금 밤 10시 10분            | 60분           |       |
|                                        |                             |               |       |
