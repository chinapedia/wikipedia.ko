**프로세싱**(Processing)은 [컴퓨터 프로그래밍의](../Page/컴퓨터_프로그래밍.md "wikilink") 본질을 시각적 개념으로 프로그래머가 아닌 사람들에게 교육할 목적으로 [뉴 미디어 아트](https://ko.wikipedia.org/wiki/뉴_미디어_아트 "wikilink"), [시각 디자인](https://ko.wikipedia.org/wiki/시각_디자인 "wikilink") 공동체를 위해 개발된 오픈 소스 프로그래밍 언어이자 [통합 개발 환경](../Page/통합_개발_환경.md "wikilink")(IDE)이다. 2001년 [MIT 미디어 연구소에서](https://ko.wikipedia.org/wiki/MIT_미디어_연구소 "wikilink") [케이시 리아스와](https://ko.wikipedia.org/wiki/케이시_리아스 "wikilink") [벤자민 프라이가](https://ko.wikipedia.org/wiki/벤자민_프라이 "wikilink") 시작하였다.\[1\]

## 헬로 월드

``` java
void setup() {
  println("Hello World!");
}
```

이것이 올바른 헬로 월드 프로그램이지만, 다음의 코드가 프로세싱 언어를 잘 표현해 주는 좋은 예라고 할 수 있다.

``` java
void setup() {
 PFont font = loadFont("myfont.vlw");
 textFont(font,20);
}

void draw() {
 text("Hello World!", 30,50);
}
```
