**구글 앱 엔진**(Google App Engine, **GAE**)은 [구글](https://ko.wikipedia.org/wiki/구글 "wikilink")에서 제공하는 서비스 중 하나이며, 웹 응용 프로그램을 PHP · Python · Java · Go 언어를 사용하여 개발하고 구글의 인프라에서 실행하고 버전 관리할 수 있다. [구글 클라우드 플랫폼의](https://ko.wikipedia.org/wiki/구글_클라우드_플랫폼 "wikilink") 일부이다.

## 사용 할당량

### 하드 리밋

| 할당           | 제한                              |
| ------------ | ------------------------------- |
| 요청 당 시간      | 일반 요청 시 60초, 태스크에 10분, 백엔드에 무제한 |
| HTTP 응답 크기   | 64 MB                           |
| 데이터스토어 항목 크기 | 1 MB                            |
