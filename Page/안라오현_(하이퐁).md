**안라오현**()은 [베트남](../Page/베트남.md "wikilink") [하이퐁시](https://ko.wikipedia.org/wiki/하이퐁시 "wikilink")의 [현이다](../Page/후옌.md "wikilink"). 2003년을 기준으로 안라오현의 면적은 114 km², 인구는 124,592명이었다.\[1\]

## 행정구역

안라오현은 2개의 [티쩐](../Page/티쩐.md "wikilink")(市鎭)과 15개의 싸(社)를 관할하고 있다.

### 티쩐

  - 티쩐 안라오（Thị trấn An Lão / 市鎭安老）
  - 티쩐 쯔엉선（Thị trấn Trường Sơn / 市鎭長山）

### 싸

  - 안타이 (Xã An Thái / 安泰社)
  - 안탕 (Xã An Thắng / 安勝社)
  - 안토 (Xã An Thọ / 安壽社)
  - 안띠엔 (Xã An Tiến / 安進社)
  - 밧짱 (Xã Bát Tràng / 八場社)
  - 찌엔탕 (Xã Chiến Thắng / 戰勝社)
  - 미이덕 (Xã Mỹ Đức / 美德社)
  - 꽝흥 (Xã Quang Hưng / 光興社)
  - 꽝쭝 (Xã Quang Trung / 光中社)
  - 끄옥쭈언 (Xã Quốc Tuấn / 國俊社)
  - 떤던 (Xã Tân Dân / 新民社)
  - 떤비엔 (Xã Tân Viên / 新園社)
  - 타이선 (Xã Thái Sơn / 泰山社)
  - 쯔엉토 (Xã Trường Thọ / 長壽社)
  - 쯔엉타인 (Xã Trường Thành / 長成社)
