《**ZARD Request Best \~beautiful memory\~**》는 일본의 밴드 [ZARD](https://ko.wikipedia.org/wiki/ZARD "wikilink")의 네 번째 [베스트 음반으로](https://ko.wikipedia.org/wiki/베스트_음반 "wikilink"), CD 2장으로 구성되어 있으며 2008년 1월 23일 발매되었다. ([CD](https://ko.wikipedia.org/wiki/콤팩트_디스크 "wikilink"): JBCJ-9027, 9028) 2007년 6월 25일부터 팬들의 투표를 통해 앨범에 수록될 곡들을 응모했으며, 약 50만 통의 응모 중에서 가장 많이 선택된 30곡이 수록되었다.\[1\]
