**인천숭의초등학교**는 [인천광역시](https://ko.wikipedia.org/wiki/인천광역시 "wikilink") [미추홀구](../Page/미추홀구.md "wikilink") [숭의동](../Page/숭의동.md "wikilink")에 있는 공립 초등학교이다.\[1\]

## 연혁

| 1937년   | 12월 10일              | **인천대화공립보통학교** 개교(설립 4월 10일)\[2\]\[3\] |
| ------- | -------------------- | -------------------------------------- |
| 1938년   | 4월 1일                | **인천대화공립심상소학교** 개칭                     |
| 1941년   | 4월 1일                | **인천대화공립국민학교** 개칭                      |
| 1946년   | 10월 11일              | **인천숭의국민학교** 개칭                        |
| 1996년   | 3월 1일                | 현재의 교명으로 변경                            |
| 2001년   | 4월 6일                | 학교교사 대보수 공사 완공                         |
| 2004년   | 12월 17일              | 숭의 디지털도서관 개관                           |
| 2006년   | 6월 14일               | 학교 공원화 사업(도란터) 준공식                     |
| 2007년   | 8월 1일                | 제2과학실 현대화 인테리어 공사                      |
| 2009년   | 3월 1일                | 52학급 편성, 동관 3층 증축                      |
| 2010년   | 3월 22일               | 급식소 개소                                 |
| 10월 19일 | 야구실내연습장 개장           |                                        |
| 2011년   | 11월 4일               | 숭의 창의인성예술제 발표(격년 실시)                   |
| 2013년   | 6월 11일               | 어학실 현대화사업 리모델링 인테리어 공사                 |
| 2014년   | 9월 1일                | 제25대 류인욱 교장 부임                         |
| 2016년   | 2월 13일               | 제74회 졸업식(207명, 누계 31,867명)             |
| 3월 2일   | 49학급 편성(특수학급 2학급 포함) |                                        |
| 11월 10일 | 숭의 Dream Up 행복축제     |                                        |
| 2017년   | 3월 2일                | 49학급 편성(특수학급 2학급 포함)                   |
