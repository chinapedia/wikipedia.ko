**목천초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [천안시](../Page/천안시.md "wikilink") [동남구](../Page/동남구.md "wikilink") [목천읍](https://ko.wikipedia.org/wiki/목천읍 "wikilink") 서리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1913년        | 6월 1일                             | 목천공립보통학교 개교(설립인가 4월 1일, 4년제 2학급) |
| ------------ | --------------------------------- | -------------------------------- |
| 1934년        | 5월 4일                             | 도장간이학교 설치                        |
| 1938년        | 4월 1일                             | 목천공립심상소학교로 개칭                    |
| 1939년        | 8월 2일                             | 6년제로 개편                          |
| 1941년        | 4월 1일                             | 목천공립국민학교로 개칭                     |
| 1947년        | 9월 1일                             | 신계분실 설치                          |
| 1949년        | 1월 22일                            | 신계분교장 인가                         |
| 1950년        | 5월 1일                             | 덕전분교장 설치                         |
| 1981년        | 3월 1일                             | 병설유치원 개원                         |
| 1991년        | 3월 1일                             | 덕전국민학교 본교에 통폐합                   |
| 1996년        | 3월 1일                             | 목천초등학교로 명칭 개명                    |
| 2013년        | 3월 28일                            | 느티나무 작은 도서관 개관                   |
| 6월 1일        | 목천초등학교 개교 100주년 기념식               |                                  |
| 2014년        | 3월 1일                             | 충남행복공감학교 지정                      |
| 2017년        | 2월 7일                             | 제101회 졸업장 수여식                    |
| 3월 1일        | 목천초등학교 13학급(특수 2학급)               |                                  |
| 병설유치원 1학급 편성 |                                   |                                  |
| 2018년        | 2월 28일                            | 제 102회 졸업장 수여식                   |
| 3월 1일        | 목천초등학교 10학급(특수 2학급), 병설유치원 1학급 편성 |                                  |
