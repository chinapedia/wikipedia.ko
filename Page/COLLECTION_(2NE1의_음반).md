〈**COLLECTION**〉은 [대한민국](../Page/대한민국.md "wikilink")의 음악 그룹 [2NE1](../Page/2NE1.md "wikilink")의 일본 첫 번째 정규 음반이다. 2012년 03월 28일 [YGEX](../Page/YGEX.md "wikilink")를 통해 4가지 버전으로 발매되었다.

## 발매 버전

  - Type A (2CD + DVD + PHOTOBOOK)
  - Type B (CD + DVD)
  - Type C (CD ONLY)
  - Type D (CD + GOODS)
