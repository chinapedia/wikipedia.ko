**당진시의 시내버스**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [당진시](https://ko.wikipedia.org/wiki/당진시 "wikilink")를 중심으로 운행하는 버스를 이용한 대중교통 수단으로, 운수업체로는 [당진여객](https://ko.wikipedia.org/wiki/당진여객 "wikilink")이 유일하다.

## 운임

2018년 7월 기준 요금이다. 하차후 1시간 이내 1회에 한해 무료환승이 가능하다.

| 종류                  | 나이            | 카드 (원) | 현금 (원) |
| ------------------- | ------------- | ------ | ------ |
| 일반                  | 어른 (만 19세 이상) | 1350   | 1400   |
| 청소년 (만 13세 \~ 18세)  | 900           | 1100   |        |
| 어린이 (만 6세 \~ 12세)   | 650           | 700    |        |
| 좌석 (920번,960번,961번) | 어른 (만 19세 이상) | 1750   | 1800   |
| 청소년 (만 13세 \~ 18세)  | 1240          | 1440   |        |
| 어린이 (만 6세 \~ 12세)   | 850           | 900    |        |

시계외구간은 킬로미터당 116.14원이나 별도로 시계외요금은 받지 않는데다가

시내버스 전체 운행구간이 성인기준1400원이다.

## 당진시 차적의 버스 노선

당진군이었던 시절인 [2005년](https://ko.wikipedia.org/wiki/2005년 "wikilink") [3월 1일부터](https://ko.wikipedia.org/wiki/3월_1일 "wikilink") 당진 지역을 운행하는 모든 노선에 번호제를 도입하였다.\[2\]

운행 노선에서 다음 명칭들은 다른 명칭으로도 쓰인다.

  - 시내 : 구터미널 (구터)
  - [터미널](https://ko.wikipedia.org/wiki/당진종합버스터미널 "wikilink") : 신터미널

**굵은 글씨**로 표기된 지역은 회차지이다.
