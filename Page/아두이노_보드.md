[섬네일](https://ko.wikipedia.org/wiki/파일:ARDU-UNO-03-Front_\(16421989212\).jpg "wikilink") **아두이노 보드**는 [아두이노](../Page/아두이노.md "wikilink")를 기반으로 여러 종류가 생산되어 판매되고 있다. 주로 AVR의 아트메가(ATmega)를 위주로 만들어지고, ARM 계열도 포함되어 있다.\[1\]

|            | MCU                                                                                                                           | 아두이노 보드\[2\] |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------- | ------------ |
| AVR        |                                                                                                                               |              |
| ATmega168  | Pro(168), Mini(168), LilyPad (168V)                                                                                           |              |
| ATmega328  | UNO, ETHERNET, Fio, Nano, Pro(328), Mini(328, Rev5), Pro Mini, LilyPad/Simple (328V)                                          |              |
| ATmega2560 | Mega 2560, Mega ADK                                                                                                           |              |
| ATmega32U4 | YÙN, [Leonardo](https://ko.wikipedia.org/wiki/Arduino_Leonardo "wikilink"), Esplora, Micro, LilyPad USB, ROBOT(Control,Motor) |              |
| ATtiny85   | GEMMA                                                                                                                         |              |
| ARM        |                                                                                                                               |              |
| Cortex-M0+ | Zero, Zero Pro, M0, M0 PRO                                                                                                    |              |
| Cortex-M3  | Due                                                                                                                           |              |

### 쉴드 (Shield)

쉴드는 아두이노 보드의 마이크로컨트롤러와 호환 가능한 여러 부가 모듈을 커넥터로 연결하여 기능을 추가하는 부품이다. 주로 입출력 장치를 부가한다.
