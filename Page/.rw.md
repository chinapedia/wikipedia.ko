**.rw**는 [르완다](../Page/르완다.md "wikilink")의 [인터넷](../Page/인터넷.md "wikilink") [국가 코드 최상위 도메인이다](https://ko.wikipedia.org/wiki/국가_코드_최상위_도메인 "wikilink").

## 2차 도메인

| 일반 2차 도메인 존 | 단체 유형                 |
| ----------- | --------------------- |
| .rw         | 범용 도메인 이름             |
| .co.rw      | 상용, 사업체               |
| .org.rw     | 영리 목적이 아니고 정부 기관이 아님  |
| .net.rw     | 네트워크 인프라스트럭처 제공자      |
| .ac.rw      | 르완다 학술 기관.            |
| .gov.rw     | 르완다 정부 기관.            |
| .mil.rw     | 르완다 방위군(RDF)을 위해 예약됨. |
| .coop.rw    | 르완다 협동조합.             |
| .ltd.rw     | 르완다 유한 책임 회사.         |
