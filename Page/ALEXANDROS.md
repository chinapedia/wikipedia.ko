**\[ALEXANDROS\]**는 일본의 록 밴드이다. 2014년 3월까지 **\[Champagne\]**이라는 이름을 사용했다.

## 일원

### 현재 일원

  - 가와카미 요헤이 (, ) - 보컬, 기타 (2007년-현재)
  - 이소베 히로유키 (, ) - 베이스 기타, 코러스 (2007년-현재)
  - 시라이 마사키 (, ) - 기타 (2007년-현재)
  - 쇼무라 사토야스 (, ) - 드럼 (2010년-현재)

### 전 일원

  - 이시카와 히로키 (, ) - 드럼 (2007년-2010년)
