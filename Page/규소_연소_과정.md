**[규소](https://ko.wikipedia.org/wiki/규소 "wikilink") 연소 과정**(silicon burning process)은 질량이 적어도 태양의 12배인 큰 별에서 매우 짧은 시간 동안\[1\] 일어나는 [핵융합](../Page/핵융합.md "wikilink")반응이다. 규소 연소는 항성이 [헤르츠스프룽-러셀 도표상의](https://ko.wikipedia.org/wiki/헤르츠스프룽-러셀_도표 "wikilink") [주계열성](https://ko.wikipedia.org/wiki/주계열성 "wikilink")기간에서 그들의 연료를 오랜기간 소모한 후 수명이 다하기 직전에 일어난다. 규소연소과정은 항성의 중력이 항성의 핵을 27–35 억켈빈 까지 끌어올렸을 때 일어난다. 정확한 온도는 질량에 따라 다르다. 항성이 규소 연소를 끝냈을 때, [II형 초신성으로의](https://ko.wikipedia.org/wiki/초신성#II형_초신성 "wikilink") 폭발이 일어나는 것이 알려져 있다.

## 핵과정과 [알파 과정](https://ko.wikipedia.org/wiki/알파_과정 "wikilink")

질량이 태양 질량의 3배 이하인 보통 별들은 [수소](../Page/수소.md "wikilink")를 헬륨으로 융합시킨다. 중간 질량 (태양의 3배 이상, 8-11배 이하)의 항성은 [헬륨](../Page/헬륨.md "wikilink")을 다른 [탄소](../Page/탄소.md "wikilink")의 핵으로 융합시킨다. 그보다 더 큰 질량의 항성은 그들의 질량으로 인한 위치에너지로 인해 탄소를 융합시킬 수 있다. 그런 별에서는 핵이 6억켈빈으로 가열되어 새로운 핵을 만든다.

|  |                                                                                          |   |                                                                  |
|  | ---------------------------------------------------------------------------------------- | - | ---------------------------------------------------------------- |
|  | <sup>12</sup>[C](../Page/탄소.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")  | → | <sup>16</sup>[O](../Page/산소.md "wikilink")                       |
|  | <sup>16</sup>[O](../Page/산소.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")  | → | <sup>20</sup>[Ne](../Page/네온.md "wikilink")                      |
|  | <sup>20</sup>[Ne](../Page/네온.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink") | → | <sup>24</sup>[Mg](https://ko.wikipedia.org/wiki/마그네슘 "wikilink") |

이다.

원소의 종류는 양성자수에 의해 결정되며, 중성자가 붙으면 비슷한 다른 [동위원소](https://ko.wikipedia.org/wiki/동위원소 "wikilink")가 생길 수 있다.

|  |                                                                                                               |   |                                                                |
|  | ------------------------------------------------------------------------------------------------------------- | - | -------------------------------------------------------------- |
|  | <sup>24</sup>[Mg](https://ko.wikipedia.org/wiki/마그네슘 "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink") | → | <sup>28</sup>[Si](https://ko.wikipedia.org/wiki/규소 "wikilink") |

더 큰 항성에서는 오직 [황](../Page/황.md "wikilink")이나 [규소](https://ko.wikipedia.org/wiki/규소 "wikilink")만을 그들의 핵에 가지고 있다. 핵이 27-35억 켈빈까지 가열되면 규소연소과정이 시작된다. 규소 연소는 [알파 과정을](https://ko.wikipedia.org/wiki/알파_과정 "wikilink") 수반한다. 즉, 각 단계에서 [헬륨](../Page/헬륨.md "wikilink")원자핵이 핵에 달라붙는 방식으로 새로운 원소를 합성한다.

|  |                                                                                                             |   |                                                                |
|  | ----------------------------------------------------------------------------------------------------------- | - | -------------------------------------------------------------- |
|  | <sup>28</sup>[Si](https://ko.wikipedia.org/wiki/규소 "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink") | → | <sup>32</sup>[S](../Page/황.md "wikilink")                      |
|  | <sup>32</sup>[S](../Page/황.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")                      | → | <sup>36</sup>[Ar](../Page/아르곤.md "wikilink")                   |
|  | <sup>36</sup>[Ar](../Page/아르곤.md "wikilink")+ <sup>4</sup>[He](../Page/헬륨.md "wikilink")                    | → | <sup>40</sup>[Ca](https://ko.wikipedia.org/wiki/칼슘 "wikilink") |
|  | <sup>40</sup>[Ca](https://ko.wikipedia.org/wiki/칼슘 "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink") | → | <sup>44</sup>[Ti](../Page/타이타늄.md "wikilink")                  |
|  | <sup>44</sup>[Ti](../Page/타이타늄.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")                  | → | <sup>48</sup>[Cr](../Page/크로뮴.md "wikilink")                   |
|  | <sup>48</sup>[Cr](../Page/크로뮴.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")                   | → | <sup>52</sup>[Fe](../Page/철.md "wikilink")                     |
|  | <sup>52</sup>[Fe](../Page/철.md "wikilink") + <sup>4</sup>[He](../Page/헬륨.md "wikilink")                     | → | <sup>56</sup>[Ni](../Page/니켈.md "wikilink")                    |

[니켈](../Page/니켈.md "wikilink")-56은 6.02일의 반감기를 가지고 [코발트](../Page/코발트.md "wikilink")-56으로, 다시 77.3일의 반감기로 [철](../Page/철.md "wikilink")-56으로 붕괴한다. 하지만 질량이 큰 별내부에서 니켈이 붕괴할 수 있는 시간은 몇분 뿐이다. 비록 철-58과 니켈–62가 [철](../Page/철.md "wikilink")-56보다 핵자당 질량이 적지만\[2\], 56은 알파 과정에서 생기는 원자핵중 단위핵자당 질량이 가장 작은 질량수이다. 다음 산물인 [아연](../Page/아연.md "wikilink")-60은 질량이 조금더 크기 때문에 에너지를 흡수한다. 중력이 커서 50억 켈빈까지 가열되기 때문에 붕괴가 지연되지만, 결국 융합에 의해 생성되는 열이 없어져 수초만에 붕괴가 일어난다. 별의 핵은 [중성자별](https://ko.wikipedia.org/wiki/중성자별 "wikilink")이나 [블랙홀](https://ko.wikipedia.org/wiki/블랙홀 "wikilink")이 되고, 외부는 [II형 초신성이](https://ko.wikipedia.org/wiki/초신성#II형_초신성 "wikilink") 된다. 초신성 폭발은 중성자 다발을 분출하고, [r-과정](https://ko.wikipedia.org/wiki/r-과정 "wikilink")에 의해 새 원소를 합성한다.
