**의령군의 농어촌버스**는 [경상남도](https://ko.wikipedia.org/wiki/경상남도 "wikilink") [의령군](../Page/의령군.md "wikilink")의 대중교통 수단 중심으로 운행하는 버스를 이용한 대중교통 수단이다. 운영 업체는 [경전여객](../Page/경전여객.md "wikilink")이 유일하며, 마을버스 업체로는 [의령버스](https://ko.wikipedia.org/wiki/의령버스 "wikilink")도 있다.

## 요금

2013년 3월 2일 인상된 요금 기준이다.\[1\] 기본요금은 2013년 1월 12일부터 먼저 인상되었다.

| 종류                 | 나이            | 현금 (원) | 카드 (원) |
| ------------------ | ------------- | ------ | ------ |
| 일반                 | 어른 (만 19세 이상) | 1,150  | 1,050  |
| 청소년 (만 13세 \~ 18세) | 850           | 800    |        |
| 어린이 (만 7세 \~ 12세)  | 600           | 550    |        |
| 마을                 | 어른(만 19세 이상)  | 500    |        |
| 청소년 (만 13세 \~ 18세) | 400           |        |        |
| 어린이 (만 7세 \~ 12세)  | 250           |        |        |

  - 거리비례제로 운영되어 의령읍 경계를 포함한 10km 거리까지 기본 요금으로 이용할 수 있으며, 의령읍 경계를 벗어난 10km 이상 이용하는 거리에 대해서 1km 당 116.14원씩 추가되고 소수점이하는 절삭한다. 청소년은 20%, 어린이는 50% 할인된다.
  - [2013년](../Page/2013년.md "wikilink") [12월 1일](../Page/12월_1일.md "wikilink") 부터 [교통카드](../Page/교통카드.md "wikilink")를 이용해 버스를 승차할 수 있다. 현재 [캐시비](../Page/캐시비.md "wikilink") 계열 교통카드가 통용되며, 후불 교통카드로 이용할 수 있다. 교통카드를 사용할 경우 기본요금을 제외한 거리비례 요금의 10%가 할인된다. (예 : 버스요금 4,000원 → 400원 할인)\[2\]

### [마을버스](../Page/마을버스.md "wikilink")

[의령버스터미널](../Page/의령버스터미널.md "wikilink")을 기점으로 의령 읍내를 순환하여 의령터미널로 돌아오는 마을버스가 운행하고 있다. 운영 업체는 [의령버스](https://ko.wikipedia.org/wiki/의령버스 "wikilink")가 유일하며 의령버스터미널에서 이용하려면 터미널 앞에서 승차한다.
