**클린**(Clean)은 [범용](https://ko.wikipedia.org/wiki/범용_프로그래밍_언어 "wikilink") [순수 함수형](https://ko.wikipedia.org/wiki/순수_함수형_언어 "wikilink") [프로그래밍 언어이다](../Page/프로그래밍_언어.md "wikilink"). 언어의 활동적인 개발 역사 대부분에서 **컨커런트 클린**(Concurrent Clean)이라는 이름이 사용되었으나 어느 지점에서 이 이름의 사용은 중단되었다. 클린은 1987년 Nijmegen의 Radboud University의 연구원 단체에 의해 개발되었다.

## 기능

클린이라는 언어는 1987년에 처음 등장했으며 지금도 계속 개발되고 있다.\[1\] [하스켈](../Page/하스켈.md "wikilink")과 수많은 속성을 공유하고 있다: [참조 투명성](../Page/참조_투명성.md "wikilink"), [리스트 캄프리헨션](../Page/리스트_캄프리헨션.md "wikilink"), [가드](https://ko.wikipedia.org/wiki/가드 "wikilink")(guard), [쓰레기 수집](../Page/쓰레기_수집_\(컴퓨터_과학\).md "wikilink"), [고차 함수](../Page/고차_함수.md "wikilink"), [커링](../Page/커링.md "wikilink"), [느긋한 계산법](../Page/느긋한_계산법.md "wikilink").

윈도우에서는 클린 배포판에 [통합 개발 환경](../Page/통합_개발_환경.md "wikilink")(IDE)이 포함되어 있다.

[피보나치 수](../Page/피보나치_수.md "wikilink"):

``` clean
  module fibonacci
  fib 0 = 0
  fib 1 = 1
  fib n = fib (n - 2) + fib (n - 1)
  Start = fib 7
```

``` clean
  (^) infixr 8 :: Int Int -> Int
  (^) x 0 = 1
  (^) x n = x * x ^ (n-1)
```

## 컴파일

1.  소스 파일(.icl)과 정의 파일(.dcl)이 클린의 기초적인 일종인 코어 클린(Core Clean)으로 변환된다.
2.  코어 클린은 [C와](../Page/C_\(프로그래밍_언어\).md "wikilink") 클린으로 구현된 클린의 플랫폼 독립적 바이트코드(.abc)로 변환된다.
3.  바이트코드는 [C를](../Page/C_\(프로그래밍_언어\).md "wikilink") 사용하여 오브젝트 코드(.o)로 변환된다.
4.  오브젝트 코드는 모듈과 런타임 시스템에서 다른 파일들과 링크되며 클린의 일반적인 실행 파일로 변환된다.

## 플랫폼

클린은 [마이크로소프트 윈도우](../Page/마이크로소프트_윈도우.md "wikilink"), [매킨토시](../Page/매킨토시.md "wikilink"), [솔라리스](../Page/솔라리스_\(운영_체제\).md "wikilink"), [리눅스](../Page/리눅스.md "wikilink")에서 사용할 수 있다.
