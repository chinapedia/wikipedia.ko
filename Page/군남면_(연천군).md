**군남면**(郡南面)은 대한민국 [경기도](https://ko.wikipedia.org/wiki/경기도 "wikilink") [연천군](../Page/연천군.md "wikilink")의 [면이다](../Page/면_\(행정_구역\).md "wikilink").

| 옥계리 | 玉溪里 | 옥계1리, 옥계2리, 옥계3리 |            |
| 삼거리 | 三巨里 | 삼거1리, 삼거2리       | 행정복지센터 소재지 |
| 왕림리 | 旺林里 | 왕림리              |            |
| 선곡리 | 仙谷里 | 선곡리              |            |
| 황지리 | 黃地里 | 황지리              |            |
| 남계리 | 楠溪里 | 남계1리, 남계2리       |            |
