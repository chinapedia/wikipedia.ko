**게마인자메 노름다타이**(, , **GND**)는 [독일 국립도서관이](https://ko.wikipedia.org/wiki/독일_국립도서관 "wikilink") 운영하는 [전거 파일이다](https://ko.wikipedia.org/wiki/전거_파일 "wikilink").

## GND 엔티티 종류

GND에는 7개의 주요 엔티티가 있다:\[1\]

| 종류    | 의미          |
| ----- | ----------- |
| **p** | 사람 (개인)     |
| **n** | 사람 (개인이 아님) |
| **k** | 회사          |
| **v** | 사건          |
| **w** | 작품          |
| **s** | 시사 용어       |
| **g** | 지질학적 장소     |
