**발라**()는 [셀프 호스팅 컴파일러를](https://ko.wikipedia.org/wiki/셀프_호스팅_컴파일러 "wikilink") 가진 [객체 지향](https://ko.wikipedia.org/wiki/객체_지향 "wikilink") [프로그래밍 언어이다](https://ko.wikipedia.org/wiki/프로그래밍_언어 "wikilink"). 발라 컴파일러는 [GObject](https://ko.wikipedia.org/wiki/GObject "wikilink") 시스템을 사용하여 [C](https://ko.wikipedia.org/wiki/C_\(프로그래밍_언어\) "wikilink") 소스코드를 생성한다. 생성된 C 소스코드는 단지 발라 언어의 실행만을 위한 게 아니라 실제로 다른 C 언어 프로젝트에서도 사용할 수 있게끔 적절한 API를 가지고 있다.

발라는 문법적으로 [자바나](https://ko.wikipedia.org/wiki/자바_\(프로그래밍_언어\) "wikilink") [C\#과](https://ko.wikipedia.org/wiki/C_샤프 "wikilink") 비슷하며, [익명 함수](https://ko.wikipedia.org/wiki/익명_함수 "wikilink"), 시그널, 프로퍼티, [제네릭](https://ko.wikipedia.org/wiki/제네릭_프로그래밍 "wikilink"), [자동 메모리 관리](https://ko.wikipedia.org/wiki/쓰레기_수집 "wikilink"), [예외 처리](https://ko.wikipedia.org/wiki/예외_처리 "wikilink"), [자료형 추론](https://ko.wikipedia.org/wiki/자료형_추론 "wikilink"), [foreach문](https://ko.wikipedia.org/wiki/Foreach_루프 "wikilink") 등의 기능을 제공한다.

## 예제

### Hello world 프로그램

``` vala
void main (string[] args) {
    print("Hello World!");
}
```

이 예제는 다음과 같이 객체 지향 방식으로도 작성될 수 있다.

``` vala
public class Sample : Object {
    static void main (string[] args) {
        print("Hello World!");
    }
}
```

또한 아래와 같이 스크립트 형태로도 작성될 수 있다. 이는 실험적인 기능이며 암시적으로 main 메소드를 가지고 있다.

``` vala
print("Hello World!");
```

### GUI 버전 Hello World

[GTK+](https://ko.wikipedia.org/wiki/GTK+ "wikilink")를 사용한 예제이다

``` vala
// hellogtk.vala
using Gtk;

void main (string[] args) {
    Gtk.init(ref args);

    var window = new Window();
    window.title = "Hello, World!";
    window.border_width = 10;
    window.window_position = WindowPosition.CENTER;
    window.set_default_size(350, 70);
    window.destroy.connect(Gtk.main_quit);

    var label = new Label("Hello, World!");

    window.add(label);
    window.show_all();

    Gtk.main ();
}
```

컴파일:

``` bash
valac --pkg gtk+-3.0 hellogtk.vala
```
