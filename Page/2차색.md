## 전통적인 채색 규칙 (RYB)

|                                |                                         |    |
| ------------------------------ | --------------------------------------- | -- |
|                                |                                         |    |
| [빨강](../Page/빨강.md "wikilink") | <span style="color:#FF003C;">(●)</span> | \+ |
| [노랑](../Page/노랑.md "wikilink") | <span style="color:#FFE000;">(●)</span> | \+ |
| [파랑](../Page/파랑.md "wikilink") | <span style="color:#0078FF;">(●)</span> | \+ |
|                                |                                         |    |

RGB 색 공간에 색을 첨가하면, 더 밝은 색을 얻기 위해는 어두운 단계의 색부터 출발한다. RYB는 안료를 사용하기 때문에 가산혼합을 사용하지 않아 RYB 색 체계를 사용하여 색을 혼합하면 좀 더 어두운 색을 얻을 수 있다.
