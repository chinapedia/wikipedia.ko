《**오쓰\! 싸워라\! 응원단**》()은 [닌텐도](../Page/닌텐도.md "wikilink")가 발매한 [닌텐도 DS용](../Page/닌텐도_DS.md "wikilink") 응원 리듬 액션 게임이다.

## 등장 캐릭터

### 석양마을

응원단원은 모두 교복을 착용하고 있고, 응원단의 마크도 학교의 마크처럼 보이지만, 단원들이 학생이라는 설정은 없고, 「응원단」이라고 하는 단체와 함께 모든 정체가 수수께끼에 싸여 있다.(차기작인 [불타올라라\! 열혈 리듬혼 오쓰\! 싸워라\! 응원단 2에서는](../Page/불타올라라!_열혈_리듬혼_오쓰!_싸워라!_응원단_2.md "wikilink") 전작으로부터 6년 후의 에피소드가 있지만 타나카가 아직도 신입인 등 이상한 점이 있다.)

## 난이도

난이도는 4개의 난이도 모드가 있는데, 난이도가 올라갈수록 세심한 터치펜 컨트롤이 필요하며, 히트 마크는 늘어나지만, 슬라이드 마크가 줄어든다.

| 난이도           | 설명                |
| ------------- | ----------------- |
| 명칭            | 다른 명칭             |
| **가볍게 응원모드**  | 신입단원 모드 (Easy)    |
| **과감하게 응원모드** | 리더 모드 (Normal)    |
| **격렬하게 응원모드** | 단장 모드 (Hard)      |
| **화려하게 응원모드** | 치어걸스 모드 (Extreme) |

### 난이도별 응원단원

| 난이도명                | 응원단원                       |
| ------------------- | -------------------------- |
| 가볍게 응원모드 (Easy)     | 다나카 하지메, 스즈키 잇테츠, 사이토 아츠시  |
| 과감하게 응원모드 (Normal)  | 잇폰기 류타, 스즈키 잇테츠, 사이토 아츠시   |
| 격렬하게 응원모드 (Hard)    | 도메키 카이, 스즈키 잇테츠, 사이토 아츠시   |
| 화려하게 응원모드 (Extreme) | 아메미야 사야카, 안나 린드허스트, 칸다 아오이 |
