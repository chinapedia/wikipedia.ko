**17회 TV 바둑 아시아 선수권대회**는 [대한민국](https://ko.wikipedia.org/wiki/대한민국 "wikilink"), [중국](https://ko.wikipedia.org/wiki/중국 "wikilink"), [일본](https://ko.wikipedia.org/wiki/일본 "wikilink")의 속기전 상위 입상자가 참가한 바둑 기전이다. 결승에서는 [일본](https://ko.wikipedia.org/wiki/일본 "wikilink")의 [장쉬](../Page/장쉬.md "wikilink")가 [대한민국](https://ko.wikipedia.org/wiki/대한민국 "wikilink")의 [조한승](https://ko.wikipedia.org/wiki/조한승 "wikilink")을 꺾고 우승했다.

## 대회 일정

| 구분             | 일정     | 장소               |
| -------------- | ------ | ---------------- |
| 개막식            | 6월 13일 | 중국 베이징 화룬(華潤) 호텔 |
| 1회전 1\~2국      | 6월 14일 |                  |
| 1회전 3국, 준결승 1국 | 6월 15일 |                  |
| 준결승 2국         | 6월 16일 |                  |
| 결승전, 시상식 및 폐막식 | 6월 17일 |                  |
|                |        |                  |

## 참가 선수 명단

  - [일본](https://ko.wikipedia.org/wiki/일본 "wikilink"): [장쉬](../Page/장쉬.md "wikilink")([52기 NHK배](https://ko.wikipedia.org/wiki/52기_NHK배 "wikilink") 우승자), [요다 노리모토](../Page/요다_노리모토.md "wikilink")([52기 NHK배](https://ko.wikipedia.org/wiki/52기_NHK배 "wikilink") 준우승자)
  - [대한민국](https://ko.wikipedia.org/wiki/대한민국 "wikilink"): [이창호](https://ko.wikipedia.org/wiki/이창호 "wikilink")([23기 KBS 바둑왕전](https://ko.wikipedia.org/wiki/23기_KBS_바둑왕전 "wikilink") 우승자), [조한승](https://ko.wikipedia.org/wiki/조한승 "wikilink")([23기 KBS 바둑왕전](https://ko.wikipedia.org/wiki/23기_KBS_바둑왕전 "wikilink") 준우승자)
  - [중국](https://ko.wikipedia.org/wiki/중국 "wikilink"): [위빈](https://ko.wikipedia.org/wiki/위빈_\(바둑_기사\) "wikilink")(16회 전기 우승자 준결승 시드), [류싱](https://ko.wikipedia.org/wiki/류싱 "wikilink")([CCTV배](https://ko.wikipedia.org/wiki/CCTV배 "wikilink") 우승자), [구리](https://ko.wikipedia.org/wiki/구리_\(바둑_기사\) "wikilink")([CCTV배](https://ko.wikipedia.org/wiki/CCTV배 "wikilink") 준우승자)
