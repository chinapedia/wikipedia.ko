**코시-오일러 방정식**()은 선형 동차 [상미분 방정식이다](https://ko.wikipedia.org/wiki/상미분_방정식 "wikilink").

## 정의

*n*차 **코시-오일러 방정식**은 미지 함수 \(y(x)\)에 대한, 다음과 같은 *n*차 [상미분 방정식이다](https://ko.wikipedia.org/wiki/상미분_방정식 "wikilink").

\[a_nx^n y^{(n)}(x) + a_{n-1}x^{n-1} y^{(n-1)}(x) + \cdots + a_0 y(x) = 0\] 여기서 \(a_0,\dots,a_n\)은 주어진 계수들이다.

### 2차 코시-오일러 방정식

2차 코시-오일러 방정식은 다음과 같다.

\[x^{2}y''+axy'+by=0\] 이는 다음과 같이 풀 수 있다. 우선 다음과 같은 [가설 풀이를](../Page/가설_풀이.md "wikilink") 사용하자.

\[y=x^{m}\] 이를 첫 번째 식에 대입하면,

\[x^{2}m(m-1)x^{m-2}+axmx^{m-1}+bx^{m}=0\] 이 되고, \(x\ne0\)일 때 공통인자 \(x^{m}\)을 제거하면,

\[m^{2}+(a-1)m+b=0\] 이 된다.

여기서 \(m\)에 따라 [미분방정식](../Page/미분방정식.md "wikilink")의 해를 구할 수 있다.

| 근의 수       | 해                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------- |
| 서로 다른 두 실근 | \(y=c_{1}x^{m_{1}}+c_{2}x^{m_{2}}\)                                                               |
| 중근         | \(y=\left( c_{1}+c_{2}\ln x \right)x^{m}\)                                                        |
| 공역 복소근     | \(y=x^{u}\left[A\cos\left(v\ln x\right)+B\sin\left(v\ln x\right)\right], m_{1}=u+iv, m_{2}=u-iv\) |
|            |                                                                                                   |
