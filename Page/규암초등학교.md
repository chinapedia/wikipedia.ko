**규암초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [부여군](https://ko.wikipedia.org/wiki/부여군 "wikilink") [규암면](https://ko.wikipedia.org/wiki/규암면 "wikilink") 반산리에 있는 공립 초등학교이다.

## 학교 연혁

| 1931년 | 12월 15일                       | **규암공립보통학교** 개교(4년제)                             |
| ----- | ----------------------------- | ------------------------------------------------ |
| 1938년 | 4월 1일                         | **규암금천공립심상소학교**(6년제)로 개칭                         |
| 1941년 | 4월 1일                         | **규암금천공립국민학교**로 개칭                               |
| 1947년 | 4월 1일                         | **규암국민학교**로 개칭                                   |
| 1949년 | 7월 31일                        | [백강국민학교](../Page/백강초등학교.md "wikilink")(신리분교장) 분리 |
| 1957년 | 3월 1일                         | [합송국민학교](../Page/합송초등학교.md "wikilink")(합송분교장) 분리 |
| 1982년 | 3월 9일                         | 병설유치원 개원                                         |
| 1996년 | 3월 1일                         | 규암초등학교로 개칭                                       |
| 2016년 | 2월 19일                        | 제83회 졸업식(졸업생 70명)                                |
| 3월 2일 | 22학급 편성(특수 2학급), 병설유치원 3학급 편성 |                                                  |
| 2017년 | 3월 2일                         | 22학급 편성(특수 2학급), 병설유치원 3학급 편성                    |
| 2018년 | 2월 9일                         | 제85회 졸업식(졸업생 12,411명)                            |
| 3월 1일 | 22학급 편성(특수 2학급), 병설유치원 3학급 편성 |                                                  |
