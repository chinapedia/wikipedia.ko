**겹침이음**(-splice)이란 규격화되어 공장에서 생산되는 철근을, 건설 현장에서 필요에 의해 이어붙여 길이를 늘이는 것이다. 겹침이음은 구조적으로 권장되지는 않으며, 이음하지 않은 철근보다 안전하지 않기 때문에 설계 또는 시방서 또는 책임구조기술자가 승인하는 경우에만 실시할 수 있다. 즉 겹침 이음은 단순히 철근의 길이가 모자라다고 임의로 실시해서는 안 된다.

휨부재 겹침이음 시 두 철근이 직접 닿지 않게 겹침이음할수도 있는데 이때 두 철근이, 소요 겹침이음 길이 l<sub>s</sub>의 1/5 또는 150mm 중 작은값 이상 떨어지면 안 된다.

다발철근을 이음하는 경우 한 다발 내에서 각 철근의 이음이 한 군데에서 중복되면 안 된다.

원형철근 겹침이음 시는 갈고리를 쓰지만, 이형철근을 겹침이음할 때는 일반적으로 갈고리를 쓰지 않는다.

d<sub>b</sub> ≤ 35mm이면 기계적인 이음(슬리브, 너트 등을 사용) 또는 겹침이음을 할 수 있지만, 35mm를 초과하는 철근에 대해서는 일반적으로 겹침이음을 사용하지 않고 용접이음한다. 기계적 이음 또는 용접 이음하는 경우 강도가 1.25f<sub>y</sub> 이상이어야 한다.

## 겹침이음 등급

겹침이음 등급은 A, B로 나누며, 기준은 다음과 같다.

  - A급: \(\frac{\text{배 근 } A_s}{\text{소 요 } A_s} \geq 2, \frac{\text{겹 침 이 음 } A_s}{\text{전 체 } A_s} \leq \frac{1}{2}\)인 경우
  - B급: \(\frac{\text{배 근 } A_s}{\text{소 요 } A_s} < 2, \frac{\text{겹 침 이 음 } A_s}{\text{전 체 } A_s} > \frac{1}{2}\)인 경우

## 인장철근 최소 겹침이음 길이

인장철근 최소 겹침이음 길이 l<sub>s</sub>는 300mm 이상이어야 하며, A, B급 겹침이음으로 구분한다. A급이 더 적게 겹쳐서 철근을 잇는다.

  - A급: \(l_s = 1.0 l_d\)
  - B급: \(l_s = 1.3 l_d\)

<!-- end list -->

  -
    l<sub>d</sub>: [인장 이형철근 정착길이](https://ko.wikipedia.org/wiki/정착#인장이형철근_최소_정착길이 "wikilink"). 최소 인장이형철근 정착길이 300mm 규정, 철근 보정량 계수 \(\frac{\text{소 요 } A_s}{\text{배 근 } A_s}\)를 사용하지 않은 값\[1\]\[2\]\[3\]

## 압축철근 최소 겹침이음 길이

압축철근 최소 겹침이음 길이 l<sub>s</sub>는 300mm 이상이어야 한다. 콘크리트 설계기준압축강도 \< 21MPa인 경우 겹침이음길이를 1/3 증가시켜야 한다.

  - \(f_y \leq 400MPa: l_s = \left( \frac{1.4f_y}{\lambda \sqrt{f_{ck}}} - 52 \right) d_b \leq 0.072f_y d_b\)
  - \(f_y > 400MPa: l_s = \left( \frac{1.4f_y}{\lambda \sqrt{f_{ck}}} - 52 \right) d_b \leq (0.13f_y - 24) d_b\)

## 주철근에의 사용

주철근에 이형 철근으로 시공하는 경우 겹침이음 길이가 짧아지는 장점이 있다.\[4\]
