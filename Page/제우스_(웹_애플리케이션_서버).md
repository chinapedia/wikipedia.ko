**제우스**(JEUS, Java Enterprise User Solution의 준말)는 [티맥스소프트](https://ko.wikipedia.org/wiki/티맥스소프트 "wikilink")사에서 제작한 한국산 [웹 애플리케이션 서버](../Page/웹_애플리케이션_서버.md "wikilink")(WAS)이다. 현재 기업에서 사용하는 웹 시스템에 사용되고 있으며, [웹 서버인](../Page/웹_서버.md "wikilink") [웹투비](https://ko.wikipedia.org/wiki/웹투비 "wikilink")와 같이 사용되고 있다.

## Java EE 인증

  - 제우스 5 : J2EE 1.4 인증
  - 제우스 6 : Java EE 5 인증
  - 제우스 7 : Java EE 6 인증
  - 제우스 8 : Java EE 7 인증\[1\]

## 에디션

  - Standard Edition
  - Enterprise Edition
