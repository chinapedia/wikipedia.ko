**이집트 숫자**는 [고대 이집트에서](../Page/고대_이집트.md "wikilink") 기원전 3000년 무렵부터 쓰인 [상형 문자로](https://ko.wikipedia.org/wiki/상형_문자 "wikilink") 표기된 [숫자](../Page/숫자.md "wikilink")이다.\[1\] 기원후 1000년 무렵까지 쓰였다. 이집트 수의 [명수법](https://ko.wikipedia.org/wiki/명수법 "wikilink")은 [십진법](https://ko.wikipedia.org/wiki/십진법 "wikilink")에 기반하고 있었지만 [위치 기수법을](https://ko.wikipedia.org/wiki/위치_기수법 "wikilink") 사용하지 않았기 때문에 수의 표기는 상형문자를 병렬하여 나타내었다.

## 역사

[섬네일](https://ko.wikipedia.org/wiki/파일:Rhind_Mathematical_Papyrus.jpg "wikilink") \]\] 고대 이집트는 오래 전부터 많은 기록을 남겼고 이 기록들을 통해서 숫자의 사용 시기도 알 수 있다. 기원전 3100년 무렵 제작된 철퇴에는 전투의 승리를 기념하는 상형문자가 새겨져 있으며 그 가운데에는 여러 가지 수가 포함되어 있다. 기원전 1850년 무렵 제작된 [모스크바 파피루스](https://ko.wikipedia.org/wiki/모스크바_파피루스 "wikilink")\[2\]에는 여러 가지 기초 산술 문제가 이집트 숫자를 사용하여 실려 있다. 기원전 1650년 무렵 고대 이집트의 수학자 아메스가 [신성문자](../Page/신성문자.md "wikilink")를 사용하여 85개의 문제를 수록한 수학책을 발간하였다. 이 수학책은 아메스 파피루스 또는 [린드 파피루스로](https://ko.wikipedia.org/wiki/린드_파피루스 "wikilink") 불린다. 기원전 1167년 무렵 [라메시스 4세가](https://ko.wikipedia.org/wiki/라메시스_4세 "wikilink") 왕위를 계승하면서 아버지인 [라메시스 3세의](https://ko.wikipedia.org/wiki/라메시스_3세 "wikilink") 업적을 기리는 파피루스를 제작하였다. 헤리스 파피루스라고 불리는 이 문서에는 당시 사원의 재산 목록이 포함되어 있어 당시 실용적인 산술의 예를 알 수 있게 해 준다.\[3\]

## 분수의 표기

이집트 숫자 표기에서 일반적인 [유리수](../Page/유리수.md "wikilink")는 [정수](../Page/정수.md "wikilink")의 비로 나타내었다. 그러나 [단위분수](https://ko.wikipedia.org/wiki/단위분수 "wikilink")는 특별히 별도의 기호를 사용하였다.

<hiero>D21</hiero>

위 기호는 “입” 또는 “그것들 사이에”란 의미의 상형문자이다. 이 기호 밑에 수를 쓰면 해당 정수의 [역수](https://ko.wikipedia.org/wiki/역수 "wikilink")를 의미했다. 예를 들어 은 다음과 같이 나타낼 수 있다.

|                               |                   |
| ----------------------------- | ----------------- |
| <hiero>D21:Z1\*Z1\*Z1</hiero> | \(= \frac{1}{3}\) |

또한 , , 는 아래와 같이 나타내었다.

|                     |                   |  |                    |                   |  |                    |                   |
| ------------------- | ----------------- |  | ------------------ | ----------------- |  | ------------------ | ----------------- |
| <hiero>Aa13</hiero> | \(= \frac{1}{2}\) |  | <hiero>D22</hiero> | \(= \frac{2}{3}\) |  | <hiero>D23</hiero> | \(= \frac{3}{4}\) |
