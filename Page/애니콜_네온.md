**애니콜 네온폰**은 [삼성전자](https://ko.wikipedia.org/wiki/삼성전자 "wikilink")에서 [2007년](../Page/2007년.md "wikilink")에 출시한 [휴대 전화이다](../Page/휴대_전화.md "wikilink"). 휴대폰 전면 및 숫자 키보드에 불이 들어오는 것이 특징이다.

## 모델별 차이

각 모델별 차이는 아래와 같다.

| 모델명         | 통신사                                                            | 기타 |
| ----------- | -------------------------------------------------------------- | -- |
| SCH-W290    | [SK텔레콤](../Page/SK텔레콤.md "wikilink")                           |    |
| SPH-W2900   | [KT](../Page/KT.md "wikilink")                                 |    |
| SPH-W2900ML | [KT](../Page/KT.md "wikilink")                                 |    |
| SCH-W390    | [SK텔레콤](../Page/SK텔레콤.md "wikilink")                           |    |
| SPH-W390M   | [멜론](../Page/멜론_\(온라인_음악_서비스\).md "wikilink") 무제한 프로모션 모델\[1\] |    |

디자인에서도 W290모델은 휴대폰 하단 전체가 슬라이드 아랫부분에 있고, 검정색이지만, W390모델은 슬라이드 윗부분이 휴대폰 전면을 덮는다는 점에서 차이가 있다.
