[파나마](https://ko.wikipedia.org/wiki/파나마 "wikilink")는 기본적으로 공산 국가와는 외교 관계를 맺지 않는 것을 원칙으로 하고 있다.

<td><p>파나마는 <a href="https://ko.wikipedia.org/wiki/1948년" title="wikilink">1948년</a> <a href="https://ko.wikipedia.org/wiki/12월_12일" title="wikilink">12월 12일</a> <a href="https://ko.wikipedia.org/wiki/대한민국" title="wikilink">대한민국</a>을 승인, <a href="https://ko.wikipedia.org/wiki/1962년" title="wikilink">1962년</a> <a href="https://ko.wikipedia.org/wiki/9월_30일" title="wikilink">9월 30일</a> 국교를 수립했다. <a href="https://ko.wikipedia.org/wiki/조선민주주의인민공화국" title="wikilink">조선민주주의인민공화국</a>과는 외교관계가 없다. <a href="https://ko.wikipedia.org/wiki/1973년" title="wikilink">1973년</a> <a href="https://ko.wikipedia.org/wiki/5월_5일" title="wikilink">5월 5일</a> 파나마 상주 대사관이 개설되었으며, <a href="https://ko.wikipedia.org/wiki/1979년" title="wikilink">1979년</a> <a href="https://ko.wikipedia.org/wiki/3월_1일" title="wikilink">3월 1일에는</a> 주한 파나마 상주 대사관이 개설되었다. <a href="https://ko.wikipedia.org/wiki/1984년" title="wikilink">1984년</a> 7월 대통령 당선자 바를레타가 대한민국을 방문하여 우호를 다졌다. 양국간에 문화협정(1974. 6)·항공협정(1979. 12)이 체결되었다. 해외지사 14개, 교포상사 9개가 진출해 있다. 대한민국은 의류·선박을 수출하고, 파나마는 선박·석유를 수출한다. <a href="https://ko.wikipedia.org/wiki/1996년" title="wikilink">1996년</a> 기준으로 대한수입액은 31억 9,501만 달러이고, 대한수출액은 5억 4,765만 달러이다. <a href="https://ko.wikipedia.org/wiki/1997년" title="wikilink">1997년</a> 기준으로 교민수는 89명, 체류자는 239명에 이른다.</p></td>

<td><p>미국에서 자금을 조달하여 건설된 <a href="https://ko.wikipedia.org/wiki/파나마_철도" title="wikilink">파나마 철도가</a> <a href="https://ko.wikipedia.org/wiki/1855년" title="wikilink">1855년</a> 개통되었으며, 곧바로 <a href="../Page/콜론_(파나마).md" title="wikilink">콜론이라는</a> 도시가 세워졌다. 1850년부터 1900년까지 40개의 행정부, 50번의 폭동, 5번 차례에 일어안 분리 시도, 13차례에 걸친 미국의 간섭 등이 있었으며, 20세기초 운하에 대한 미국의 관심이 고조되었다. 콜롬비아 상원이 파나마 운하 조약의 비준을 연기했을 때 미국은 파나마 분리주의자의 운동을 지지했다. 그로 인해 <a href="https://ko.wikipedia.org/wiki/1903년" title="wikilink">1903년</a> <a href="https://ko.wikipedia.org/wiki/11월_6일" title="wikilink">11월 6일</a> 파나마 폭동이 일어났으며, <a href="https://ko.wikipedia.org/wiki/11월_18일" title="wikilink">11월 18일</a> <a href="https://ko.wikipedia.org/wiki/헤이-뷔노-바리아_조약" title="wikilink">헤이-뷔노-바리아 조약이</a> 체결되었고 파나마는 콜롬비아에서 분리된다. 이 조약으로 미국은 <a href="https://ko.wikipedia.org/wiki/파나마_운하" title="wikilink">파나마 운하지대의</a> 영구 소유권을 얻어냈으며, 완공된 운하는 <a href="https://ko.wikipedia.org/wiki/1914년" title="wikilink">1914년</a> 개통되었다. 미국의 침공과 21년간의 군사 독재 이래로 파나마는 성공적으로 세 번이나 평화적인 정권 교체를 이루었다. <a href="https://ko.wikipedia.org/wiki/제2차_세계_대전" title="wikilink">제2차 세계 대전</a> 이후에 파나마는 혼란에 빠졌다. 폭동이 자주 일어났으며 반미감정이 퍼져나갔다. 파나마가 예전의 미국 운하지대에 대한 사법권을 맡게 되었으나, 운하 자체의 운용권은 <a href="https://ko.wikipedia.org/wiki/1999년" title="wikilink">1999년</a>까지 미국에 남아 있었지만, 현재는 파나마 정부에 넘어간 상태이다.</p></td>
