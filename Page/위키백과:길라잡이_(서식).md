위키백과에 서식을 적용하는 방법은 워드 프로세서와 차이가 있습니다. [위지위그](../Page/위지위그.md "wikilink") 방식 대신, 위키백과는 텍스트를 사용하여 각 요소를 추가합니다. 이것은 편리한 편집을 위해 만들어진 **위키텍스트**라고 알려진 언어입니다. 위키텍스트로 표현할 수 없는 것들은 [XHTML](../Page/XHTML.md "wikilink")로 표현할 수 있습니다.

## 대표적인 위키 태그

|                                            |                         |
| ------------------------------------------ | ----------------------- |
| **이렇게 쓰면**                                 | **이렇게 표시됩니다**           |
| `''기울어지게'' 나타내기`                           | *기울어지게* 나타내기            |
| `'''진하게''' 나타내기`                           | **진하게** 나타내기            |
| `'''''진하고 기울어지게''''' 나타내기`                 | ***진하고 기울어지게*** 나타내기    |
| `<nowiki>'''''위키텍스트 적용하지 않기'''''`</nowiki> | '''''위키텍스트 적용하지 않기''''' |

위키백과에서는 문서에서 처음 나온 표제어를 **진하게** 표시합니다. 예를 들어 [대한민국](../Page/대한민국.md "wikilink") 문서는 다음과 같이 시작합니다.

  - **대한민국**(大韓民國)은 [동아시아](https://ko.wikipedia.org/wiki/동아시아 "wikilink")의 [한반도](../Page/한반도.md "wikilink") 남부에 자리한 [공화국](https://ko.wikipedia.org/wiki/공화국 "wikilink")이다.

## 제목과 부제목

제목과 부제목은 문서를 조직적으로 구성하는 데에 도움을 줍니다. 하나의 문서가 여러 소주제로 나누어진다면 각각 부분으로 나눌 수 있습니다.

## 자주 쓰이는 XHTML 태그

  - <small>작게</small> 나타내기
      - (양쪽에 <small>, </small>을 붙입니다.)
  - <big>크게</big> 나타내기
      - (양쪽에 <big>, </big>을 붙입니다.)
  - ~~취소선이 그어지게~~ 나타내기
      - (양쪽에 ~~,~~을 붙입니다.)
  - <ins>밑줄이 그어지게</ins> 나타내기
      - (양쪽에 <ins>, </ins>를 붙입니다.)
  - <sup>위쪽에 작게</sup> 나타내기
      - (양쪽에 <sup>,</sup>를 붙입니다.)
  - <sub>아래쪽에 작게</sub> 나타내기
      - (양쪽에 <sub>,</sub>를 붙입니다.)
