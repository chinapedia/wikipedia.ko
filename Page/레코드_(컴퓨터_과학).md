[컴퓨터 과학에서](../Page/컴퓨터_과학.md "wikilink") **레코드**(, [struct](https://ko.wikipedia.org/wiki/struct "wikilink"))는 기본적인 [자료 구조이다](../Page/자료_구조.md "wikilink"). [데이터베이스](../Page/데이터베이스.md "wikilink")나 [스프레드시트](../Page/스프레드시트.md "wikilink")의 레코드는 보통 로우(row)라고 부른다.\[1\]\[2\]\[3\]\[4\]

레코드는 각기 다른 자료형에 속할 수 있는 [필드](https://ko.wikipedia.org/wiki/필드 "wikilink")의 모임이며, 보통 고정 숫자나 시퀀스로 이루어져 있다.\[5\] 레코드의 필드들은 특히 [객체 지향 프로그래밍에서](../Page/객체_지향_프로그래밍.md "wikilink") 멤버(member)로도 부른다.

## 예시

다음은 레코드 정의의 예를 보여준다.

\* PL/I:

``` text
  declare 1 date,
            2 year  fixed binary,
            2 month fixed binary,
            2 day   fixed binary;
```

  - Algol 68:

<!-- end list -->

  -
    `  `**`mode`**`   `**`date`**`  =  `**`struct`**`  ( `**`int`**`  year,  `**`int`**`  month,  `**`int`**`  day); `

\* C:

``` c
struct date {
   int year;
   int month;
   int day;
};
```

\* [Go](../Page/Go_\(프로그래밍_언어\).md "wikilink"):

``` go
type Date struct {
        year  int
        month time.Month
        day   int
}
```

\* [파스칼](../Page/파스칼.md "wikilink"):

``` pascal
type TDate = record
   Year: Integer;
   Month: 1..12;
   Day: 1..31;
end;
```

\* [러스트](../Page/러스트_\(프로그래밍_언어\).md "wikilink"):

``` rust
struct Date {
   year: u32,
   month: u32,
   day: u32,
}
```

\* [하스켈](../Page/하스켈.md "wikilink"):

``` haskell
data Date = Date { year :: Integer
                 , month :: Integer
                 , day :: Integer
                 }
```

\* [줄리아](../Page/줄리아.md "wikilink"):

``` julia
struct Date
    year::Int
    month::Int
    day::Int
end
```

\* [Standard ML](https://ko.wikipedia.org/wiki/Standard_ML "wikilink"):

``` sml
type date = {year:int, month:int, day:int}
```
