일반적으로 **카파 상관계수**는 코헨(Cohen)의 카파 상관계수(Kappa)를 가리키며 이는 2명의 관찰자(또는 평가자)의 [신뢰도](https://ko.wikipedia.org/wiki/신뢰도 "wikilink")를 확보하기위한 [확률](https://ko.wikipedia.org/wiki/확률 "wikilink")로서 평가지표로 사용되는 상관계수이다. 2명이상에서 신뢰도를 얻기위해서는 [플레이스 카파 상관계수](https://ko.wikipedia.org/wiki/플레이스_카파_상관계수 "wikilink")(Fleiss' kappa)를 사용할수있다.

\[K_{C} = {{p_o - p_e}\over{1 - p_e}}\]

## 예

조건

| 평가자     | B       |
| ------- | ------- |
| 긍정(Yes) | 부정(No)  |
| A       | 긍정(Yes) |
| 부정(No)  | c       |

예시

| 평가자 | B  |
| --- | -- |
| 긍정  | 부정 |
| A   | 긍정 |
| 부정  | 10 |

관찰된 동의할경우의 비례값 P<small>o</small>는 다음과 같다.

\[p_o = \frac{a+d}{a+b+c+d} = \frac{20+15}{50} = 0.7\]

인 확율값(the probability of random agreement)을 계산하기위해서

  -
    \({{a+b}\over{a+b+c+d}} =0.5\)
    평가자 A 의 긍정 확율값은 50% 이고
    \({{a+c}\over{a+b+c+d}} =0.6\)
    평가자 B 의 긍정 확율값은 60%이다.
    \({{a+b}\over{a+b+c+d}} =0.5\)
    평가자 A 의 부정 확율값은 50% 이고
    \({{a+c}\over{a+b+c+d}} =0.4\)
    평가자 B 의 부정 확율값은 40%이다.

따라서

  -
    \(p_\text{Yes} = \frac{a+b}{a+b+c+d} \cdot \frac{a+c}{a+b+c+d} = 0.5 \times 0.6 = 0.3\)
    \(p_\text{No} = \frac{c+d}{a+b+c+d} \cdot \frac{b+d}{a+b+c+d} = 0.5 \times 0.4 = 0.2\)

따라서 이 경우 계산되는 전반적인 임의 동의 확률은 '긍정' 또는 '부정'에 동의한 확률값이다. 즉

\[p_e = p_\text{Yes} + p_\text{No} = 0.3 + 0.2 = 0.5\]

이때 코헨(Cohen)의 카파 상관계수(Kappa) K<small>C</small>는

\[K_{C} = \frac{p_o - p_e}{1 - p_e} = \frac{0.7-0.5}{1-0.5} = 0.4 \!\] 이다.

## 변형

\[K_{C} = {{p_o - p_e}\over{1 - p_e}}=1- {{1 - p_o}\over{1 - p_e}}\] P<small>e</small>=0일때

\[K_{C} = {{p_o - p_e}\over{1 - p_e}}={{p_o -0}\over{1 - 0}}={{p_o}\over{1}}=p_o\] P<small>o</small>=0일때

\[K_{C} = {{p_o - p_e}\over{1 - p_e}}={{0-p_e}\over{1 - p_e}}={{p_e}\over{p_e - 1}}\]은 P<small>e</small>가 클수록 1에 가까워지며 P<small>e</small>가 작을수록 0에 가까워진다.
