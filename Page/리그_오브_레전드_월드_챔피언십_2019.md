**리그 오브 레전드 월드 챔피언십 2019** (League of Legends World Championship 2019)(**2019롤드컵**)는 라이엇 게임즈가 주최, 주관하는 [리그 오브 레전드 월드 챔피언십의](../Page/리그_오브_레전드_월드_챔피언십.md "wikilink") 9번째 대회이다.

## 개요

  - 주최 : 라이엇 게임즈
  - 주관 : 라이엇 게임즈
  - 후원 : 마스터카드, 에일리언웨어, 시크릿랩, 루이비통
  - 주제가 : Phoenix
  - 기념 스킨 : 챔피언십 라이즈
  - 우승팀 기념 스킨 : 갱플랭크, 리 신, 말파이트, 베인, 쓰레쉬

### 방식

  - 플레이-인 : 메이저 상위 4지역 3번 시드 4팀, 마이너 지역 우승팀 8팀이 참가
      - 그룹 스테이지 : 4개 조로 나누어 더블 라운드 로빈 방식으로 진행, 각 조 상위 2팀이 2라운드에 진출
      - 녹아웃 토너먼트 : 5전 3선승제
  - 메인 이벤트 : 메이저 지역 12팀, 플레이-인 통과한 4팀이 참가
      - 그룹 스테이지 : 4개 조로 나누어 더블 라운드 로빈 방식으로 진행, 각 조 상위 2팀이 결승 토너먼트에 진출
      - 결승 토너먼트 : 5전 3선승제

### 변경된 점

  - 이번 대회부터 북부 라틴 아메리카 리그(LLN)와 남부 라틴 아메리카 컵(CLS)가 라틴 아메리카 리그(LLA)로 통합되었다.
  - 리그 오브 레전드 챔피언십 시리즈 유럽(LCS EU)은 리그 오브 레전드 유로피언 챔피언십(LEC)로 변경되었다.
  - 리그 오브 레전드 챔피언십 시리즈 북아메리카(LCS NA)는 LCS로 변경되었다.
  - 동남아시아 투어(SEA Tour)는 리그 오브 레전드 동남아시아 투어(LST)로 변경되었다.
  - [미드 시즌 인비테이셔널 2019에서](../Page/리그_오브_레전드_미드_시즌_인비테이셔널_2019.md "wikilink") 가장 성적이 좋은 마이너 지역의 출전 팀 수가 1팀(그룹 스테이지 1팀)에서 2팀(그룹 스테이지 1팀 + 플레이-인 스테이지 1팀)으로 늘어났다.
  - 예비 로스터 한 명 더 추가하여 팀당 최대 7인 로스터로 구성하게 되었다.

### 일정

  - 플레이-인
      - 그룹 스테이지 : 10월 2일 (수) \~ 5일 (토)
      - 녹아웃 토너먼트 : 10월 7일 (월) \~ 8일 (화)
  - 메인 이벤트
      - 그룹 스테이지
          - 1주차 : 10월 12일 (토) \~ 15일 (화)
          - 2주차 : 10월 17일 (목) \~ 20일 (일)
      - 8강 : 10월 26일 (토) \~ 27일 (일)
      - 준결승 : 11월 2일 (토) \~ 3일 (일)
      - 결승 : 11월 10일 (일)

### 경기장

  - 플레이-인 :  독일 베를린 LEC 스튜디오
  - 메인 이벤트
      - 그룹 스테이지 :  독일 베를린 베르티 뮤직 홀
      - 8강전 \~ 준결승 :  스페인 마드리드 팔라시오 비스탈레그레
      - 결승 :  프랑스 파리 아코르호텔 아레나

### 상금

  - 총 상금 : 기본 상금 ($ 2,250,000) + α (크라우드 펀딩 전체 판매 수익의 25%)
  - 우승 : 총 상금의 37.5%
  - 준우승 : 총 상금의 13.5%
  - 준결승 : 총 상금의 7%
  - 8강 : 총 상금의 4%
  - 그룹 스테이지 각 조 3위 : 총 상금의 2.25%
  - 그룹 스테이지 조 공동 3위 : 총 상금의 1.75% (순위 결정전 없음)
  - 그룹 스테이지 각 조 4위 : 총 상금의 1.25%
  - 플레이-인 스테이지 2라운드 : 총 상금의 0.75%
  - 플레이-인 스테이지 1라운드 : 총 상금의 0.5%
