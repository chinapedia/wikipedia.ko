《**이영돈·신동엽 젠틀맨**》은 [채널A](https://ko.wikipedia.org/wiki/채널A "wikilink")의 예능프로그램이다.

## 시청률

### 2013년

| 회차         | 방송일                                                         | 시청률 |
| ---------- | ----------------------------------------------------------- | --- |
| TNmS (수도권) | AGB닐슨 (수도권)                                                 |     |
| 1          | [12월 22일](https://ko.wikipedia.org/wiki/12월_22일 "wikilink") |     |
| 2          | [12월 29일](https://ko.wikipedia.org/wiki/12월_29일 "wikilink") |     |

  - TNmS와 AGB닐슨 미디어 리서치에서 공개한 표를 바탕으로 작성한 시청률이다. 빨간색 글씨는 최고 시청률, 파란색 글씨는 최저 시청률을 나타낸다.
