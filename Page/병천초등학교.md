**병천초등학교**는 [충청남도](https://ko.wikipedia.org/wiki/충청남도 "wikilink") [천안시](../Page/천안시.md "wikilink") [동남구](../Page/동남구.md "wikilink") [병천면](../Page/병천면.md "wikilink") 병천리에 있는 공립 초등학교이다.\[1\]

## 학교 연혁

| 1929년 | 5월 2일  | 갈전공립보통학교(4년제) 개교                |
| ----- | ------ | ------------------------------- |
| 1937년 | 4월 1일  | 학년 6년제 실시                       |
| 1938년 | 4월 1일  | 갈전공립심상소학교로 개칭                   |
| 1941년 | 4월 1일  | 갈전공립국민학교로 개칭                    |
| 1943년 | 4월 1일  | **병천공립국민학교** 개칭                 |
| 1949년 | 9월 30일 | 봉성분교장 승격 분리                     |
| 1981년 | 3월 1일  | 병설유치원 개원                        |
| 1991년 | 3월 1일  | 송정초등학교 분교장으로 편입                 |
| 1993년 | 3월 1일  | 송정분교장 통폐합                       |
| 1995년 | 3월 1일  | 봉성초등학교 분교장으로 편입                 |
| 1996년 | 3월 1일  | 병천초등학교로 개칭                      |
| 2007년 | 3월 1일  | 봉성분교장 통폐합                       |
| 2018년 | 2월 9일  | 제86회 28명 졸업(계 8,296명)           |
| 2018년 | 3월 1일  | 13학급 편성(학습도움반 포함), 병설유치원 1학급 편성 |
