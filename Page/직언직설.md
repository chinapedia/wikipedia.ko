《**직언직설**》은 [대한민국](../Page/대한민국.md "wikilink")에서 월\~금 오후 2시 50분 \~ 4시 20분에 [텔레비전](../Page/텔레비전.md "wikilink")으로 방송된 [채널A](../Page/채널A.md "wikilink")의 [시사](../Page/시사.md "wikilink")프로그램이다.

| 역대 | 진행자                                                           | 진행 기간                        | 비고 |
| -- | ------------------------------------------------------------- | ---------------------------- | -- |
| 1기 | [이언경](https://ko.wikipedia.org/wiki/이언경 "wikilink")           | 2012년 12월 24일 \~ 2016년 2월 5일 |    |
| 2기 | [조수진](https://ko.wikipedia.org/wiki/조수진_\(1972년\) "wikilink") | 2016년 2월 10일 \~ 2016년 5월 27일 |    |
| 3기 | [이남희](https://ko.wikipedia.org/wiki/이남희 "wikilink")           | 2016년 5월 30일 \~ 2017년 1월 13일 |    |
|    |                                                               |                              |    |

## 타이틀 변천사

현재 타이틀은 [2012년](../Page/2012년.md "wikilink") [12월 24일부터를](../Page/12월_24일.md "wikilink") 기준으로 한다.

| 역대 | 타이틀                                | 방송 기간                         |
| -- | ---------------------------------- | ----------------------------- |
| 1대 | 이언경의 세상만사                          | 2012년 7월 16일 \~ 2012년 12월 21일 |
| 2대 | [직언직설](../Page/직언직설.md "wikilink") | 2012년 12월 24일 \~ 2017년 1월 13일 |
