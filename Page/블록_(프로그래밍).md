[프로그램 코드에서](https://ko.wikipedia.org/wiki/프로그램_코드 "wikilink") **블록**(block)이란 마치 한 [문단](https://ko.wikipedia.org/wiki/문단 "wikilink")처럼 보이는, 코드의 한 부분을 뜻하며, [중괄호](https://ko.wikipedia.org/wiki/중괄호 "wikilink")로 묶여 있는 경우가 많다. 보통 1개 이상의 명령어를 가지고 있으나, [주석으로](../Page/주석_\(프로그래밍\).md "wikilink") 이루어진 블록이나, 아무 내용도 없는 빈 블록도 가능하다.

## 전형적인 블록

``` c
int main()
{
  return 0;
}
```

## 중첩된 블록

``` c
int main()
{
  int x = 1;

  if (x == 1)
  {
     x++;
  }
  return 0;
}
```

## 다른 형태

일부 프로그래머들(주로 자바 프로그래머)은 약간 다른 형태로 블록을 작성하기도 하는데, 여는 괄호가 [함수](../Page/함수.md "wikilink")와 같은 줄 오른쪽에 위치하도록 하는 방식을 많이 쓴다.

``` java
int main() {
  return 0;
}

int main() {

  int x=1;

  if (x == 1) {
     x++;
  }

  return 0;
}
```

[비주얼 베이직은](../Page/비주얼_베이직.md "wikilink") 다음과 같이 종료 문장을 명확히 기술해야 한다.

``` vb
If x > 0 Then

  y = y + x

End If

For i = 1 To 10

  DoSomething(i)

Next ' or Next i
```

[SQL 서버나](https://ko.wikipedia.org/wiki/SQL_서버 "wikilink") [파스칼](../Page/파스칼_\(프로그래밍_언어\).md "wikilink") 등 다른 몇몇 언어들은 시작(Begin)문과 종료(End)문으로 블록을 구성한다.

``` sql

IF y IS NOT NULL
BEGIN
  SELECT * FROM employee WHERE name = y
END
```
