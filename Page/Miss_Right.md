〈**Miss Right**〉는 [방탄소년단](../Page/방탄소년단.md "wikilink")의 첫 번째 리패키지 음반 《[SKOOL LUV AFFAIR SPECIAL ADDITION](../Page/SKOOL_LUV_AFFAIR_SPECIAL_ADDITION.md "wikilink")》의 타이틀 곡이다. 2014년 5월 14일에 발표하였다.

| 차트                                       | 최고순위 |
| ---------------------------------------- | ---- |
| 가온 주간 디지털 차트\[1\]                        | 43   |
| 가온 주간 다운로드 차트\[2\]                       | 31   |
| 가온 주간 BGM 차트\[3\]                        | 37   |
| 가온 월간 BGM 차트\[4\]                        | 80   |
| 가온 주간 모바일 차트 <small>(통화연결음)</small>\[5\] | 79   |
|                                          |      |
