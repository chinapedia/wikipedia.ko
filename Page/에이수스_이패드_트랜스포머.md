**에이수스 이패드 트랜스포머**(, TF101)는 2011년 3월 30일 [에이수스](../Page/에이수스.md "wikilink")가 CES 2011에서 발표한 [안드로이드](https://ko.wikipedia.org/wiki/안드로이드_\(운영_체제\) "wikilink") [태블릿 컴퓨터이다](../Page/태블릿_컴퓨터.md "wikilink").\[1\]

## 하드웨어

| ------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
|                                                               |                                                                                      |
| 디스플레이                                                         | 10.1인치 IPS 디스플레이                                                                     |
| [중앙 처리 장치](https://ko.wikipedia.org/wiki/중앙_처리_장치 "wikilink") | 1 [GHz](https://ko.wikipedia.org/wiki/GHz "wikilink") dual-core nVidia Tegra 2       |
| 메모리                                                           | 1GB \[2GB로 확장 가능\]                                                                   |
| 저장용량                                                          | 16GB, 32GB [SSD](https://ko.wikipedia.org/wiki/SSD "wikilink")                       |
| 무선네트워크                                                        | | 내장 [802.11a/b/g/n](https://ko.wikipedia.org/wiki/IEEE_802.11 "wikilink"), 블루투스 2.1 |
