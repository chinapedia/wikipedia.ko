**스튜디오 1035**는 [TBN 경북교통방송](https://ko.wikipedia.org/wiki/경북교통방송 "wikilink")(포항 FM 103.5㎒, 울진 FM 103.7㎒)에서 평일 아침 9시부터 11시까지 방송되는 경북 동해안 지역의 출근길 아침 라디오 프로그램이다.

## 매일 코너

  - 사투리 퀴즈, 가가 가가 : 주말부부의 전화통화를 듣고 풀어보는 사투리 퀴즈
  - 아침, 책갈피 : 아침에 듣는 책속 한마디

## 요일 코너

  - <월요일>
      - 이야기 좀 합시다 : 소통강사 박동주의 속시원한 대화
      - 동행 프로젝트, 힘내라 경북\!\! : 창업 성공 스토리
  - <화요일>
      - 포항 읽어주는 남자 : 포항시 승격 70주년을 맞아 듣는 포항의 옛이야기, 이재원 포항지역학연구회장
      - 이 소리에 집중하라 : 소리로 풀어보는 청취자 참여 퀴즈
  - <수요일>
      - 알쏭달쏭 우리말 : 제대로 알고가는 우리말, 한글학회 성기지 연구편찬실장
      - 푹빠진 사람들\! : 한가지 일에 푹빠진 지역민들의 이야기, 취재리포터 코너
  - <목요일>
      - 중계차 타고 동네 한바퀴 : 중계차가 지역민들을 직접 찾아갑니다. 박경덕 아나운서
      - Quiz, 다시보는 운전면허 : 퀴즈로 풀어보는 운전면허시험, 도로교통공단 박지은교수
  - <금요일>
      - 생활의 꿀팁\! : 놓칠 수 없는 생활속 지혜
      - 이번 주말, 여기 어때요? : 지역 명소를 찾아서, 박희경,정종훈 숲해설가
